## 1. 心智模型 (6 个)

### 1.1 codifiable 原料 ⇄ tacit 嗅觉记忆（第一性张力）

- **(figures: Jean-Claude Ellena / Jean Carles / Edmond Roudnitska)**
- **一句话**：这行可背诵的一半（原料、香料化学、accord 公式、挥发度、IFRA 限值）能进教科书；另一半——成为「the nose」的嗅觉记忆与品味——不可言传，只能靠每天盲嗅背料一克一克长出来。学校先让学徒「记忆几百个原料」是把 tacit 工程化的脚手架，不是终点。
- **应用方式**：面对任何「怎么学/怎么评/谁更强」的问题，先把它拆成 codifiable 层（可查 TGSC、可算 IFRA、可背家族）与 tacit 层（要反复闻、要时间、无捷径）；把承诺「速成大师」「软件替你调香」的说法判为外行——tacit 层没有快进键。
- **局限**：放大版心智模型⚠️——「显性知识 vs 隐性技能」并非香水独有（厨艺、品酒、音乐皆然），香水的排他性在于 tacit 层被推到极致（嗅觉无书面记法、上千原料靠条件反射识别）且 codifiable 层被法规（IFRA）硬约束；用在跨行业类比时须强调这种「极致 + 强法规」的放大度，否则会退化成「熟能生巧」的废话。
- **evidence**: [T04-S004, T04-S001, T01-S013]

### 1.2 暗示而非复刻：香水是「说真话的谎言」

- **(figures: Jean-Claude Ellena / Dominique Ropion / Chandler Burr)**
- **一句话**：顶级调香不复刻自然，而是用（多为合成的）分子制造一个比真实更可信的**嗅觉幻觉**——Ellena 自称 illusionist，Ropion 的 Carnal Flower 是「扭曲/暗示一朵晚香玉」而非忠实还原，Burr 引 Ellena 引毕加索「艺术是说真话的谎言，那对我就是香水」。
- **应用方式**：评判一支香或一个创作意图时，问「它在制造哪种幻觉、用什么分子骗过鼻子」，而不是「它像不像真的花」；把「天然才真、合成是假」当外行框架——合成分子恰是幻觉的实现手段（hydroxycitronellal 重现不可萃取的铃兰、Iso E Super 造无对应自然物的木质绒感）。
- **局限**：偏「艺术派/法系」视角——功能香精（洗涤/家居）与部分商业线确实追求「闻起来就是柠檬/海洋」的写实复刻，幻觉论对这类需求解释力弱；用时须标明它描述的是 fine fragrance 的创作美学，不是全行业的统一目标。
- **evidence**: [T01-S001, T01-S021, T01-S029]

### 1.3 accord 是创作的「单词」，挥发度金字塔是它的语法

- **(figures: Jean Carles / Sophia Grojsman / Sylvaine Delacourte)**
- **一句话**：创作不是「堆原料」而是「搭和弦」——accord 是 5–12 个原料失去各自身份、融成一个不可拆分的新气味单位（amber accord 由 labdanum+vanilla+benzoin 构成，单料都不像 amber），它决定家族归属；再按挥发度把 accord/原料组织进 top→heart→base 金字塔（Jean Carles 法明确「自底向上」先搭 base）。
- **应用方式**：拿到一个香气目标，先想「核心 accord 是什么、由哪几个料承重」（Grojsman 常用 4–7 个料造主和弦，其余只是支撑），再排挥发度结构；不要把 Fragrantica 的 note 金字塔当配方——那是感知叙事/营销文案，配方还原要 GC-MS。
- **局限**：accord/金字塔是「结构骨架」而非「美学保证」——能正确搭出平衡 accord 不等于做出好香（好坏仍是 tacit 品味，见 1.1）；且金字塔描述的是「随时间的感知」，与实际用料表不一一对应，过度用结构语言会滑向「读金字塔=懂配方」的外行坑。
- **evidence**: [T04-S040, T04-S033, T01-S031]

### 1.4 overdose 是审美手法，不是失误

- **(figures: Dominique Ropion / Sophia Grojsman / Mathilde Laurent)**
- **一句话**：**故意**把某个分子用到远超常规比例，把它推到香水结构顶端做签名效果——这是审美决策不是「放太多/超 IFRA」。Ropion 拒绝「overdose」一词（「高剂量恰是配方平衡之美」）、Grojsman 以「反转经典比例」开宗、Laurent 的 soloist/choir 是「旗舰原料 overdose + 周围尽量少放」。
- **应用方式**：闻到某个原料「突出到几乎失衡却没散」时，判它是有意的 overdose 签名而非新手错误；设计签名时考虑「哪一个料值得被推到顶端承重」。外行把 overdose 当「超量/超标」是顶级破绽（内行是手法，且仍须过 IFRA 闸门）。
- **局限**：overdose 与「无意超量/超 IFRA 限值」在成品上可能听起来相近，判定依赖对作者意图与平衡的 tacit 判断（见 1.1），不可机械地把「某料很突出」一律读成审美 overdose；且这是 fine fragrance 技法层模型，对功能香精不适用。
- **evidence**: [T01-S021, T01-S031, T01-S039]

### 1.5 法规是「隐形合著者」：合规先于创作自由，经典被迫重配

- **(figures: Thierry Wasser / Olivier Polge / Christine Nagel)**
- **一句话**：IFRA Standards（按修订版变的限用浓度）+ EU 致敏原标签是上市前的硬闸门，天然与合成同等受查、不豁免；橡苔（chypre 的灵魂定香）因 atranol 致敏被限用，逼得 Mitsouko / Chanel No.5 这类经典被迫**重配（reformulation）**——创作自由 ⇄ 安全合规是结构性矛盾，不是偶发麻烦。
- **应用方式**：任何「为什么经典款味道变了/这料还能用吗/我的配方能上市吗」的问题，先查现行 IFRA 修订版 + EU 2023/1545（caveat：限值随版本变，引用必标版本号）；把「这是几十年没变的原版真配方」当外行叙事——多数老香已被 reformulate。
- **局限**：法规是创作的**约束条件**而非创作本身——它解释「为什么不能这样做/为什么味道变了」，不解释「怎样做出好香」（后者回到 1.1–1.4）；且 IFRA 是行业协会自律（成员约占全球香料量 80%，业内口径），EU 才是法律，二者效力不同，混为「政府强制」是常见误读。
- **evidence**: [T03-S021, T03-S022, T06-S019]

### 1.6 创作自由的光谱：B2B consensus 命题创作 ⇄ niche/indie carte blanche

- **(figures: Frédéric Malle / Christine Nagel / Andy Tauer)**
- **一句话**：这行有两套对「香水是什么」的世界观——B2B：四大香精公司为品牌 core list 竞标，evaluator 派多名 perfumer 接同一 brief、常复用 performing formula、多国盲测定胜负、追「最 consensus 的配方」；niche/indie：从灵感（非 brief）起、carte blanche、不盲测、手工小批、自持配方。关键是**同一大师常横跨两端**（Ropion 同做 Malle niche 与 Lancôme 商业；Kurkdjian 既是 indie 创始人又掌 Dior），不是非此即彼，而是同一人在不同约束下切换工作流。
- **应用方式**：评一支香或一个职业选择，先定位它在哪种约束里——商业线的「相似/安全/复用」是机制使然不是才华不足；niche 的「作者性/自由」也含营销成分（Hermès「5 人选香不盲测」既是真流程也是品牌话术）。判断一个大师时看他在两端各能做到什么。
- **局限**：光谱两端是理想型，现实是连续谱且边界在融（indie 也要过 IFRA、designer 也有高水准作者作品）；「创作自由」叙事本身有营销修饰，不能照单全收为事实——须把它当「世界观/定位」而非「质量高低」的判据。
- **evidence**: [T01-S040, T01-S017, T03-S005]

## 2. 标准 Playbook

> 形如「**如果 {场景}**，则 {决策}」。每条配 1–2 个具体案例。

1. **如果有人承诺「速成调香大师」或「软件/AI 替你调出好香」**，则判为外行/营销——tacit 的嗅觉记忆只能靠反复闻料内化（无书面记法、无捷径），工具与 AI 只是脚手架，最终评估/品味/accord 微调仍是人鼻工作。
   - 案例：Givaudan 内部学校年均仅收约 3 名学员、训练 4 年，先记忆约 500 个原料（厂商自述，T03-S015）；Carto/Philyra 被四大定位为「辅助不决策」，Becker 喻 Carto 为「像 iPhone——汇集信息但不替 perfumer 做决定」(T03-S025)。

2. **如果一个配方要上市（哪怕全天然、哪怕个人小批）**，则先过 IFRA + EU 致敏原闸门、且引用任何限值必标修订版本号——合规先于创作自由，天然不豁免。
   - 案例：EU Reg 2023/1545 把可申报致敏原由约 26 扩到 80+（约 82 含分组，T03-S021），新品强制 2026-07-31；天然 EO 含 limonene/linalool 等本身即申报致敏原（T06-S025），「天然=安全」站不住。

3. **如果要选工具/原料供应**，则按身份分层——学习者用「方法（Jean Carles 法）+ 训练 kit + TGSC」、indie 用「小量供应商 + 自带 IFRA 检测的轻量软件」、pro 用「公司调色板（含 captives）+ 企业 PLM + 内部 AI」；身份错配（indie 上企业 PLM、pro 用裸 Excel）是常见坑。
   - 案例：个人买不到四大 captive 专利分子，只能走 PA/Pell Wall/Hekserij 等小量供应商（T02-S004/005）；裸 Excel 无 IFRA 自动校验最易漏限用，故 indie 软件把 IFRA 检测做成标配卖点（T02-S008）。

4. **如果要「还原一支香的配方」**，则用 GC-MS + headspace 而非读 note 金字塔——金字塔是营销叙事/感知文案，不必对应实际用料。
   - 案例：Fragrantica「金字塔=营销小说」社区吐槽（T02-S019）；deformulation 用色谱谱图识别拆解，且天然混合物/痕量 captive 难定量、识别仍多靠手工 marker，有真实边界（T02-S015）。

5. **如果要判断一支香的价值**，则等 maceration（陈化）后、追到 dry-down（尾调）再下结论，不在喷上前 5 分钟或新调时评判。
   - 案例：EDP 常需静置约 1 个月起、复杂配方 1–3 月，陈化前后气味差异大、跳过会误判留香（T03-S032，约定俗成各家不同）；前调几分钟即逝，dry-down 才是一支香的真身（T06-S008）。

6. **如果用户在商业（B2B）语境创作**，则中枢是「core list → brief → creation → submission → mods → win/lose」，且 evaluator 是 brief 翻译器 + 客观性闸门——别用「每支都是独一无二人手艺术」的浪漫叙事去解释商业线。
   - 案例：香精公司开发期不收费、不赢就没钱，故有强动机做「最 consensus 的配方」、复用 performing formula（T03-S005，机制为业内出版物口径）；evaluator 把「离创作太近」的 perfumer 拉回客观性、成为「你鼻子的延伸」(T03-S002)。

7. **如果遇到「天然 vs 合成」「niche vs designer」的优劣之争**，则祛魅——价格挂钩稀缺/品牌而非「好闻或表现」，niche 常用与 designer 相同的合成麝香/木质琥珀再翻价；天然不更安全、合成不更低级，是世界观差异不是优劣。
   - 案例：现代 musk/ambergris 几乎全合成（Ambroxan 替龙涎、Galaxolide 白麝香，T06-S010）；发酵法分子（Firsantol 檀香、Clearwood Prisma）正重写「天然 vs 合成」的成本/可持续叙事，但是否算「天然」在 ISO 9235 口径上仍有分歧（T02-S032，厂商自报商业化里程碑）。

8. **如果听到「pheromone 吸引异性 / therapeutic-grade 能治病 / 保证 24h 持久 / dupe 和原版一模一样」**，则识别为被资深从业者拒绝的营销话术、给一句机理反驳——这是行业价值观的试金石。
   - 案例：PNAS 2022 评 51 个候选化合物无证据起人类 pheromone 作用、VNO 已退化（T06-S016/S017）；「therapeutic-grade/CPTG」无监管定义、是单家公司注册商标（T06-S018）；连原版都因法规被 reformulate，dupe 多换合成替代、化学上做不到真等同（T06-S019）。

## 3. 工具栈与选型决策树

> 直接消化 Track 02。数量：必备 **6** / 场景特化 **8** / 新兴 **4**。本行最核心的「工具」不是软件，而是**原料调色板 (the palette) + 方法框架 (Jean Carles 法 / 香风琴 orgue) + 嗅觉记忆 (the nose)**；软件/仪器是辅助层。

- **必备层（6，业内口径约 80%+ 认真从业/学习者用）**：① 原料调色板（天然 EO/absolute/CO2 + 合成 aroma chemicals/captives，本行「字母表」）② 精密电子秤（0.001–0.01g，配方按重量可复现可缩放的前提）③ 闻香纸 mouillettes + 稀释方案 ④ TGSC 原料数据库（2972 种 aroma chemical 免费查，originator 自报）⑤ 配方管理（indie 轻量软件 ⇄ pro 企业 PLM）⑥ IFRA 合规计算器/配方校验。
- **场景特化层（8）**：indie 供应商套件、GC-MS/headspace、企业 PLM（Selerant/Coptis）、Fragrantica/Basenotes（查询非配方）、Michael Edwards《Fragrances of the World》+ Fragrance Wheel、稀释/陈化/称量套件、Jean Carles 法 + orgue（方法即工具）、评论 canon 作评估标尺（Turin & Sanchez）。
- **新兴层（4，Decay risk = high，12 月可能剧变）**：AI 配方生成（Carto/Philyra/Codex/Scentmate，闭源内部）、生物科技/发酵法可持续分子、AI 香气预测/数字嗅觉（Osmo POM，厂商自报约 53% 优于平均人类评审，T02-S026）、headspace 活体捕香 + AI 谱图识别。

**选型决策树（轴 = 目的 学习/创作/分析/合规 × 身份 pro/indie × 天然 vs 合成）**：
- 学习 → Jean Carles 法 + 训练 kit + TGSC + 闻香纸（嗅觉无书面记法，唯一路径是结构化重复闻料）。
- 创作·indie → 小量供应商 + 含 IFRA 检测的轻量软件 + 0.01g 秤 + maceration。
- 创作·pro → 公司调色板（含 captives）+ 企业 PLM + 内部 AI。
- 分析/逆向 → GC-MS + headspace（caveat：不是一键复制，痕量/captive 难定量）。
- 合规 → IFRA 计算器（标版本！）+ EU 致敏原校验（天然不豁免）。
- 天然 vs 合成 → 追可持续/珍稀替代看发酵法分子；坚持天然用天然供应商但别假设「天然=安全」。

**避坑清单**：天然≠安全/无致敏；别迷信「24h 持久」营销；note 金字塔≠配方；别跳过 IFRA/EU 校验；按重量精确计量（高效力分子先配约 10% 母液，T02-S037）；「therapeutic-grade」是营销标签；GC-MS 不是一键复制；别 indie 上企业 PLM/pro 用裸 Excel。
> last_checked = 2026-06-07。Decay risk：必备/场景层 low–medium，新兴层 high（AI/发酵 12 月内可能改写）。

## 4. 工作流 / Pipeline

> 直接消化 Track 03。真中枢 = B2B「core list → brief → creation → mods → win/lose」+ evaluator；与 niche/indie 直营流结构性对立（大师横跨两端）。

### 4.1 入门 SOP：一支最小完整香水（brief→定稿）

- **入门链路**：自定命题 → 选小调色板（查 TGSC）→ 搭 accord 和弦（起步只滴几滴、逐步加量、边调边试、记录配比）→ 排 top/heart/base 结构 %（caveat：parfum ~20–30% / EDP ~15–20% / EDT ~5–15% / EDC ~2–5%，约定俗成各家不同，无强制标准，T03-S008）→ trials/mods 按重量迭代 → 稀释/陈化（maceration ~1 月起）→ 闻香纸+皮肤+旁人评香 → IFRA/致敏原校验 → 定稿可缩放公式。
- **资深路径差异**：① **跳过**盲目试错——凭嗅觉记忆直接从 orgue 起手、在脑中构想目标再翻译成配方（Kurkdjian「不是混原料，是心智构建」）；② **优化**收敛速度——心算挥发度金字塔、按比例搭而非凭猜、更早知道「何时停手」（好 evaluator 让人更自信地停）；③ **额外**复用——从 accord 库/performing formula 复用而非从零（尤其 brief 模糊时）。caveat：与「每支全新创作」的浪漫叙事有张力，业内真相更接近「复用 + 微调」(T03-S005)。
- last_checked = 2026-06-08。Decay risk：low（入门链路是行业地基）。

### 4.2 B2B 竞标循环：core list → brief → creation → submission → mods → win/lose

- **机制**：品牌只把 brief 发给 core list 上 3–4 家（业内估，各品牌不同）accredited 香精公司；每家内部 evaluator 派多名 perfumer 竞争同一 brief；客户每家保留 1–2 提案，在 AD/panel 指导下反复 mods、回看 brief+moodboard；多国消费者测试定最终赢家、赢者负责生产。开发期不收费、表现差次年被剔出 core list。
- **资深/组织差异**：① **跳过**——资深 perfumer 复用 performing formula 骨架、twist 成贴 brief，**跳过**从零起步；② **优化**——公司内部「互相把各自 accord 喂进同一配方」帮彼此、做最 consensus 的配方以**优化**赢率；③ **额外**——evaluator 这个**额外**中间角色（1970s 末引入）解码 brief、每日嗅评、带回客观性、组织 panel test、把客户反馈回灌。
- **矛盾保留**：此工业真相与「每支独一无二艺术创作」叙事、与 niche carte blanche 并存不调和。Decay risk：medium（机制极稳，但 AI 正改写 creation 段速度）。last_checked = 2026-06-08。

### 4.3 niche/indie 直营创作流

- **机制**：从灵感（记忆/地点/情绪）起而非外部 brief；自供小批稀有原料、亲手称量/混合/灌装、直营 DTC 讲故事、不进 core list 不竞标；单支常投入超一年。IAO「artisan」三条件：本人写初始配方 + 拥/共有品牌 + 保有配方法律所有权。
- **相对 B2B 的差异**：① **跳过**——**跳过**竞标/盲测/consensus 妥协；② **优化**——把周期与成本约束**优化**掉（carte blanche、无 muse 无市场指令），重心全放嗅觉本身；③ **额外**——**额外**承担产能/分销/IFRA 合规的天然劣势（个人买不到 captive、规模化难）。
- **caveat**：indie「自由」叙事含营销成分；大师横跨两端（同一人不同约束切换）。Decay risk：low。last_checked = 2026-06-08。

### 4.4「the nose」tacit 校准 SOP（Jean Carles 三阶段，无捷径）

- **三阶段**：① 盲嗅背料——精选必备原料放闻香纸盲嗅，日复一日直到毫不犹豫即刻识别每个 note、区分嗅觉相近者；② 相似→对比——先按相似性闻（熟悉主题层次）再按对比闻（扩记忆容量），经典做法 60 个 odorants 按 class 排表；③ deformulation 还原——给一支成品只凭鼻子重建它、做几十个 trials 逼近 target。配 orgue（500–1500 瓶料半圆环绕，约定俗成）。
- **资深差异**：① **跳过**——资深者已内化上千料、**跳过**逐个查证直接识别；② **优化**——每天半小时雷打不动维持校准、**优化**长期记忆稳定性；③ **额外**——AI/数字嗅觉可作**额外**辅助但不替代人鼻（没有捷径替代反复闻料）。
- last_checked = 2026-06-08。Decay risk：low（百年量级方法论）。

### 4.5 IFRA / 致敏原合规 + reformulation（decay 较快层）

- **合规闸门**：按 IFRA Standards（Prohibition/Restriction/Specification）逐料算最大用量、按 product category 算累积暴露（QRA2）；当前 IFRA 51st Amendment（caveat：限值随版本变，引用必标版本号）；EU 2023/1545 致敏原约 26→80+、新品强制 2026-07-31/旧品退市 2028-07-31。
- **reformulation 差异**：① **跳过**——不能**跳过**重过 IFRA/EU 校验 + 稳定性测试 + 评香（这是硬约束）；② **优化**——找替代/重构料尽量保留原香性格（Guerlain 自育并专利符合 IFRA 的 oakmoss 品种，据称花至少 10 年，厂商自报）；③ **额外**——每出新 Amendment 触发**额外**一轮经典配方复核重配（oakmoss/atranol 是历史范本）。
- **张力**：创作自由/经典完整性 ⇄ 安全合规结构性矛盾；vintage vs reformulation 是社区高频议题（低-atranol 替代常挥发更快、丧失定香，故「持久」感下降）。Decay risk：medium–high（EU 2026-07 落地持续触发）。last_checked = 2026-06-08。

### 4.6 近期变化（decay 最快，Decay risk = high）

- **AI 配方生成**：嵌在 brief→creation 段加速候选生成（不在最终评香/定稿）；四大内部闭源（Carto/Philyra/Codex/Scentmate），indie 几乎用不到。① **跳过**新手部分试错、② **优化**迭代速度、③ **额外**引 perfumer 走「意想不到的原料配对」。
- **发酵法/headspace**：designer yeast 发酵造珍稀分子改写选型；headspace 活体捕香 + AI 谱图识别捕「不可萃取」的鲜花香。① 可**跳过**部分天然采购约束、② **优化**可持续/留香、③ **额外**带来「是否算天然」的 ISO 9235 合规**额外**争议。
- last_checked = 2026-06-08。Decay risk：high（业内估 >40%/12 月，闭源使外部追踪难）。

## 5. 表达 DNA

> **DNA 特征**：法语词根深嵌（accord / sillage / nez / orgue / mouillette / soliflore / extrait）+ 严格区分「感知 vs 配方」+ 浓度词不是气味词 + 合规反射（提限值必带「IFRA 第 N 修订版」）+ 祛魅（反 pheromone/天然神话/裸分崇拜）。说话像内行 = 会区分 sillage（移动尾迹）/projection（静止扩散）/longevity（时长）、用 extrait 而非「parfum 泛指」、把 dry-down 当真身。
> voice_confidence = **medium**：核心 figure 原话几乎全为英/法文，中文样本为忠实转译标 `转述`；Grojsman/Aftel/Malle/Edwards/Frolova 五位逐字原话因 bot 墙/403/会员墙未直接取回（标 `转述/推断`）。诚实边界已明示，无法谎标 `原话`。

### 5.1 客户版（面客解释 / 科普）

- 「我创作时追求微妙、干净、清脆、轻盈、清新、诱人的香气，它们必须唤起一种永恒的优雅与安静的精致。」(source: T01-S004, 转述，Ellena 谈极简美学)
- 「调香师是创造、想象、思考的人——他有能力在脑海里构想香味，再把它翻译成某种可触可感的东西。」(source: T01-S013, 转述，Kurkdjian 定义 perfumer)
- 「一种原料的价值不由价格衡量，而由稀有度衡量；我在突尼斯 Nabeul 亲自蒸馏的那种橙花，启发我去做一支香。」(source: T01-S024, 转述，Wasser 谈采料溯源)

### 5.2 同业版（私下 / 内训 / 创作自述）

- 「当我开始做一件东西时，我不知道最后会是什么。完全不知道。我毫无概念。」(source: T01-S021, 转述，Ropion 谈不预设终点)
- 「这不是混合原料。这里有一个心智构建。当你做一支香水，你有一个故事要讲。」(source: T01-S013, 转述，Kurkdjian)
- 「我通常用 4 到 7 个原料造一个主和弦，由它传达整支香的本质，其余都是围绕它的支撑。」(source: T01-S031, 推断，Grojsman 技法综合，逐字受 bot 墙阻拦)

### 5.3 专业 / 理论版（谈标准 / 美学 / 法规）

- 「一支美的香水，是给我们一记震撼的香水：先是感官的震撼，紧接着是心理的震撼。」(source: T01-S046, 转述，Roudnitska 著作原话中译)
- 「我不喜欢『overdose』这个词，因为一个配方的美有时恰恰在于某种原料的高剂量，而它没有失去平衡。」(source: T01-S022, 转述，Ropion 谈 overdose)
- 「香精行业是我见过最蠢的行业之一，它几乎独一无二地建立在『把顾客当白痴』这个前提上。」(source: T01-S006, 转述，Turin 的尖刻批评 register)

### 5.4 反例版（资深人绝不会这样说 / 被拒绝的营销话术）

- 「这款 pheromone 香水能吸引异性。」(source: T06-S016, 转述，反例：人类无确证 pheromone、VNO 已退化、PNAS 2022 评 51 化合物无证据)
- 「我们的精油是 therapeutic-grade、纯天然所以安全无敏、保证 24 小时持久。」(source: T06-S018, 转述，反例：therapeutic-grade 无监管定义、天然 EO 含高致敏成分、24h 是营销非配方事实)
- 「这是几十年没变的原版真配方，我们的 dupe 和原版一模一样。」(source: T06-S019, 转述，反例：多数经典已被 reformulate、dupe 多换合成替代、化学上做不到真等同)

## 6. 质量基准 + 反模式

**什么算「好」（3–5 条可验证基准）**：
1. **结构清晰且平衡**：能识别核心 accord 与挥发度结构，overdose 是有意承重而非失衡散架。
2. **经得起时间与 dry-down**：陈化后评、追到尾调仍成立，而非只在前调讨巧。
3. **合规且诚实**：过 IFRA/EU 闸门、不靠「天然=安全/24h 持久/pheromone」营销话术撑场。
4. **幻觉可信**：用分子制造的嗅觉意象比写实复刻更有说服力（fine fragrance 语境）。
5. **作者性清楚**：在所处约束（B2B/niche）下做出该约束能允许的最好表达。

**反模式（外行/入门常犯）**：① 把 note 金字塔当配方比例；② 只闻前 5 分钟就评判、忽略 dry-down；③ 「留香久=品质好」「闻起来像 cologne」（cologne 是浓度不是气味/性别）；④ 以为「天然=安全/低敏」；⑤ 信 pheromone/therapeutic-grade/24h/dupe=identical 营销；⑥ 把 overdose 当「超量失误」；⑦ 体积/滴数计量（不可复现）、跳过 maceration 就判留香；⑧ 跳过 IFRA/EU 校验或用旧版限值；⑨ 把 GC-MS 当「一键复制」；⑩ 身份错配工具（indie 上企业 PLM / pro 用裸 Excel）。

## 7. 智识谱系

**主干谱系（奠基 → 当代）**：
- **A 方法论派（Jean Carles 学派）**：Jean Carles（1946 创 Roure/Givaudan 学校 + coeurs 预制基 + 自底向上法）→ Calice Becker（更新主理 Givaudan 学校）、Monique Schlienger（弟子，创 Cinquième Sens）。
- **B 美学/极简派（法系艺术派）**：Edmond Roudnitska（Diorissimo；著 Le Parfum / L'Esthétique，立「香水=艺术」）→ Jean-Claude Ellena（Hermès，「两个原料就停手」）、Christine Nagel（接棒）。
- **C 分类/批评派**：G. W. Septimus Piesse（1857 创「notes」概念）→ Michael Edwards（1992 Fragrance Wheel / Fragrances of the World）；批评线 Luca Turin & Tania Sanchez（《Perfumes: The Guide》确立香评文体）+ Chandler Burr（业内纪实 + MAD 策展）。
- **D 香料化学派**：Paul Jellinek（心理-生理基础 1951）→ Charles Sell（RSC 化学）/ Günther Ohloff（分子级，原德文）。
- **天然派**：Mandy Aftel（《Essence and Alchemy》）、Andy Tauer（化学家自学 indie 范式）。

**核心分歧（保留不抹平）**：
1. **嗅觉科学**：振动理论（Turin，**少数派**）vs 形状/受体理论（主流；2004 Keller-Vosshall 在 Nature Neuroscience 反驳）——保留分歧不站队，振动论不当主流写。
2. **天然纯粹派 vs 合成实用派**：留香/稳定/IFRA 致敏监管 ⇄「天然=更真/更安全」信念（后者被科学质疑）。
3. **niche/haute parfumerie 艺术派 vs mass designer 商业派**：carte blanche/不盲测 ⇄ 爆款逻辑/市场调研（多数大师横跨）。
4. **「the nose=艺术家」vs「调香师=气味工程师」**：法系传统美学语（accord/sillage/orgue/soliflore）与分子派工程语（captive/aroma chemical/Hedione/Iso E Super）共用一套词表。
5. **学院开放培养 vs 四大闭源精英培养**：ISIPCA/GIP 对外开放、1200h ⇄ Givaudan 仅员工、年收个位数、训 4 年。
6. **极简（Ellena/Roudnitska/Laurent）vs 繁复/最大化（Grojsman 的 Hug Me、Cresp 的 gourmand）**。

## 8. 诚实边界

1. **信息截止 2026-06-07-08，工具/工作流/法规衰减最快**：新兴层（AI 配方生成/发酵分子/数字嗅觉）Decay risk = high，12 月内可能改写；**EU 2023/1545 致敏原新品强制 2026-07-31 正在本 skill 时点落地**（最高优先级时效项，触发大规模 reformulation 与标签更新），建议每 3–6 月 update 工具/工作流/法规节。
2. **闭源主导，公开一手稀薄**：核心创作 know-how 藏在四大香精公司（Givaudan/DSM-Firmenich/IFF/Symrise）内部学校 + 调香师个人脑中；真实创作 SOP（墙内 brief→mods 实操、内部评香会、内部学校课表）几乎无公开一手。figure 的「一手」几乎全是其对外出版物（自著书/官方页/长访谈）而非内部工作实录，故心智模型在「figures 一致性」维度比开放行业弱，claim 需二次校验。
3. **en+fr-primary，中文一手结构性缺失（约 30:1）**：原创一手 canon 全是英/法/德文（Roudnitska 法、Ohloff 德、Sell/Turin 英）；中文世界 = 译本（Ellena 有中译）+ 少数应用型工艺教材 + KOL/社区测评二手，且几乎全沉淀在黑名单平台（知乎/公众号/豆瓣 UGC）无法以合规 URL 进 manifest。stay-current 需读英文（核心）+ 法文（文化深度），中文用户存在结构性语言门槛。
4. **voice 维度 medium（结构性，非疏漏）**：核心 figure 原话几乎全为英/法文，中文 voice 库为忠实转译标 `转述`；Grojsman/Aftel/Malle/Edwards/Frolova 五位逐字原话受 bot 墙/403/会员墙阻拦未直接取回——硬标「原话」即 fabrication，故 §5 多为 `转述`、voice_confidence = medium。
5. **数字多为厂商自报/平台口径/约定俗成/法规版本**：市场份额、四大「约 1/3 市场」、浓度分级 %、原料阈值、留香小时、竞标命中率、Givaudan「年收约 3 人」均为业内估/厂商自述/约定俗成各家不同；IFRA 限值随修订版变、引用须标版本——synthesis 已逐处挂 source_id 或 caveat，引用时务必保留。
6. **嗅觉科学有真实未决争议**：振动理论 vs 形状/受体理论未平，Turin 是少数派——不站队、不把振动论当主流。
7. **一手厚度靠 surrogate + 个人源易衰减**：本行「first-hand」（约 51%，业内 surrogate 口径）本质是香评人自有 blog/podcast + 会议/协会官方 + 调香师自著书，无 arXiv/标准机构式客观一手；多数高价值源是个人源（Bois de Jasmin 迁 Substack、Candy Perfume Boy 转 podcast、On the Nose 已 hiatus），需每 6–12 月复查存续/迁移与付费墙变化。

## 9. Agentic Protocol — 研究维度

### 9.1 原料 / accord 定位

- **看什么**：用户问到的某个原料/香气是什么气味/强度/挥发度，归哪个家族，是天然还是合成、是否 captive，能否构成或属于某个 accord。
- **在哪看**：原料查 TGSC（thegoodscentscompany.com，气味/CAS/IFRA/用量）；成品香型坐标查 Michael Edwards《Fragrances of the World》+ Fragrance Wheel（14 家族）；accord 概念查 canon（T04-S040）。**不要**用 Fragrantica note 金字塔当配方依据。
- **输出格式**：1–2 句「该料/香气 = {家族}，{天然/合成/captive 状态}，挥发度落 {top/heart/base}，典型用于 {accord/场景}」+ 来源。

### 9.2 IFRA / EU 合规版本核对

- **看什么**：某原料在目标品类（如 Cat 4 fine fragrance / Cat 12 candle）的现行限用浓度、是否被 Prohibition/Restriction、是否触发 EU 致敏原申报。
- **在哪看**：IFRA Standards 官方（标明当前 Amendment 版本号，currently 51st）+ EUR-Lex Reg (EU) 2023/1545 原文；执行靠内建 IFRA 检测的配方软件。caveat：限值随版本变、天然不豁免。
- **输出格式**：「按 IFRA 第 N 修订版，{料} 在 {category} 限 {值/状态}；EU 2023/1545 下{是否需申报}」+ 版本号 + 来源（绝不用记忆里的旧限值）。

### 9.3 创作约束语境定位（B2B brief ⇄ niche carte blanche）

- **看什么**：用户/作品处在哪种创作世界——商业命题创作（core list 竞标、evaluator、consensus、复用 performing formula）还是 niche/indie 直营（从灵感起、carte blanche、自持配方）。
- **在哪看**：Track 03 工作流（B2B 模型 T03-S005 / IAO artisan 三条件 T03-S027）；判断信号：有无外部 brief、是否盲测、是否进 core list、是否自持品牌。
- **输出格式**：「此为 {B2B / niche} 语境，决策受 {该约束} 支配；同一大师在另一端会 {如何不同}」——用约束解释行为，不用才华高低。

### 9.4 天然 vs 合成 + 可持续选型

- **看什么**：该需求该用天然还是合成/发酵分子；可持续/濒危/气候约束下有无发酵法替代（檀香/ambrox/patchouli）；替代分子是否算「天然」。
- **在哪看**：Track 02 新兴层 #16（发酵法 Firsantol/Clearwood Prisma）+ ISO 9235「天然」口径 + 目标市场标签法规；天然安全性交叉 NCBI/ACHS（天然≠无致敏）。
- **输出格式**：「{天然/合成/发酵} 路线，理由 {成本/可持续/留香/合规}；注意 {ISO 9235 是否算天然 / 天然≠安全} 的争议」+ caveat（商业化里程碑多为厂商自报）。

### 9.5 流派 / figure 视角定位

- **看什么**：用户的问题落在哪个学派——方法论（Jean Carles）、极简美学（Roudnitska/Ellena）、overdose 技法（Ropion/Grojsman）、分类（Edwards）、批评（Turin/Burr）、天然派（Aftel/Tauer）。
- **在哪看**：Track 01 figures + Track 04 谱系（§7 智识谱系）；用「这个问题谁会怎么答」反推视角。
- **输出格式**：「此问题主要是 {学派} 的地盘，代表 {figure} 会主张 {立场}；对立派 {figure} 会反驳 {点}」——给出分歧而非和稀泥。

### 9.6 祛魅校验：营销话术 vs 配方真相

- **看什么**：用户引用的说法是否是被资深从业者拒绝的营销话术（pheromone 吸引/therapeutic-grade/all-natural=safe/clean·non-toxic/24h lasting/dupe=identical/niche=always better）。
- **在哪看**：Track 06「厂商话术拒绝」7 条 + outsider-tell top 12，每条配机理反驳（PNAS pheromone 研究、CPTG 商标、reformulation 事实）。
- **输出格式**：「{该说法} 是营销话术，机理上 {一句反驳}；内行实际意思是 {正解}」——祛魅 + 给正确替代认知。

### 9.7 时效 / reformulation 风险 + stay-current
