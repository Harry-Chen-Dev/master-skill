### 🅰 组：AI 写作 / 提示词方法论布道者（与本行业「怎么写」最直接相关，蒸馏价值最高）

#### A1. 宝玉 @dotey —「翻译即重写」
- **核心一句话**：把「提示词工程」变成人人可抄的写作手法——独特之处不在懂 AI,而在把「如何用 AI 把英文技术内容变成读起来像母语作者写的中文」提炼成方法论。本质是技术内容的二次创作翻译家 + 知识再加工工程师,核心动作=**降低理解门槛**(硬核论文→大白话+类比)。
- **文案/内容生产手法**(招牌一手 [T01-S028,S029])：
  1. **翻译心智模型(类比开场)**：把翻译想成「校长协调两个老师」——英文好中文差的老师先**直译**(忠实但生硬)→中文好的老师**意译重写**(通顺)→校长融合。用这个类比向不懂技术者解释 prompt。
  2. **方法三阶段(自总结)**：提示词时代→推理模型时代→Agent时代；从「直译→反思→意译」演进到「一英文老师直译+两中文老师不同风格意译+校长融合」,最新做成 `baoyu-translate` skill(快/标/精三档,长文自动拆分并行翻译) [T01-S030]。
  3. **硬规则写死在 prompt**：保留专有名词/术语/引用标号/图表标签/Markdown,全角括号转半角加空格,术语表映射(LLM→大语言模型)。
  4. **选题/声音**：「知识翻译者」定位,紧盯 Anthropic 工程博客；开场常用「今天给朋友解释…/举个例子」口语化降门槛；**诚实标注来源**(「我不是原创,从这里学到的」)。
- **代表作品**：X《一个简单的Prompt大幅提升ChatGPT翻译质量》系列(流传最广,平台原生一手) / baoyu.io《自用"科技文章翻译GPT"和它的Prompt》[T01-S028] / GitHub `baoyu-skills` [T01-S030] / 极客时间《软件工程之美》。
- **3 件值得读/看**：① 《翻译GPT和它的Prompt》(写作手法最浓缩) ② 「翻译三阶段」帖 + `baoyu-translate` skill(看手法如何从一句prompt工程化为Agent) ③ 《为什么我不"凭感觉编程"》(2026-05,展示不追风口、强调spec/验证驱动的人格)。
- **争议**：无实质丑闻。轻微两面——有人视其为「搬运工」(大量翻译/转述英文一手),另一面他主动反复标来源、强调字幕翻译是爱好非生意,属定位非争议。
- **最近12个月**：高频追踪 Claude Code/Codex/Agent编程；把翻译法产品化为 `baoyu-translate` 并开源 `baoyu-skills`；2026出新书放样章；博客活跃(《为啥Codex还不推出Codex Design?》2026-06-13)。

#### A2. 李继刚 —「Prompt之神」/ 概念压缩
- **核心一句话**：把整本哲学浓缩成一个词,再让模型解压成文章——做的是「**概念压缩**」,Lisp 括号只是审美外壳；卖的不是 prompt,是「读得多、想得深」的人设和它的可复制外形。
- **文案/内容生产手法**：
  1. **场域式开头,不堆指令**(一手 [T01-S021])：「分配角色时实际是在构建一个'场域',注入巨大信息量」——给模型一个身份/世界而非步骤清单。
  2. **极致压缩是核心纪律**(一手 [T01-S020])：自述用 Lisp 写 prompt 唯一感受是「压缩…直到用一个词汇表达它」；反复打磨(输出→换词→再看),用精准概念词(如「尼采的生命意志」)当解压钥匙触发模型 latent knowledge。
  3. **Lisp 作视觉签名**：`;;` 注释头标作者/版本/模型,嵌套 `(defun…)` 组织 Role/Profile/Goals/Constraints/Workflow；自辩十年 Emacs+Lisp 契合「压缩」哲学。
  4. **鲁迅/林语堂式辛辣人设声音**：成名作「**汉语新解**」指定风格=隐喻、一针见血、辛辣讽刺、抓本质,输出配 SVG 卡片(文字洞察+视觉成品双交付=爆火关键)。
  5. **统一「读书人」品牌叙事**：站点 tagline「你要对这个世界有点意见」,GitHub bio「Read→Think→Write→Publish」。
- **代表作品**：汉语新解(2024,即刻+GitHub,成名作) / 公文笔杆子(2023,压缩转向前早期爆款) / `ljg-skills`(5.9k★,现役主力,已转 Claude Code Skills [T01-S018]) / 《Prompt道与术》(原文在公众号黑名单)。
- **3 件值得读/听**：① GitHub `ljg-skills`(看 prompt→Skills 演进) ② 澎湃《专访"Prompt之神"》[T01-S021](最完整方法论自述) ③ 小宇宙《压缩之道》(本人语音)。
