## 流派分歧矩阵（framework 甜区 — 保留分歧不软化）

> 5 轴。每轴：主张 / 代表(引 T01) / 适用 / **核心分歧**。平台规则(T01-S008/T04-S006)整体站「原创精做+真人+辅助+标识」一边，但**批量派在变现速度上仍有现实优势**——分歧未被制度性消灭，只是被压制，故全部保留。

### 轴 1：图文 vs 短视频

| | 图文派 | 短视频派 |
|---|--------|----------|
| **主张** | 干货/清单/对比靠多页图文+大字报，制作轻、可收藏、搜索长尾强 | 口播/教程/测评靠视频，完播+停留权重高、信息密度+信任感强 |
| **代表** | 那珂(工具整理笔记)、灵琪(AI绘画图)(T01-S002) | 卡兹克(实操+玩梗口播)、数字人口播号(T01-S009) |
| **适用** | 知识干货、工具合集、好物清单、SEO 长尾 | 教程演示、人设建立、强情绪种草、出海 |
| **核心分歧** | **收藏率 vs 完播率谁是北极星**：图文派盯收藏(存档价值)，视频派盯完播+停留。AI 时代图文制作成本更低→更易被批量同质化；视频更难批量但「活人感」更强 (evidence: [T04-S010, T06-S001]) |

### 轴 2：干货教程 vs 测评种草 vs 资讯搬运

| | 干货教程派 | 测评种草派 | 资讯搬运派 |
|---|-----------|-----------|-----------|
| **主张** | 给方法/信息增量，利他价值最大化，接 B 端广 | 痛点+对比+好物，转化导向，接品牌商单 | 蹭热点快产量大，靠时效抢流量 |
| **代表** | 歸藏(系统测评)、那珂(工具干货)(T01-S006/S002) | 数字人种草带货号、好物博主(T01-S003) | 碧根果批量起号实验(资讯/攻略)(T01-S001) |
| **适用** | 知识赛道、长期人设、B端变现 | 美妆/家居/数码带货 | 起号测方向、信息差垂类 |
| **核心分歧** | **信息增量 vs 流量套路**：干货派认为「利他=护城河」；搬运派认为「速度=护城河」。平台明确**严打搬运/洗稿**(自动判抄袭)，碧根果亲测批量起号「几百条同一流量密码反复发」→无差异化死穴 (evidence: [T01-S001, T02-S022]) |

### 轴 3：真人出镜 vs 数字人 / 无人起号

| | 真人出镜派 | 数字人/无人起号派 |
|---|-----------|------------------|
| **主张** | 真人IP+专业背书=信任与人设壁垒，长期可商业化 | 数字人省人力、可批量、不露脸，AI出形象+对口型量产 |
| **代表** | 卡兹克(真人IP)、灵琪(设计师背书)(T01-S009) | AYAYI/cyberAngel 等虚拟人号、AI口播号(T01-S003/S013) |
| **适用** | 人设号、知识IP、商单号 | 不愿露脸者、矩阵号、出海多语种(HeyGen) |
| **核心分歧** | **最尖锐之一**：数字人省事但**低质数字人+未标识=2026新规重点打击**(仿冒/AI托管号已处置80万+)；真人信任度高但产能低。无人起号「躺赚」叙事与平台「严打AI托管/自动养号」直接对撞 (evidence: [T01-S008, T02-S016, T04-S006]) |

### 轴 4：原创精做 vs 批量搬运洗稿

| | 原创精做派 | 批量搬运洗稿派 |
|---|-----------|---------------|
| **主张** | 单篇深做+真实素材+差异化，质量取胜、合规护城河 | 多账号矩阵、洗稿/搬运、量取胜，赌概率出爆款 |
| **代表** | 歸藏/卡兹克/勘云工造(T01-S007) | 1327矩阵号、无人起号课、批量营销号(T01-S003) |
| **适用** | 长期账号、商单、品牌合作 | 短期套利、广告分成、带货收割 |
| **核心分歧** | **最尖锐**：精做派引平台规则(原创度权重升至35%、严打重复内容)为据；批量派现实仍在跑量变现。分歧本质是**「平台规则」vs「套利窗口」的时间差**——规则在收紧，但窗口未完全关闭 (evidence: [T01-S008, T06-S001, T02-S020]) |

### 轴 5：AI 全自动 vs AI 辅助精做

| | AI 全自动派 | AI 辅助精做派 |
|---|------------|--------------|
| **主张** | 豆包写+即梦图+剪映成片一键量产，全链 AI、人只点按钮 | AI 只做选题/结构/初稿，正文「人话化」重写+真实封面 |
| **代表** | 无人起号课/AI托管号(T01-S015) | 碧根果反思后转辅助、歸藏强调差异化(T01-S001/S007) |
| **适用** | 起号测方向、资讯合集、量大不求精 | 人设号、长期主义、合规优先 |
| **核心分歧** | **AI 能用多少的边界**：平台**鼓励有「信息增量」的AI、严打「批量同质化」的AI**——但「增量 vs 同质」边界模糊，是 still-debated 灰区。全自动派赌「量大盖过限流」，辅助派赌「精做躲过限流+建人设」 (evidence: [T01-S008, T04-S007, T02-S020]) |

> **三个最尖锐分歧**（Phase 2 智识谱系优先）：① 原创精做 vs 批量搬运(轴4，平台规则 vs 套利窗口时间差)；② 真人 vs 数字人/无人起号(轴3，新规重点打击 vs 躺赚叙事直接对撞)；③ AI 全自动 vs 辅助精做(轴5，「信息增量 vs 同质化」边界模糊的灰区)。

## 关键差异对照

### A. 本 skill（AI 赛道博主·文案内容）vs 通用 `xiaohongshu-ops`（综合运营/带货）

| 维度 | 本 skill：AI 自媒体博主（文案+内容） | 通用 xiaohongshu-ops（综合运营/带货） |
|------|-----------------------------------|--------------------------------------|
| **北极星** | 内容质量/收藏率/活人感/爆款率(让笔记被看见) | GMV/转化/ROI/客单价(让东西卖出去) |
| **核心动作** | 写文案、做封面、定选题、控AI味、标AIGC | 选品、铺量、投流、私域、店播、客服 |
| **AI 的角色** | AI 是**生产工具+合规风险源**(写稿/出图/数字人，但要防AI味+必标识) | AI 是辅助(客服/选品/数据)，非内容主体 |
| **特有红线** | AIGC 标识、AI 味限流、洗稿撞稿、虚假人设 | 广告法、价格欺诗、刷单、虚假交易 |
| **变现路径** | 蒲公英商单 / B端广 / 知识付费 / 数字人带货 | 店铺/橱窗/直播带货/品牌代运营 |
| **关键工具** | 豆包/DeepSeek(写)+即梦/可图(图)+剪映(剪)+新红/千瓜(选题) | 千瓜(选品)+蒲公英+电商后台+投放工具 |
| **本质区别** | 解决「**怎么把内容做爆且合规**」，AIGC 是核心变量 | 解决「**怎么把货卖好**」，内容是带货手段 (evidence: [T02-S020, T04-S014]) |

### B. 小红书 vs 抖音 vs 公众号 — 内容机制差异（决定文案/制作策略截然不同）

| 机制 | 小红书 | 抖音 | 公众号 |
|------|--------|------|--------|
| **核心指标** | **收藏率+搜索**(CES：关注×8>评/转×4>赞/藏×1) | **完播率+停留**(前3秒权重~40%，行为概率×价值权重) | 打开率+在看+转发(社交分发) |
| **流量来源** | 搜索 60-70% + 推荐(搜推互通) | 推荐为主(兴趣/行为算法) | 私域订阅+朋友圈/在看社交链 |
| **内容寿命** | **长尾强**：2-3月前优质笔记仍靠搜索来量 | 短：靠即时推荐，过峰即沉 | 中：靠转发二次扩散，无搜索长尾 |
| **内容形式适配** | 双排瀑布流，图文+短视频均强，封面定生死 | 单列沉浸视频，前3秒定生死 | 长图文推送，不适合短图文 |
| **文案重心** | SEO 关键词布局+利他+收藏诱导+封面大字报 | 钩子+节奏+完播设计+评论引导 | 标题党+深度+情感共鸣+转发钩子 |
| **AI 博主含义** | 选题先想「用户会搜什么词」；做「值得存档」的内容 | 选题想「3秒怎么留住」；做「忍不住看完」的内容 | 做「忍不住转发给朋友」的内容 (evidence: [T06-S001, T06-S007, T06-S008, T04-S008]) |

> **一句话**：小红书是「**搜索引擎化的内容社区**」——收藏与搜索是命脉；抖音是「兴趣推荐流」——完播是命脉；公众号是「社交订阅流」——转发是命脉。同一篇 AI 文案搬三平台必死，机制不同决定文案 DNA 不同 (evidence: [T06-S008, T04-S008])。
