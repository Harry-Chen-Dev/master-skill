### <a name="1"></a>1. 正面管教方法论体系 (Positive Discipline)

- **One-liner**: 基于阿德勒个体心理学的非惩罚、非溺爱管教框架，全球68个国家和地区主流家庭教育方法论
- **Tier**: 必备
- **类型**: 方法论体系
- **开发者/维护者**: Jane Nelsen & Lynn Lott（美国）；中国授权机构：北京天略匹迪教育咨询有限公司
- **适用场景**: 2-18岁儿童行为引导；家长工作坊教学；学校-家庭协同管教
- **不适合/替代方案**: 婴幼儿（0-2岁）适用性弱，需配合依恋理论；极端行为障碍需专业ABA干预
- **实践案例**: 北京天略匹迪经认证授权，在中国大陆开设家长讲师班和学校讲师班，培训达百万人次；覆盖27张工具卡，包括家庭会议、鼓励代替惩罚、自然后果等
- **中国本土化情况**: 2000年代引入，由天略匹迪独家运营认证；书籍《正面管教》（北方妇女儿童出版社）为樊登读书播放量第一亲子书；本土化争议：部分内容与中国集体主义文化存在张力，强调"班级会议"在应试背景下落地困难
- **争议/局限**: 仅天略匹迪一家拥有中国大陆认证授权，形成垄断；认证费用较高，市场上存在大量未授权"正面管教"培训；循证基础相对薄弱（非RCT研究为主）
- **来源**: evidence: [T02-S001] [T02-S002]
- **可信度**: high

### <a name="2"></a>2. PET 父母效能训练 (Parent Effectiveness Training)

- **One-liner**: Thomas Gordon创立的以沟通技术为核心的父母培训体系，全球400万册销量，强调无输赢冲突解决
- **Tier**: 必备
- **类型**: 方法论体系
- **开发者/维护者**: Thomas Gordon（1962年）；Gordon Training International（1974年）；中国大陆授权运营商
- **适用场景**: 亲子沟通障碍改善；青春期冲突化解；家庭教育指导师进阶培训
- **不适合/替代方案**: 学龄前幼儿效果有限；需要父母有较高语言表达能力和自我觉察；家庭冲突严重时需配合家庭治疗
- **实践案例**: 中科院教育科研成果：小学"父母效能训练"课程开发与实施项目，四川华西医院将PET纳入ADHD父母训练营课程体系
- **中国本土化情况**: 工作坊、读书会、讲座形式推广；中国大陆授权运营蓬勃发展，工作坊覆盖人群达百万人次；中文译本由中国人民大学出版社出版
- **争议/局限**: Thomas Gordon强调彻底平等对话，部分中国家长认为与传统权威教育理念冲突大；国内PET讲师质量参差不齐；缺乏大规模中国本土循证研究
- **来源**: evidence: [T02-S020]
- **可信度**: medium

---

### <a name="3"></a>3. 非暴力沟通 NVC (Nonviolent Communication)

- **One-liner**: Marshall Rosenberg创立的四步骤沟通框架（观察-感受-需要-请求），减少评判式语言，强化情感连接
- **Tier**: 必备
- **类型**: 咨询框架 / 方法论体系
- **开发者/维护者**: Marshall B. Rosenberg；Center for Nonviolent Communication (CNVC)
- **适用场景**: 亲子情绪冲突调解；家庭教育指导师个案沟通；隔代教育摩擦化解
- **不适合/替代方案**: 急性危机（家暴、严重对抗）场合不适用；需要相当练习才能自然运用；认知能力低的年龄段儿童难以配合
- **实践案例**: 中国救助儿童会香港将NVC四步骤整合进家长课堂；多所学校将NVC引入班会和家庭教育讲座
- **中国本土化情况**: 中文版《非暴力沟通》（华夏出版社）传播广泛；CNVC官网提供中文资源；国内认证培训师数量增长，但尚无系统中国本土化研究
- **争议/局限**: 批评者认为NVC话语模式在高情境文化中显得刻板；缺乏中国大规模RCT研究；部分培训机构以NVC为噱头进行过度商业化包装
- **来源**: evidence: [T02-S014] [T02-S015] [T02-S023]
- **可信度**: medium

---

### <a name="4"></a>4. 积极倾听与 I-message 技术

- **One-liner**: PET框架下的两大核心沟通工具：积极倾听解决孩子问题，I-message解决家长问题，合用降低亲子对抗
- **Tier**: 必备
- **类型**: 咨询框架
- **开发者/维护者**: Thomas Gordon；与NVC的感受-需要表达高度兼容
- **适用场景**: 孩子情绪宣泄场景（积极倾听）；家长表达被侵犯感受场景（I-message）；亲子每日沟通习惯培养
- **不适合/替代方案**: I-message在多动症/ASD儿童中效果有限；需配合行为干预；不能替代结构化行为分析
- **实践案例**: 广东、浙江多所中小学家庭教育讲座将积极倾听和I-message列为核心教学内容；华西医院ADHD父母训练营纳入I-message模块
- **来源**: evidence: [T02-S020]
- **可信度**: medium

### <a name="5"></a>5. 行为 ABC 分析框架 (Applied Behavior Analysis Simplified)

- **One-liner**: 前因(Antecedent)-行为(Behavior)-后果(Consequence)三段式行为分析，将ABA核心工具简化为家长可用的行为干预框架
- **Tier**: 必备
- **类型**: 方法论体系 / 评估工具
- **开发者/维护者**: B.F. Skinner 行为主义理论；ABA由Baer, Wolf & Risley 1968年系统化；简化版家庭应用由Autism Speaks等机构推广
- **适用场景**: 儿童行为问题分析（攻击性、发脾气、拒绝上学）；ADHD/ASD家庭行为管理；正向强化设计
- **不适合/替代方案**: 复杂行为功能评估需专业BCaBA/BCBA；不适合情绪发泄型行为的一线干预；单独使用忽视情绪维度
- **实践案例**: Autism Speaks《ABA家长指南》中文版将ABC框架系统化呈现；多家康复机构（星空智程、东方启音）将ABC表作为家长培训核心工具
- **争议/局限**: 简化版容易被误用（过度强化/强化错误行为）；与NVC/正面管教的情感导向存在哲学张力；ABA在中国主要被用于自闭症领域，一般家庭教育使用率偏低
- **来源**: evidence: [T02-S025]
- **可信度**: medium

### <a name="6"></a>6. 家庭会议结构化流程 (Family Meeting)

- **One-liner**: 正面管教核心工具之一，定期全家民主议事流程，培养儿童责任感和家庭归属感
- **Tier**: 必备
- **类型**: 方法论体系 / 咨询框架
- **开发者/维护者**: Jane Nelsen（正面管教第13号工具卡）；阿德勒心理学家族起源（Dreikurs 1964年即提出）
- **适用场景**: 二孩或多孩家庭矛盾协调；青春期子女争取发言权场景；家庭规则制定
- **不适合/替代方案**: 婴幼儿家庭无法参与；单亲家庭形式需调整；高冲突家庭需先建立安全感再启动
- **实践案例**: 北京天略匹迪官网发布"有效家庭会议9步骤"作为家长实操指南；澎湃新闻报道二孩家庭用家庭会议改善日常冲突案例
- **结构要素**: ①感谢与欣赏 ②上周决议回顾 ③议题讨论（轮流发言） ④头脑风暴解决方案 ⑤投票/共识 ⑥下周家庭活动计划
- **来源**: evidence: [T02-S001] [T02-S024]
- **可信度**: high

### <a name="7"></a>7. ECBI 艾伯格儿童行为量表 (Eyberg Child Behavior Inventory)

- **One-liner**: 专为2-16岁儿童品行问题筛查设计的父母报告量表，36题双维度（问题频率+困扰度），临床使用便捷
- **Tier**: 必备
- **类型**: 评估工具
- **开发者/维护者**: Sheila Eyberg；已有多国中文版本
- **适用场景**: 儿童行为障碍早期筛查；ADHD/ODD初步评估；家庭教育指导前基线测量
- **不适合/替代方案**: 仅靠父母报告存在偏差，需配合教师版（SESBI）；不适用于2岁以下；重度行为问题需临床诊断
- **争议/局限**: 中国本土化常模不完整；部分条目的中国文化适用性存疑
- **来源**: evidence: [T02-S016]
- **可信度**: medium

### <a name="8"></a>8. PSI 养育压力指数 (Parenting Stress Index)

- **One-liner**: 评估父母-儿童系统压力的权威量表，PSI-4含120题，SF版36题，已有中文版信效度验证
- **Tier**: 必备
- **类型**: 评估工具
- **开发者/维护者**: Richard Abidin（1983年）；Psychological Assessment Resources (PAR Inc.) 商业发行；PSI-4为最新版
- **适用场景**: 家庭教育指导前评估父母压力来源；特殊需求儿童家庭支持方案设计；研究与临床双用途
- **不适合/替代方案**: 商业授权版本，使用需购买；SF版精度低于全版；不直接评估儿童行为，需配合CBCL/SDQ
- **实践案例**: 中文版PSI信效度研究由Johns Hopkins大学团队完成（2002年PubMed发表）；PSI-SF-15精简版2021年Sage期刊发表，证实在中国父母群体中良好信效度
- **来源**: evidence: [T02-S003] [T02-S004] [T02-S005]
- **可信度**: high

### <a name="9"></a>9. CBCL Achenbach 儿童行为评定量表

- **One-liner**: 覆盖1.5-18岁的儿童心理行为综合评估量表，含父母版/教师版/自评版，是中国儿童心理临床最常用量表之一
- **Tier**: 必备
- **类型**: 评估工具
- **开发者/维护者**: Thomas M. Achenbach（1966年始创，持续更新）；ASEBA系统商业授权
- **适用场景**: 儿童情绪行为问题多维评估；学校-家庭联合筛查；临床诊断辅助工具
- **不适合/替代方案**: 版权限制，需授权使用；条目多（113-120题），耗时约20分钟；不适合作为单纯家庭教育指导工具，需专业人员解读
- **实践案例**: 国内多家三甲医院儿科/儿童心理科常规使用；家庭教育指导师使用时通常由合作机构的心理咨询师完成评估和解读
- **来源**: evidence: [T02-S016]
- **可信度**: high

### <a name="10"></a>10. SDQ 优势与困难问卷 (Strengths and Difficulties Questionnaire)

- **One-liner**: 简短25题的儿童心理健康筛查工具，强调优势维度，有父母/教师/自评三版，免费授权使用
- **Tier**: 必备
- **类型**: 评估工具
- **开发者/维护者**: Robert Goodman（1997年）；官网 sdqinfo.org 提供多语言免费下载
- **适用场景**: 家庭教育指导首次会谈快速筛查；学校心理健康普查；资源有限环境首选工具
- **不适合/替代方案**: 仅5个维度，不能替代CBCL做深度评估；适合筛查不适合诊断；情绪症状维度灵敏度相对低
- **实践案例**: 中科院心理研究所完成中文版系统心理测量综述，上海地区建立常模；多项学校心理健康研究以SDQ作为主要筛查工具
- **争议/局限**: 免费是最大优势，但国内研究显示部分因子结构与原版有差异；教师版在中国使用率低于父母版
- **来源**: evidence: [T02-S018]
- **可信度**: high

### <a name="12"></a>12. 依恋关系评估工具 (SSP / AAI)

- **One-liner**: 陌生情境测验(SSP)评估婴幼儿依恋类型，成人依恋访谈(AAI)评估父母依恋状态，两者共同揭示依恋代际传递
- **Tier**: 必备（高阶指导师）
- **类型**: 评估工具
- **开发者/维护者**: SSP: Mary Ainsworth（1978年）；AAI: George, Kaplan & Main（1984年）
- **适用场景**: 0-18个月依恋评估（SSP）；探索父母自身童年依恋对养育方式影响（AAI）；创伤知情型家庭教育
- **不适合/替代方案**: SSP需专业实验室设置，普通家庭教育指导师无法施测；AAI是复杂半结构化访谈，需专门培训；一般场景用依恋风格问卷(ECR)替代

- **争议/局限**: 专业门槛极高；SSP实验室设置成本大；AAI编码培训需2年以上；两者均非普通家庭教育指导师日常工具，更多用于心理治疗领域
- **来源**: evidence: [T02-S016]

### <a name="17"></a>17. 山寨证书避坑清单

- **One-liner**: 2021年人社部清理职业资格后家庭教育指导师认证市场严重混乱，教育部明确表示未颁发任何家庭教育指导师证书
- **Tier**: 认证体系
- **类型**: 认证体系（风险提示）
- **山寨特征识别**:

  - 以"450万人才缺口""年薪50万"为招生噱头
  - 交钱即发证，无课时要求，无考核
  - 证书颁发机构无法在民政部社会组织查询系统核实
- **相对正规机构（需自行核实最新动态）**: 中国家庭教育学会（学会证书）；中国婚姻家庭研究会（民政部登记，全国妇联主管）；正面管教PDA认证（可在positivediscipline.org查验）；ICF认证（可在ICF官网查验）
- **法律背景**: 《家庭教育促进法》2022年1月1日施行后，市场需求爆发式增长；2022年7月人社部正式将"家庭教育指导师"加入职业分类大典但仍无资格认定，各类培训机构趁机炒作
- **来源**: evidence: [T02-S007] [T02-S008] [T02-S009] [T02-S021]
- **可信度**: high（风险提示可信，具体机构现状需持续核实）
