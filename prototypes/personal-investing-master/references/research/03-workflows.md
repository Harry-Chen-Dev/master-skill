# 03-workflows.md — 个人投资理财 WORKFLOWS（入门 SOP + 资深路径 + 近期变化）

> **行业**: 个人投资理财 (Personal Investing & Financial Planning)，locale = **global**，profile = practitioner（自主投资者管理自己的组合，非持牌投顾展业）。
>
> **本 track 的核心交付**：不是教科书流程图，而是「**自主投资者怎么真正走完一个组合的搭建与管理**」的 SOP/方法论 —— 每个 workflow = 入门最少步骤 + 资深差异（跳过/优化/额外）+ 近期变化（带日期+触发事件）+ 失败模式自检。四大流派分叉：价值 / 被动指数 / 技术 / 趋势。
>
> **本行业 workflow 的结构性特征 = 多数稳态、少数高衰减**。组合管理的核心 SOP（配置→账户结构→选品→执行→再平衡→复盘）是几十年稳态的「**慢变量**」，衰减来自**外围**：税优账户限额年度微调、直接指数化下沉、robo/AI 投顾洗牌、CN 个人养老金 2024-12 全国推开、公募费率改革。**这是真实特征，不是研究不足**。
>
> **非投资建议**: 本文是认知 OS 素材（自主投资者「怎么操作」的流程），不构成任何具体证券 / 买卖时点 / 配置比例建议。所有数字（4% 法则 / 5-25 再平衡规则 / 税优限额 / 费率）均挂源或标 caveat，未编造。
>

### 1. 组合搭建与管理主工作流（端到端，最重要）

- **One-liner**: 从「我有钱想投资但不知从何下手」到「一个低成本、税务高效、自动化、能长期管住手的组合」—— 本行业**最核心、最高复用**的元工作流，价值/被动/技术/趋势四派都把它当地基（分歧只在「选品」一步往哪走）。
- **Trigger**: 有了可投资的钱（工资结余 / 一笔意外之财 / 换工作要 rollover 401k）+ 已搭好地基（应急金 3–6 个月、还掉高息债）→ 启动组合搭建。(evidence: [T03-S001, T03-S012])
- **Output**: 一个写下来的「投资政策」：目标股债比 + 用对税优账户装好 + 选定宽基/产品 + 设好自动定投 + 定好再平衡触发规则 + 行为纪律预案。(evidence: [T03-S001])
- **入门 SOP（minimum viable steps）**:
  1. **定目标 / 时间轴 / 风险承受度**：写下「为什么投、什么时候要用钱、跌 30% 会不会半夜割肉」。风险承受度 = 意愿(心理) × 能力(时间轴/收入稳定性) × 需求(目标)。(evidence: [T03-S002, T03-S012])
     - **跳过后果**：没有时间轴和风险底线 → 熊市底部恐慌清仓（损失厌恶 + 近因偏差），把长期策略毁在一次情绪操作上。这是散户头号亏损来源（→ T06 行为术语）。
  2. **定资产配置（股 / 债 / 现金比）**：先定大类权重（如 80/20、60/40），再谈具体标的。学术与实务共识：长期回报主要由配置而非选股/择时决定。常用启发：股票% ≈ 「110/120 − 年龄」或按风险承受度问卷，但**是启发不是定律**。(evidence: [T03-S002, T03-S012])
     - **跳过后果**：直接奔「买哪只」→ 拿一堆同涨同跌的标的，自以为分散实则零分散；或股债比与风险承受度不匹配，下跌时扛不住。
  3. **账户结构先于产品**（本工作流最被低估、资深最看重的一步）：先选对**税优容器**再谈买什么。
     - **US 税优填充顺序**（Bogleheads「Prioritizing investments」官方 SOP）：① 拿满 401(k)/403(b) **雇主匹配**（即时 100% 回报，错过 = 白扔钱）→ ② 还高息债 → ③ **HSA**（唯一三重免税：投入抵税/增长免税/医疗支出免税）→ ④ IRA / Roth IRA（费率更低、选择更多）→ ⑤ 顶满 401(k) → ⑥ taxable 应税账户。(evidence: [T03-S003, T03-S013])
     - **CN 税优顺序**：① 个人养老金 **Y 账户**（税延，每年限额内抵扣应纳税所得，领取时按 3% 计税，Y 份额费率优惠）→ ② 普通券商/基金账户。(evidence: [T03-S019, T03-S020])
     - **跳过后果**：把钱直接丢进 taxable 却没拿雇主匹配 → 等于主动放弃一份「保证 100% 回报」；账户类型选错（该 Roth 选了 Traditional 或反之）→ 几十年后多缴大量税。账户结构的税后杠杆常**大于**选基金。
  4. **选品（宽基指数 vs 主动）**：装进容器的产品。被动默认 = 低费率宽基（三基金：美股全市场 VTI + 国际 VXUS + 债 BND；或单只全球 VT；CN：沪深300/标普500 QDII + 宽基指基）。想主动选股 → 转 Workflow 2/4/5。(evidence: [T03-S004, T03-S010])
     - **跳过后果**：买了高费率主动基金/主题基金 → 费率复利侵蚀（1% vs 0.05% 在 30 年吞掉巨量终值）+ 大概率跑输基准（SPIVA: YE2025 美国 79% 大盘主动跑输 S&P 500）。(evidence: [T03-S021])
  5. **执行（定投 DCA vs 一次性 lump sum）**：有一大笔闲钱 → 研究显示因 time-in-market，多数历史区间 **lump-sum 期望回报 > 把同一笔钱分期 DCA**；但 DCA 降低后悔/情绪风险。靠工资持续投 = 天然定投。(evidence: [T03-S005])
     - **跳过后果**：迟迟不入场等「更好时点」（择时）→ 踏空（市场最佳交易日常紧贴最差日，错过少数几天重创长期收益）。
  6. **再平衡（阈值 / 日历）**：设触发规则把偏离的权重拉回（卖涨买跌）。详见 Workflow 6。(evidence: [T03-S006])
     - **跳过后果**：牛市后股票占比越滚越高，风险敞口悄悄翻倍，下一次回撤的痛远超你当初签字接受的水平。
  7. **年度复盘 + 税损收割**：每年看一次配置漂移、费率、目标变化；应税账户顺手做 TLH。详见 Workflow 6。(evidence: [T03-S007])
     - **跳过后果**：配置长期跑偏、持有高费基金不自知、错过合法省税。
  8. **行为纪律（stay the course）**：写下「市场跌 X% 时我不动/继续定投」的预案，把决策从情绪手里拿走。(evidence: [T03-S001, T03-S016])
     - **跳过后果**：被噪音（财经新闻/社媒/FOMO）驱动追涨杀跌，过度交易吃摩擦成本 + 系统性高买低卖。
- **资深路径（差异点）**：
  - **skip**: 资深人**跳过**「逐年手动微调配置 / 频繁查看账户」—— 一旦写下 IPS（投资政策声明），他们一年只动一两次（再平衡+复盘），刻意**减少**看盘频率以降低行为失误；纯被动资深甚至用**一只目标日期基金**把 step 2/4/6 全自动化（见 Workflow 3）。(evidence: [T03-S001, T03-S009])
  - **optimize**: 资深人**优化** step 3+4 为「资产位置 (asset location)」—— 不只选对账户**类型**，还把**税低效资产**（债券/REIT/高频交易）放税优账户、**税高效资产**（宽基股票 ETF）放 taxable，整体税后再多榨几十 bps；执行上用「**新钱定向**」代替卖出来再平衡（把新供款投到欠配的资产，零税零摩擦）。(evidence: [T03-S008, T03-S006])
  - **add**: 资深人**额外做**：① 写**成文 IPS** 并在恐慌时拿出来读（行为锚）；② 跨账户**整体**视角看配置（用 Empower/X-Ray 穿透，而非每个账户各看各的）；③ 预设**熊市行动清单**（跌 X% 加投、TLH、Roth 转换的时机）；④ 退休临近时关注 **sequence-of-returns risk**（提取期前 5–10 年回报顺序决定成败）。(evidence: [T03-S008, T03-S011])
- **近期变化**（近 12 个月）：
  - **2024-12 起（CN，五部门通知触发）**：个人养老金制度由 36 城试点**全国推开**，指数基金首次纳入 Y 份额可投名录（首批权益指基）→ CN 投资者的「账户结构」步骤新增了一个真正的税优容器，主流 SOP 的 step 3 在中国语境从「几乎无税优账户可用」变为「先填个人养老金 Y」。触发类型: 法规变化。(evidence: [T03-S019, T03-S020])
  - **2026 年初（US，IRS Notice 2025-67 触发）**：401(k) 员工上限升至 $24,500、IRA $7,500；新增 60–63 岁增强 catch-up；高收入者(上年工资>$150k)的 catch-up 强制 Roth 形式 → step 3 的**具体数字**每年初须刷新（结构稳定，数字微调）。触发类型: 标准/限额更新。(evidence: [T03-S013])
  - **持续（费率改革 / robo / AI 投顾）**：公募费率改革（CN「以降促升」）与持续费率竞赛把被动成本压到趋零；robo-advisor / CN「帮你投」把整条主工作流打包托管；AI 投顾兴起但**不可当建议直接执行**（无 fiduciary、有幻觉）。触发类型: 行业事件 + 新工具。(evidence: [T03-S018, T03-S021])
- **典型耗时**:
  - 入门 SOP: 初次搭建 1 个周末（定配置+开户+下首单），真正难的是 step 8 的「几十年管住手」。
  - 资深路径: 搭建后每年 1–2 小时（再平衡+复盘+TLH），其余时间刻意「什么都不做」。
- **关键工具（与 Track 02 关联）**:
  - 低费率券商（Vanguard/Fidelity/Schwab；CN 各基金公司）（必备）、宽基指数 ETF（必备）、税优账户类型（必备级）、Portfolio Visualizer（回测配置）、Empower/X-Ray（跨账户体检）、目标日期基金（一键自动化）。
- **关键人物（与 Track 01 关联）**:
  - John Bogle（成本至上/市场组合/stay the course，整条工作流的被动派地基）、Ben Felix（证据派，在 Rational Reminder 反复讲配置/择时/守纪）、William Bernstein（四支柱：理论/历史/心理/生态）、Morgan Housel（行为>智商，step 8 的心态 OS）。
- **常见失败模式**:
  - **先选产品后想账户**：开了 taxable 买基金，却没拿满 401k 雇主匹配 —— 等于每年白扔一份「保证 100% 回报」。自检：你拿满雇主匹配了吗？
  - **配置与风险承受度不匹配**：纸面接受「长期投资」，实际 80% 股票在 −30% 时半夜清仓。自检：你写下「跌 X% 我也不卖」的预案了吗？
  - **被费率慢性放血**：长期持有 1%+ 费率的主动基金/含申购费产品，30 年吃掉巨量复利却不自知。自检：你用 Morningstar/Empower 查过整体加权费率吗？
  - **定投瘫痪 / 等更好时点**：有一笔闲钱却迟迟不入场择时，踏空。自检：你设了自动定投把决策自动化了吗？
  - **从不再平衡**：牛市后股票占比失控，下次回撤痛感远超预期。自检：你定了再平衡触发规则（日历或 5/25）吗？

### 2. 价值投资工作流（选股 → 估值 → 安全边际 → 持有/卖出）

- **One-liner**: 从「我想自己选股票而非买指数」到「以显著低于内在价值的价格买入并理解的好生意、长期持有、按 thesis 卖出」—— 价值派把股票当「企业的一小块所有权」而非可炒作的纸片。对照 Workflow 3 的被动派：本派赌「主动选股能跑赢」（与被动派的核心 canon 分歧）。
- **Trigger**: 决定主动配置一部分仓位选股（多数资深价值派仍把宽基指数作核心，只用「卫星」仓位选股）+ 某公司进入你的能力圈雷达（日常观察 / 筛选器命中 / 估值分位偏低）。(evidence: [T03-S014])
- **Output**: 一份「这家公司值多少、我愿意付多少、为什么、什么情况卖」的书面投资 thesis + 一笔在安全边际内的仓位。(evidence: [T03-S014, T03-S015])
- **入门 SOP（minimum viable steps）**:
  1. **划能力圈 (circle of competence)**：只研究你真正看得懂商业模式/赚钱逻辑/风险的行业。Buffett：「圈的大小不重要，知道边界在哪才重要。」(evidence: [T03-S014])
     - **跳过后果**：研究看不懂的热门赛道（AI/新能源「这是未来」）→ 无法判断生意好坏，本质是赌方向，一遇黑天鹅无力评估该不该补还是该跑。
  2. **建选股池 / 筛选**：在能力圈内用筛选器（GuruFocus F-Score/Z-Score、理杏仁 PE/PB 历史分位）缩小到少数候选。(evidence: [T03-S015])
     - **跳过后果**：在全市场无差别翻找 → 精力分散，且容易被「故事性强但圈外」的标的诱入。
  3. **商业理解（看生意不看股价）**：读财报/年报/股东信，理解护城河、ROE 质量（DuPont 拆解防杠杆虚高）、自由现金流（比会计利润难造假）、管理层。(evidence: [T03-S014])
     - **跳过后果**：只看 P/E「低=便宜」就买 → 踩价值陷阱（低 P/E 是因为盈利将崩）。
  4. **估值（内在价值 / DCF / 相对）**：Damodaran 路线——把「故事」翻译成「数字」（增长率/利润率/再投资），DCF 折现求内在价值 + 相对估值（同业倍数）交叉；DCF 是 sensitivity 工具不是精确科学，重区间和逻辑。(evidence: [T03-S015])
     - **跳过后果**：不估值就买 → 没有「值多少」的锚，无法判断现价是贵是贱，安全边际无从谈起。
  5. **安全边际检验（再买）**：只在**价格显著低于内在价值**时买入，用折扣吸收判断误差与坏运气（Graham 的稳健投资中心概念）。(evidence: [T03-S014])
     - **跳过后果**：好公司付了过高价格 → Marks：「好公司 ≠ 好投资，决定回报的是你付的价。」高位接盘，等估值回归就深套。
  6. **仓位 / 集中度**：决定单一标的占组合多少（价值派偏集中但有上限，控单笔最大损失）。(evidence: [T03-S014])
     - **跳过后果**：单押过重 → 一个 thesis 错误重创整个组合；或过度分散到看不过来，退化成劣质指数。
  7. **持有 + 跟踪 thesis**：长期持有，定期复核「当初买入的理由还成立吗」（护城河有没有被侵蚀、基本面有没有变）。(evidence: [T03-S014])
     - **跳过后果**：买入即不管 → thesis 已破（护城河塌了）却因锚定/确认偏差死扛。
  8. **卖出纪律（thesis 驱动）**：卖出触发 = ① thesis 被证伪（基本面恶化）② 估值远超内在价值 ③ 发现明显更好的机会 ④ 当初买入的错误被发现。**不**因股价短期波动卖。(evidence: [T03-S014])
     - **跳过后果**：用价格情绪（涨了落袋/跌了割肉）代替 thesis 判断 → 处置效应：砍掉赢家、留住输家。
- **资深路径（差异点）**：
  - **skip**: 资深价值派**跳过**绝大多数标的 —— Munger「等待好球」(extreme patience)，一年可能只动一两次；他们**跳过**纯 Graham 式「捡烟蒂」(net-net) 筛选（现代低估值股稀缺、信息高效，Buffett 本人已从纯 Graham 转向「合理价买伟大企业」），不在量化便宜上浪费时间。(evidence: [T03-S014])
  - **optimize**: 资深人**优化**估值为「**反向工程**」—— 不从零搭 DCF，而是问「现价隐含了公司未来必须交付多少营收/利润率/回报」，再用 possible/plausible/probable 三档检验现不现实（Damodaran 法）；估值时聚焦**少数关键驱动变量**而非把模型做到 50 行假设。(evidence: [T03-S015])
  - **add**: 资深人**额外做**：① 主动写 **pre-mortem / 杀死自己的论点**（找反方观点对抗确认偏差）；② 用第二层思维想「我知道的别人是不是早知道了、price 里反映了吗」；③ 区分**永久性资本损失风险**（真风险）与短期波动（非风险），只对前者要安全边际。(evidence: [T03-S014])
- **近期变化**（近 12 个月）：
  - **近 12 个月: 无重大流程变化（行业稳态）**。价值投资的 SOP（能力圈→估值→安全边际→thesis 持有/卖出）自 Graham 1949 至今骨架未变，最近一次显著的「方法论」事件是 Asness 2024-08《The Less-Efficient Market Hypothesis》提出「市场反而更不有效（社媒扭曲定价）」→ 对纪律化价值是机会但意味更长煎熬期，这是**论点**层面而非流程层面的更新。触发类型: 学术/行业讨论。(evidence: [T03-S017])
  - 估值**输入数据**年度刷新：Damodaran 全市场估值数据集每年初更新一次（ERP/beta/multiples），价值派每年初换锚。(evidence: [T03-S015])
- **典型耗时**:
  - 入门 SOP: 单只标的从初筛到完成 thesis 数天–数周（读年报+建估值最耗时）。
  - 资深路径: 看得快（圈内熟悉行业几小时成型 thesis），但出手极慢（多数时间在「等好球」不动）。
- **关键工具（与 Track 02 关联）**:
  - GuruFocus（DCF/F-Score/Z-Score，美股）、理杏仁 lixinger（A/港股 PE/PB 分位）、Damodaran online data（ERP/beta/WACC 锚）、Morningstar（基本面+护城河评级 wide/narrow/no）、Berkshire 股东信（方法论一手）。
- **关键人物（与 Track 01 关联）**:
  - Warren Buffett（能力圈/护城河/企业主思维，股东信即工作流一手）、Charlie Munger（逆向/等好球/格栅）、Benjamin Graham（安全边际/Mr. Market 奠基）、Howard Marks（价格vs价值/第二层思维/周期）、Aswath Damodaran（估值 SOP：故事→数字）、段永平（zh 价值派实践）。
- **常见失败模式**:
  - **能力圈外选股**：研究看不懂的热门行业，本质赌方向。自检：你能用一句话讲清这家公司怎么赚钱、护城河是什么吗？
  - **把 P/E 低当便宜**：忽略「便宜可以更便宜」「低 P/E = 价值陷阱」。自检：你估了内在价值、确认是 price 跌而非 value 跌了吗？
  - **没有安全边际就买**：好公司付高价 = 坏投资。自检：现价相对你估的内在价值有没有显著折扣？
  - **thesis 破了还死扛**：护城河被侵蚀、基本面恶化却因锚定（成本价）/沉没成本不卖。自检：当初买入的理由今天还成立吗？
  - **用价格而非 thesis 做买卖**：涨了落袋、跌了恐慌，处置效应。自检：这笔卖出是 thesis 驱动还是情绪驱动？
- **来源**:
  - [Primary] Berkshire 股东信（能力圈/护城河/安全边际/长持/卖出一手）evidence: [T03-S014]
  - [Primary] Damodaran NYU 主页（估值 SOP：故事→数字→DCF/相对）evidence: [T03-S015]
  - [Reference] AQR / Asness（市场更不有效 → 价值机会但更痛）evidence: [T03-S017]
- **Last_updated**: 2026-06-04（流程稳态；估值数据锚 2026 年初更新）
- **Decay risk**: **low**（方法论自 1949 稳态；唯估值输入数据年度刷新）

### 3. 被动指数工作流（定配置 → 最低费率宽基 → 自动定投 → 阈值再平衡 → 什么都不做）

- **One-liner**: 从「我不想也不该花时间选股/择时」到「一个买全市场低成本指数、自动定投、几乎不动手的组合」—— ≥80% 自主投资者的最优默认。本派是 Workflow 1 主工作流在「选品」一步走「宽基指数」的纯净版，把人为干预降到最低。SPIVA 是其弹药：YE2025 美国 79% 大盘主动跑输 S&P 500。(evidence: [T03-S001, T03-S021])
- **Trigger**: 认同「主动扣费后系统性跑输市场 + 我管不住择时的手」→ 选择被动；新手第一次投资的推荐起点。(evidence: [T03-S001, T03-S016])
- **Output**: 一个三/四基金组合（或单只目标日期基金），设好自动定投，定好再平衡规则，然后**刻意不管**。(evidence: [T03-S004, T03-S009])
- **入门 SOP（minimum viable steps）**:
  1. **定配置（三基金 / 目标日期）**：定股债比 + 国际比例。两条路：① 自建三基金（美股全市场 + 国际股 + 债券）；② 偷懒到极致 = **一只目标日期基金**（自动含配置+再平衡+随龄滑翔降风险）。(evidence: [T03-S004, T03-S009, T03-S010])
     - **跳过后果**：没定配置直接买热门指数（如全仓单一国家/单一行业指数）→ 自以为被动实则集中押注，失去分散这「唯一的免费午餐」。
  2. **选最低费率宽基**：在选定资产类别里挑费率最低、跟踪误差最小的宽基（VTI/VXUS/BND 或 VT；CN 沪深300/标普500 QDII 宽基指基）。被动派第一纪律 = 最小化费率。(evidence: [T03-S004])
     - **跳过后果**：买了高费率/主题/杠杆 ETF（「ETF」是结构不是「便宜被动」的代名词）→ 费率复利侵蚀 + 偏离全市场。
  3. **自动定投**：在券商设定额定期自动申购（工资到账即扣），把「要不要现在买」的决策彻底自动化。(evidence: [T03-S005])
     - **跳过后果**：靠手动「感觉好了再买」→ 退化成择时，踏空 + 情绪化。
  4. **阈值 / 日历再平衡**：设触发规则（日历法：每年固定一天；或阈值法：5/25 规则）把权重拉回。详见 Workflow 6。(evidence: [T03-S006])
     - **跳过后果**：股票占比随牛市膨胀，风险敞口失控。
  5. **「什么都不做」（核心纪律）**：除再平衡/复盘外，**不**响应市场噪音、不择时、不追热点。这一步是被动工作流**最难且最增值**的部分。(evidence: [T03-S001, T03-S016])
     - **跳过后果**：被新闻/社媒/FOMO 驱动频繁调仓 → 过度交易 + 系统性高买低卖，亲手摧毁被动的全部优势（行为差距 / behavior gap）。
- **资深路径（差异点）**：
  - **skip**: 资深被动派**跳过**几乎所有「优化」诱惑 —— 不看盘、不读「该不该减仓」的财经预测、不试图躲熊市；极致者**跳过** step 2/4 的全部手动操作，用一只目标日期/全天候基金把整条工作流自动化到「设了就忘」。资深的标志是**做得更少**而非更多。(evidence: [T03-S009, T03-S001])
  - **optimize**: 资深人**优化**税务维度（被动派也有可优化处）：① **asset location**——债券放税优、股票放 taxable；② **用新供款再平衡**避免卖出产生资本利得；③ taxable 账户做 **TLH**（见 Workflow 6）；④ 适度**因子倾斜**（小盘/价值，用 DFA/Avantis），但知道因子会有长期煎熬期、且费率高于纯宽基（与「成本至上」存在张力）。(evidence: [T03-S008, T03-S016])
  - **add**: 资深人**额外做**：① 写下「熊市我继续定投/再平衡」的成文预案当行为锚；② 跨账户整体看配置（穿透 401k+IRA+taxable）而非各看各的；③ 接近退休时把工作流从「积累」切到「提取」模式，关注 sequence risk 与提取率（见 Workflow 1/6）。(evidence: [T03-S008, T03-S011])
- **近期变化**（近 12 个月）：
  - **持续（费率改革触发）**：公募费率改革（CN「以降促升」）+ 全球费率竞赛把宽基被动成本压到趋零 → 被动 SOP 的「成本优势」进一步扩大；目标日期基金已成多数养老金/401k 的**默认选项**（自动配置成为新手事实起点）。触发类型: 行业事件 + 标准。(evidence: [T03-S021])
  - **2024-12（CN 法规触发）**：指数基金首次纳入个人养老金 Y 份额名录 → CN 被动派第一次能在税优容器里做指数定投，被动工作流在中国语境补上了「税优 + 被动」的拼图。触发类型: 法规变化。(evidence: [T03-S019, T03-S020])
  - **持续（robo / AI 投顾触发）**：robo-advisor / CN「帮你投」把整条被动工作流（配置+定投+再平衡+TLH）打包托管，但叠加 0.25–0.5% 顾问费在基金 ER 之上（双重收费要算清）；robo 行业 2026 洗牌（Schwab/JPM/UBS 关停高端 robo）。触发类型: 新工具 + 行业事件。(evidence: [T03-S018])
- **典型耗时**:
  - 入门 SOP: 搭建半天–1 天；之后每年 < 1 小时。
  - 资深路径: 用目标日期基金近乎 0 维护；自建组合每年 1 次再平衡+复盘。
- **关键工具（与 Track 02 关联）**:
  - 低费率券商三选一（必备）、宽基指数 ETF/基金（必备）、目标日期基金（一键自动化）、Portfolio Visualizer（回测配置）、Morningstar X-Ray（持仓穿透/重叠）、Empower（跨账户聚合+费用分析）、robo（可选托管层）。
- **关键人物（与 Track 01 关联）**:
  - John Bogle（成本至上/市场组合/stay the course，本派奠基）、Burton Malkiel（随机游走/指数核心持仓的学术代言）、Ben Felix（证据派当代旗手，RR 反复讲被动+守纪）、William Bernstein（四支柱配置）、JL Collins（Simple Path / 一只 VTSAX 极简派）。
- **常见失败模式**:
  - **熊市底部恐慌清仓**：被动工作流唯一的真失败点 —— 在最该继续买时卖出，锁死亏损。自检：你写了「跌 X% 我不卖/继续投」预案吗？
  - **以为「被动」=「随便买个 ETF」**：买了高费率/主题/杠杆 ETF 当被动。自检：你买的是覆盖全市场的最低费率宽基吗？
  - **手动择时取代自动定投**：「等跌了再买」踏空。自检：你设了自动定投把决策自动化了吗？
  - **给被动组合叠加择时/选股工具**：上 TradingView/选股软件徒增复杂度与出错面（→ T02 反模式）。自检：你是不是在给一个本该「无聊」的组合加戏？
  - **robo 双重收费不自知**：0.25% 顾问费叠加在基金 ER 之上，小账户不划算。自检：直接自持宽基 ETF 是不是更省？
- **来源**:
  - [Primary] Bogleheads「投资哲学」+ 三基金/目标日期/懒人组合 SOP evidence: [T03-S001, T03-S004, T03-S009, T03-S010]
  - [Primary] Rational Reminder（证据派被动+守纪一手）evidence: [T03-S016]
  - [Reference] SPIVA（被动有效的数据弹药）evidence: [T03-S021]
- **Last_updated**: 2026-06-04（核心流程几十年稳态；外围费率/账户/robo 在变）
- **Decay risk**: **low**（最抗衰减的工作流之一）—— 唯外围（费率绝对值、robo 格局、CN 名录）medium。

### 4. 技术分析工作流（扫 setup → 入场 → stop-loss → 风险定仓 → 出场 → 复盘）

> **诚实标注（信号薄弱）**: 本 workflow 的 SOP 主要从 canon（Edwards & Magee / Murphy / O'Neil CANSLIM）+ 工具文档（TradingView/StockCharts）+ 术语（T06 F 节）反推，**一手「资深技术交易者怎么走完一笔」的公开长材料稀缺**（技术派的真活动在交易软件社区与封闭付费群，公开高质量 newsletter/podcast 少 —— 见 T05）。bucket 上以 secondary（tool docs）+ canon 为主，标 surrogate 性质。**且学术界（EMH 弱式/随机游走）对技术分析有效性持否定**——本派与被动/价值派根本对立，不强行统一。

- **One-liner**: 从「我想靠价格图表/形态/动量做短中期择时交易」到「一套有明确入场信号、事前止损、按风险定仓、机械出场的可复盘交易系统」—— 与价值（看基本面）/被动（择时无效）三方互掐。本派信条：价格已含全部信息，可由形态/趋势/动量读出群体行为。(evidence: [T03-S021])
- **Trigger**: 选择做主动交易而非买入持有；扫描发现一个符合预设条件的 setup（突破/形态/动量/CANSLIM 命中）。
- **Output**: 一笔有记录的交易（入场价 + 止损价 + 仓位大小 + 出场规则）+ 复盘后更新的交易日志。
- **入门 SOP（minimum viable steps）**:
  1. **扫 setup（趋势 / 形态 / 动量）**：用扫描器找符合预设的标的——趋势方向（200/50 日 MA）、经典形态（头肩/双底/三角形）、动量强弱、或 CANSLIM 多条件命中（突破基底）。(evidence: [T03-S021])
     - **跳过后果**：无预设标准、看哪个顺眼买哪个 → 没有可重复、可统计的边际，本质赌博。
  2. **定入场（trigger）**：等价格**放量突破**关键阻力/形态边界再进，区分真假突破（need 放量确认 + 防回踩）。
     - **跳过后果**：提前猜测/抄底没确认的位置 → 接飞刀，假突破被反向收割。
  3. **设 stop-loss（事前）**：**入场前**就定好「跌到哪我认错出场」的价格（技术派风控核心纪律，机械执行非情绪反应）。
     - **跳过后果**：不设止损 → 单笔小亏拖成大亏，一笔失控交易吃掉数十笔盈利（交易者破产头号原因）。
  4. **按风险定仓位 (position sizing)**：由「入场价 − 止损价」的距离 + 单笔最大可亏比例（如总资金 1%）**反推**买多少股，让每笔风险敞口一致。趋势/交易派认为 sizing 比选标的更决定长期存活。
     - **跳过后果**：固定金额或梭哈 → 止损宽的标的实际风险被放大，一笔爆仓；或仓位忽大忽小无法统计期望。
  5. **管理出场**：盈利单按规则推移止损（trailing stop）让利润奔跑 / 到目标位减仓；亏损单触止损无条件出。
     - **跳过后果**：盈利单过早落袋、亏损单死扛（处置效应）→ 砍赢家留输家，期望为负。
  6. **复盘（交易日志）**：记录每笔的理由/结果，统计胜率×盈亏比，迭代系统、剔除无效 setup。
     - **跳过后果**：不复盘 → 重复同样错误，永远分不清是运气还是系统，无法改进。
- **资深路径（差异点）**：
  - **skip**: 资深交易者**跳过**绝大多数 setup（只打高胜率/高盈亏比的少数形态），**跳过**指标堆砌（新手挂十个指标，老手只用价格+少数确认）；不在震荡市强行交易（MA 滞后、假信号多）。(evidence: [T03-S021])
  - **optimize**: 资深人**优化**风控为「**先想 sizing 和止损，再想入场**」—— 风险预算前置；并把规则**写成机械系统/脚本**（TradingView Pine Strategy Tester 回测）减少主观临场决策。
  - **add**: 资深人**额外做**：① 看**大盘方向 (M in CANSLIM)** 当过滤器（逆大盘的个股 setup 胜率骤降）；② 关注**市场宽度 (breadth)** 判断内部强弱；③ 严格**交易日志 + 期望值统计**，把交易当概率生意而非「看对看错」。
- **近期变化**（近 12 个月）：
  - **持续（零佣金 + 工具触发）**：零佣金诱发**过度交易**（摩擦成本降 → 散户交易频率升，但过度交易与净收益负相关）；TradingView 等把图表+Pine 脚本回测+多触发警报做到一体，降低系统化门槛。触发类型: 行业事件 + 新工具。(evidence: [T03-S018])
  - **方法论层近 12 个月无重大变化**：经典形态/指标/CANSLIM 自 Edwards-Magee(1948)/O'Neil(1988) 稳态；新增的是**执行工具**（脚本化/警报/手机端）而非**方法**。触发类型: 工具升级（非方法）。(evidence: [T03-S021])
- **典型耗时**:
  - 入门 SOP: 单笔交易决策分钟级（扫到→定止损→下单），但建立**有正期望的系统**需数月–数年试错。
  - 资深路径: 出手快（熟悉形态），但多数时间在等高质量 setup + 复盘。

### 5. 趋势 / 动量工作流（定信号 → 分散篮子 → 系统化进出 → 风险定仓 → 跟系统）

- **One-liner**: 从「我想用机械规则跟随价格趋势、在熊市自动离场」到「一套不预测方向、纯按信号进出、分散在多资产篮子上、严格控仓的系统化策略」—— 与技术派（主观读图）的关键区别：**完全机械、规则前置、不主观**；与价值/被动的根本分歧：不看价值、不长期死扛，跟趋势 + 让利润奔跑 + 严格止损。学术地基 = Jegadeesh-Titman 1993 动量效应（→ T04-P05）。本派是散户能照着做的「可回测、可复制」择时路线，区别于技术派的酌情交易。
- **Trigger**: 想要「参与上涨 + 规避大回撤」但不信主观判断；按预设周期（如月末）检查信号是否翻转。(evidence: [T03-S023])
- **Output**: 一套写死的规则（信号定义 + 进出条件 + 仓位规则）+ 按规则机械执行的持仓状态（在市 or 离场到现金/债）。(evidence: [T03-S023])
- **入门 SOP（minimum viable steps）**:
  1. **定信号（机械、可回测）**：选一个明确规则。两条散户常引路线：① **Faber 200 日/10 个月 MA**——价格在 10 个月(≈200日)均线**之上**则持有、**之下**则移到现金（结果：约 70% 时间在市、每年 < 1 次往返、股票级回报+债券级回撤）。② **Antonacci 双动量 (GEM)**——用**相对动量**在美股 vs 非美股间选强者，用**绝对动量**（自身近 12 月正/负）在股票 vs 债券间切换。(evidence: [T03-S023, T03-S017])
     - **跳过后果**：没有写死的信号定义 → 退化成主观择时（Workflow 4 的酌情判断），失去「系统化」的全部意义和可回测性。
  2. **分散篮子（多资产）**：把信号应用到一篮**低相关**资产（美股/国际股/债/REIT/商品），而非单一标的——趋势在「几乎所有资产、所有市场」都存在（Asness「Value and Momentum Everywhere」），分散摊平单一市场失效。(evidence: [T03-S017])
     - **跳过后果**：只在单一资产上跑趋势 → 该资产长期横盘震荡时被反复假信号「锯」死（whipsaw），无处分散。
  3. **系统化进出**：信号触发就**无条件**执行进出（不二次猜测、不因「这次不一样」覆盖规则）。(evidence: [T03-S023])
     - **跳过后果**：选择性执行规则（只在「感觉对」时跟）→ 破坏了系统的统计期望，等于没系统。
  4. **风险（平价）定仓**：按**风险贡献**而非等资金分配（波动率目标 / 风险平价），让每个仓位贡献相近风险；或按 ATR/波动率反推仓位。(evidence: [T03-S017])
     - **跳过后果**：等资金分配 → 高波动资产主导组合风险，回撤集中爆发。
  5. **跟系统不主观（核心纪律）**：长期严格执行，**扛得住**策略阶段性难看（趋势策略会有长期跑输/震荡市连续小亏的「煎熬期」）。Asness：「理性、能分散的策略期望回报更高，但你得先扛得住它阶段性的难看。」(evidence: [T03-S017])
     - **跳过后果**：在策略回撤期/连续假信号后**放弃**或临时改规则 → 恰好在系统该兑现的前夜离场（趋势策略最大失败点 = 人为中断）。
- **资深路径（差异点）**：
  - **skip**: 资深趋势交易者**跳过**对单次信号的预测/解读（不问「这次趋势真不真」，只问「规则说什么」），**跳过**优化到完美的诱惑（过度拟合是趋势策略头号死法）；多数把趋势当**整体组合的一个分散来源**（与买入持有/价值并存），而非全部身家。(evidence: [T03-S017])
  - **optimize**: 资深人**优化**为「多信号/多周期组合」降低单一参数依赖（不只 200 日，用多个回看窗口投票）+ 明确的**交易成本/滑点建模**（趋势策略换手高，成本侵蚀实盘 vs 回测的主要缺口）。(evidence: [T03-S017, T03-S023])
  - **add**: 资深人**额外做**：① 样本外/多市场**稳健性检验**（不信单一历史曲线）；② 明确写下「我能忍受多长的跑输期」并据此定仓位；③ 用**风险平价/波动率目标**让组合在不同波动环境下风险敞口稳定。(evidence: [T03-S017])
- **近期变化**（近 12 个月）：
  - **方法论层近 12 个月无重大变化（行业稳态）**：趋势/动量的核心规则（MA 择时、双动量、time-series momentum）自 Faber 2006 / Antonacci 2014 / Jegadeesh-Titman 1993 稳态；2023 Jegadeesh-Titman 30 年回顾确认原效应仍稳健。最近的「讨论」是 Asness 2024-08《LEMH》认为市场更不有效 → 动量/价值机会仍在但波动更大、煎熬期更长。触发类型: 学术讨论（非流程变化）。(evidence: [T03-S017])
  - **持续（工具触发）**：Portfolio Visualizer 战术模块 / TradingView Pine 让散户能回测 MA/动量信号 → 系统化门槛下降，但也放大**过拟合**风险（回测漂亮、实盘失效）。触发类型: 新工具。(evidence: [T03-S023])
- **典型耗时**:
  - 入门 SOP: 信号检查月度 5 分钟（机械），建立+回测系统数天–数周。
  - 资深路径: 执行近乎零时间（规则化），精力在稳健性检验 + 忍受煎熬期的心理建设。
- **关键工具（与 Track 02 关联）**:
  - Portfolio Visualizer（战术配置/MA/动量信号回测）、testfol.io（日频长历史回测）、TradingView（信号警报+Pine 回测）、FRED（宏观）。
- **关键人物（与 Track 01 关联）**:
  - Michael Covel（趋势跟踪布道，《Trend Following》，本派唯一可调研在世旗手，与 Workflow 4 重叠）、Cliff Asness（time-series/cross-section momentum 学术+实务，AQR）、Meb Faber / Gary Antonacci（散户可执行的 TAA/双动量，canon 扩展）；学术源头 Jegadeesh-Titman。
- **常见失败模式**:
  - **煎熬期放弃系统**：连续假信号/跑输后人为中断，恰在该兑现前离场。自检：你写下「能忍 N 个月跑输」并据此定了仓位吗？
  - **单一资产跑趋势被锯死**：震荡市 whipsaw。自检：你的信号是应用在一篮低相关资产上吗？
  - **选择性执行规则**：只在「感觉对」时跟信号。自检：上次信号触发你是无条件执行还是覆盖了规则？
  - **过拟合回测**：调参到历史完美曲线、样本外崩（→ T02 反模式）。自检：换市场/换时段/加交易成本后系统还正吗？
  - **忽略换手成本**：趋势策略高换手，实盘被佣金+滑点吃掉回测收益。自检：你的回测含交易成本和滑点了吗？

  - [Primary] AQR Insights / Asness（time-series momentum + 「期望更高但得扛得住」一手）evidence: [T03-S017]

### 6. 再平衡 + 税损收割子工作流（横切，被多个主工作流复用）

- **One-liner**: 从「持仓权重随市场涨跌偏离了我的目标 + 应税账户有浮亏」到「权重拉回目标（纪律化低买高卖）+ 顺手实现亏损省税」—— 这是 Workflow 1/3（以及部分 2/5）都复用的**横切子工作流**，本身不产生新仓位逻辑，只维护既有组合。
- **Trigger**: ① 再平衡——到了预设日历日（如每年某天）或权重触发阈值；② TLH——应税账户某标的出现显著浮亏（市场下跌期机会最多）。(evidence: [T03-S006, T03-S007])
- **Output**: 权重回到目标配置 + （应税账户）已实现的资本亏损用于抵税，且敞口未中断、未触 wash-sale。(evidence: [T03-S006, T03-S007])
- **入门 SOP（minimum viable steps）**:
  1. **选再平衡方法（日历 vs 阈值）**：① **日历法**——固定周期（每年/每季）查一次并拉回；② **阈值法**——偏离超过带宽才动。两者机械执行**都优于**凭感觉择时。(evidence: [T03-S006])
     - **跳过后果**：从不再平衡 → 牛市后股票占比膨胀，风险敞口悄悄翻倍，下次回撤痛感远超签字时接受的水平。
  2. **（阈值法）用 5/25 规则定带宽**（Larry Swedroe，Bogleheads 收录）：当某资产类别**绝对**偏离目标 **5 个百分点**、**或相对**偏离 **25%**，取**先到者**触发再平衡。例：目标 60/40，到 65/35 或 55/45（绝对 5pp）触发；占比 < 20% 的小仓位用相对 25%（如目标 15% 国际股，偏离 ±3.75% 即 25%×15% 触发）。(evidence: [T03-S006])
     - **跳过后果**：带宽拍脑袋（太窄→频繁交易吃成本与税；太宽→长期跑偏失去风控意义）。
  3. **优先用「新钱」再平衡**：把新供款/股息**定向**投到欠配的资产，尽量**不卖出**——零资本利得、零摩擦。(evidence: [T03-S006])
     - **跳过后果**：靠卖出再平衡 → 在 taxable 触发资本利得税，本可用新钱无税完成。
  4. **再平衡尽量在税优账户内做**：需要卖出时，优先在 401k/IRA 等税优账户里调（卖出不触发当期税），避免在 taxable 卖出产生应税利得。(evidence: [T03-S006, T03-S008])
     - **跳过后果**：在 taxable 大额卖出再平衡 → 凭空多缴资本利得税。
  5. **（TLH，仅 taxable）卖浮亏 + 换相似非「实质相同」标的**：卖出浮亏标的实现亏损（抵资本利得 + 每年有限额抵普通收入），**同时**买入**相似但非 substantially identical** 的标的（如从一只标普500 ETF 换到一只全市场 ETF）维持敞口。(evidence: [T03-S007])
     - **跳过后果**：卖出后空仓等 → 错过反弹（择时风险）；或买回**实质相同**标的 → 触 wash-sale，亏损被取消不可抵。
  6. **守 wash-sale 规则**：买入/卖出前后 **30 天**内不得买回「实质相同」标的（含配偶账户、IRA 内买回），否则亏损递延作废。(evidence: [T03-S007])
     - **跳过后果**：30 天内手滑买回（或自动定投/DRIP 在窗口内买进同标的）→ wash-sale，省税泡汤还要记账。
- **资深路径（差异点）**：
  - **skip**: 资深人**跳过**「日历到点就无脑全面再平衡」—— 他们优先用现金流/新钱再平衡，很多年根本不需要卖出；也**跳过**对**微小偏离**的反应（5/25 没触发就不动），刻意降低交易频率以省成本和税。(evidence: [T03-S006])
  - **optimize**: 资深人**优化**为「**再平衡 + TLH 合并执行**」—— 在市场下跌期一次动作里既拉回权重、又顺手收割亏损；并把 TLH 和**资产位置 (asset location)** 联动（在 taxable 持税高效宽基便于换标的 TLH，债券/REIT 藏税优账户）。极致者用 robo / 直接指数化**自动**逐股 TLH。(evidence: [T03-S008, T03-S018])
  - **add**: 资深人**额外做**：① 关掉 TLH 标的的**自动定投/DRIP** 防自酿 wash-sale；② 记录每次 TLH 的**降低后的成本基准**（TLH 是**递延非免除**，未来 gain 更大，临近高税率年份或捐赠/继承 step-up 时再算总账）；③ 退休/提取期把「再平衡」与 **sequence risk + 4% 法则**联动管理。(evidence: [T03-S007, T03-S011])
- **近期变化**（近 12 个月）：
  - **持续（直接指数化下沉触发）**：直接指数化（持指数成分股而非 ETF）把 TLH 从「整只基金层面」细化到**逐股**，自动在成分股层面收割亏损 → Fidelity FidFolios（$5k 起）/ Schwab Personalized Indexing 把这能力下沉到零售（曾是超高净值专属）。但小账户/图省事用一支宽基 ETF 仍更优，别为边际 TLH 买复杂度。触发类型: 新工具/产品下沉。(evidence: [T03-S018])
  - **方法论层稳态**：再平衡（5/25、日历）与 wash-sale 规则本身近 12 月无变化；变的是**执行自动化程度**（robo/直接指数化代劳）。退休提取率（4% 法则）持续被学界/社区重估（FIRE 的 50–60 年期常降到 3–3.5%）。触发类型: 工具升级 + 学术再审视。(evidence: [T03-S011, T03-S022])
- **典型耗时**:
  - 入门 SOP: 再平衡每年 1 次、每次几十分钟；TLH 在下跌期机会性触发，每次 10–30 分钟。
  - 资深路径: 用新钱再平衡近乎免操作；robo/直接指数化自动 TLH = 0 时间。
- **关键工具（与 Track 02 关联）**:

## Phase 2 提炼提示

### 反复出现 ≥ 3 个 workflow 都包含的步骤（候选 playbook 通则）

> 这些跨 workflow 反复出现的步骤模式，是 Phase 2.2 直接成 playbook 的最强候选。

- **【账户结构先于产品】** 出现于 workflows: **1（step 3 核心）/ 3（资深 optimize）/ 6（在税优账户再平衡 + asset location）**，且与 T02 工具结论、T06 税务术语三方印证 → 候选 playbook「**如果有税优额度未用满，则先填税优容器（US: 401k 匹配→HSA→IRA/Roth→taxable；CN: 个人养老金 Y→普通账户）再谈买什么**」。本行业**最被低估、资深最看重**的通则，税后杠杆常大于选基金。
- **【定投/自动化克服择时】** 出现于 workflows: **1（step 5）/ 3（step 3）/ 6（用新钱再平衡）** → 候选 playbook「**如果面对'要不要现在买/卖'的择时决策，则把它自动化（自动定投、新钱定向再平衡），不留人为裁量口**」。底层共识：time-in-market > timing the market（T06 铁律）。
- **【阈值/规则化再平衡，机械优于择时】** 出现于 workflows: **1（step 6）/ 3（step 4）/ 5（系统化进出）/ 6（5/25 + 日历）** → 候选 playbook「**如果权重/信号偏离触发了预设规则，则机械执行；没触发就不动**」。5/25 规则是可直接落地的数字化版本。
- **【安全边际/确认再行动】** 出现于 workflows: **2（step 5 安全边际再买）/ 4（放量突破确认再入场）/ 5（绝对动量为正才持有）** → 候选 playbook「**不在'便宜/像要涨'时就动，等明确的确认条件（折扣够深 / 突破放量 / 信号转正）满足再下注**」。三派用不同语言表达同一纪律：事前要个缓冲/确认。
- **【事前定风险/仓位，再谈标的】** 出现于 workflows: **2（仓位/集中度）/ 4（按风险定仓 position sizing）/ 5（风险平价定仓）** → 候选 playbook「**先定'这笔最多承担多少风险/占多少'，再定买什么；先 sizing 后选标的**」。
- **【写下规则/预案当行为锚，跟规则不跟情绪】** 出现于 workflows: **1（成文 IPS + 熊市预案）/ 3（'什么都不做'纪律）/ 4（交易日志）/ 5（跟系统不主观）** → 候选 playbook「**把决策规则提前写下来，在情绪高/低点时拿出来执行而非临场判断**」。这是贯穿全行业、跨四派的元纪律。

### 入门 SOP 和资深路径之间最大的差距（候选 心智模型）

- **资深 = 做得更少，不是更多**：被动/价值/趋势资深的共同标志都是**降低操作频率**（一年动 1–2 次、刻意不看盘、跳过微小偏离、等好球）。入门者忙着「优化/调仓/择时」，资深者忙着「不动手 + 守纪律」。→ 候选心智模型「**不要在执行阶段过度干预；最大的 alpha 来自不犯行为错误而非多操作**」（behavior gap 是头号亏损源）。
- **资深把'选什么'问题降级，把'结构/纪律/成本/税'问题升级**：入门焦点是「买哪只」，资深焦点是「账户结构对不对、配置匹不匹配风险、费率/税最小化没有、规则守没守」。→ 候选心智模型「**配置 + 容器 + 成本 + 行为 > 选品**」（与 T02「账户结构先于产品」、T06「定配置是老手焦点」三方印证）。
- **资深路径压缩步数靠'自动化'而非'省略'**：主工作流入门 8 步，资深可用「一只目标日期基金 + 自动定投 + 自动 TLH/robo」把 2/4/6 步折叠成「设了就忘」，但**纪律步（step 8）永不省略**。→ 心智模型「**把可机械化的步骤交给系统，把省下的意志力全投到'管住手'上**」。
- **四派分歧只在'选品'一步**：6 个 workflow 里，主工作流(1) 的 step 1/2/3/5/6/7/8 是四派**共用地基**，分歧**仅在 step 4「选品」**往哪走（宽基 vs 选股 vs 交易 vs 信号）。→ 心智模型「**绝大多数投资成败由四派共享的'结构与纪律'决定，流派之争（主动vs被动/基本面vs技术/择时vs持有）只决定那一小步**」。

### 失败模式 top 10（跨 workflow 聚合，喂 Phase 2.6 反模式 + cli_writer 自检项）

> 按「跨多少 workflow 出现 + 后果严重度」排序。每条 actionable，可直接转 `cli/workflow/*.sh` 的 yes/no 自检项。

1. **先选产品后想账户结构**（W1）—— 没拿满 401k 雇主匹配就丢钱进 taxable，白扔一份「保证 100% 回报」。【跨 W1/W3/W6 的账户结构主题】
