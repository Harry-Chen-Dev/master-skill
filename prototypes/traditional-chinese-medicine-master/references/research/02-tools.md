# Track 02 — Tools: Traditional Chinese Medicine (中医诊疗)

中医诊疗工具栈 与 china-icp-filing 那种 ".gov.cn 一手 + 云厂商 vendor docs" 的双极结构 不同, 而是 **七层并存** 的 行业固有形态: (1) **国家监管 + 备案 + 中医师资格** — natcm.gov.cn 国家中医药管理局 中医诊所备案 + 中医师资格 + 95 个单病种临床路径 + 国家级名老中医工作室登记 + 国医大师传承 项目; nmpa.gov.cn 中药注册 + 古代经典名方目录 + 中药注射剂警示 + 不良反应监测; nhc.gov.cn 国家医师资格考试 (中医执业医师 + 中医执业助理医师 + 师承确有专长 三轨); 各省 卫健委 / 中医药管理局 (北京 wsjkw.beijing.gov.cn / 上海 wsjkw.sh.gov.cn / 广东 wsjkw.gd.gov.cn / 江苏 / 浙江 / 四川 / 天津) — 全部 .gov.cn auto primary. (2) **临床辅助 + 智能诊疗** — 中医辅助诊疗 AI + 智能舌诊 + 智能脉诊 + CDSS, 一手主要 通过 CACMS 中国中医科学院 + 各院校 (.edu.cn auto primary) + NATCM 立项公告; 中医病案 + 临床路径 + 诊疗规范 数据库 主入口 在 natcm.gov.cn + cacm.org.cn 中华中医药学会.
