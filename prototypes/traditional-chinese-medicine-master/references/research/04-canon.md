# Track 04 — Canon: Traditional Chinese Medicine (中医诊疗)

中医诊疗 这个行业的 canon 在 master-skill 蒸馏目标中结构最复杂, 因为它天然 六层并存 — 千年古籍 (秦汉到明清) + 院校 新世纪 规划教材 (1956 起院校化, 2002/2007/2012/2017/2022 五版规划教材迭代) + 当代名家 学术专著 (国医大师 + 全国名老中医 学术继承人项目 衍生物) + 国家监管法规 (中医药法 2017 起 + NATCM 95 个单病种临床路径 + 中医诊所基本标准 + ZY/T 001 病案规范 + 中国药典一部) + 国际化文件 (WHO ICD-11 第 26 章 Traditional Medicine Module + WHO Traditional Medicine Strategy + WHO benchmarks for training in TCM + Cochrane Library 中医 SR 数百篇) + 安全性 + 不良反应 文献 (NMPA 中药注射剂警示通告 + 国家不良反应监测年报 + NEJM 1993 马兜铃酸肾病开山文献 + Lancet / Hepatology / JASN 国际期刊中药安全文献). 这六层在 一手 / 二手 结构上 行为差异极大, 因此本 manifest 的 verified_primary / surrogate_primary 比例与 china-icp-filing 那种 ".gov.cn 一手垂直统治" 的 manifest 完全不同 — 中医 canon 必须接受 一手率 结构性偏低 (大致 35-45% verified_primary + 55-65% surrogate_primary) 是 行业客观结构, 因为古籍现代点校本 + 院校规划教材 + 当代名家专著 + 国际期刊 publisher 路径 都 不在 PRIMARY_SUFFIX 白名单, 必须走 vendor docs 通道.

古代经典层有三条一手路径: (1) 国家中医药管理局 中医古籍数字化平台 是 .gov.cn auto primary, 但实际可索引的篇目有限 + URL 稳定性差, 因此本 manifest 实际多数古籍走 (2) ctext.org 中国哲学书电子化计划 — 这个站收录 黄帝内经素问 / 灵枢 / 难经 / 神农本草经 / 伤寒论 / 金匮要略 等 中医汉唐古籍原文最全, 但 .org 域名不在 primary suffix, 必须 declared=surrogate_primary + 'vendor docs (古籍数字化平台)'; ctext.org 论坛部分严禁引用 (与黑名单 知乎 / 微信公众号 同质) — 但古籍原文部分是 学界共识可信源; 同理 (3) 现代点校本 通过 出版社 listing — 人民卫生出版社 / 中国中医药出版社 / 上海科学技术出版社 三家是 中医古籍现代点校 + 院校规划教材 主要出版方, 但出版社域名不在白名单, 必须 declared=surrogate_primary + 'vendor docs (国家级出版社)' literal. 中医世家 zysj.com.cn 是 surrogate_primary candidate (古籍数据库部分), 但论坛 + 现代评论部分 严禁, 本 manifest 谨慎使用.

院校规划教材层走 (1) 院校 .edu.cn 教研室教师页 — 北京中医药大学 bucm.edu.cn / 上海中医药大学 shutcm.edu.cn / 南京中医药大学 njucm.edu.cn / 广州中医药大学 gzucm.edu.cn / 成都中医药大学 cdutcm.edu.cn / 天津中医药大学 tjutcm.edu.cn 等 26 所中医药大学 .edu.cn 域名 自动 PRIMARY_SUFFIX, 教研室主页 + 教材主编介绍 + 国家精品课程页 + 一流课程页 + 课程大纲 直接 verified_primary; 与 (2) 中国中医药出版社 + 人民卫生出版社 教材 listing — 这是教材的二手承载, 但出版社域名不在白名单, 必须 'vendor docs (国家级出版社)' literal. 当代名家专著层同此模式 (院校教师页 + 出版社 listing 互补).

国家监管法规层是本 manifest 一手率最高的部分 — 中医药法 2017 走 npc.gov.cn (全国人大 国家法律法规数据库) 一手; 中医诊所基本标准 + 中医诊所备案管理暂行办法 + 95 个单病种中医临床路径 + 中医优势病种诊疗方案 + ZY/T 001 中医病案书写规范 走 natcm.gov.cn (国家中医药管理局) 一手; 中国药典一部 通过 chp.org.cn 国家药典委员会 surrogate + 'vendor docs (国家药典委员会 监管)' (国家药典委 .org.cn 在 primary suffix 边缘, 但保守走 vendor docs 路径). 国际化文件层 WHO who.int + icd.who.int .int 域名不在 PRIMARY_SUFFIX, 必须 declared=surrogate_primary + 'vendor docs (WHO 国际监管)' literal; Cochrane Library cochranelibrary.com .com 不在白名单, 同 'vendor docs (Cochrane 国际系统评价)'.

安全性 + 不良反应层走 (1) NMPA nmpa.gov.cn 历年中药注射剂警示通告 (鱼腥草 2006 / 双黄连 2017 / 清开灵 2017 / 喜炎平 2017 / 葛根素 2003) + 历年国家药品不良反应监测年报 中药部分 — 全部 .gov.cn auto primary; 与 (2) 国际期刊 publisher 路径 — NEJM nejm.org 1993 马兜铃酸肾病开山文献 (Vanherweghem 比利时减肥草药案例) / Lancet thelancet.com / Hepatology onlinelibrary.wiley.com / JASN journals.lww.com 后续文献; publisher 域名 .com 不在白名单, 必须 declared=surrogate_primary + 'vendor docs (国际期刊 publisher)' literal (除非有直接 DOI 解析路径 + 期刊本身受 PRIMARY_SUFFIX 承认, 这种情况罕见). 严禁 微信公众号 / 知乎 / 抖音 / 快手 / 百度系 / CSDN / developer.aliyun.com / tencent.cloud.com — 任何中医 KOL / 养生博主 / 名老中医二代传承个人账号 / 中药企业营销公众号 都不进 canon, 必须用 院校 + 协会 + 监管 + 期刊 + 出版社 替代.
