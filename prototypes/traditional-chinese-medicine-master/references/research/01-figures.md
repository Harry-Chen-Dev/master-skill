# Track 01 — Figures: Traditional Chinese Medicine (中医诊疗)

中医诊疗这条线的 figures 结构 在 master-skill 蒸馏目标里 相比 china-icp-filing 那种 "学术 + 律所 + 监管 三轨平行" 的现代法律行业 figures 结构 更加 **古今交错 + 师承叠层 + 国家级表彰显性**, 因此本 manifest 在 figure 类型选取与一手源结构上 都 与 ICP 备案那条线 显著不同, 必须先把行业结构讲清楚, 后续蒸馏 才不会把 "国医大师" "院士" "学派代表" "学术继承人" 这些 国家级表彰头衔 与 普通名医 / KOL 混为一谈.

第一, **国家级表彰轨** 是 中医 figures 最显著的 一手锚点 — 国务院 + 国家中医药管理局 自 2008 年 起 共评出 四批 国医大师 (2009 + 2014 + 2017 + 2022, 共 120 位 + 部分已故), 这是 中医界 国家级最高荣誉, 评审标准 包括 "从事中医临床 / 中药相关 工作 55 年以上 + 学术地位崇高 + 师承有序 + 临床有重大贡献"; 同时 国家中医药管理局 设 "全国名老中医药专家学术经验继承工作" (1990 起 第一批 至 2022 第七批), 每位 名老中医 配 2-3 名 学术继承人 三年带教, 出版传承专著, 这是 中医 实务知识 主体传承通道. 这两个 国家级项目 都 在 natcm.gov.cn 有 历批名单 + 工作总结 + 学术继承人名单 一手公告, 是 中医 figures 一手最稠密入口.

第二, **院士 + 学派代表 + 院校教师轨** 与 学术继承轨 在 院校 .edu.cn / CACMS .ac.cn 形成 双层呼应 — 北京中医药大学 王琦 + 黄煌 + 已故 刘渡舟 + 胡希恕 都 在 bucm.edu.cn / njucm.edu.cn 教研室 + 教师页 + 名师工作室 一手, 中科院 / 中国工程院 院士 (陈可冀 / 吴以岭 / 屠呦呦 / 张伯礼 / 王永炎 / 石学敏 / 黄璐琦 / 仝小林) 同时 在 cas.cn / cae.cn 院士名册 一手 + 在所属院校 / CACMS 教师页 一手 + 在国际期刊 (NEJM 屠呦呦 青蒿素诺奖 / Nature / Cell 等) 一手. 学派代表 — 经方派 (黄煌 / 已故 刘渡舟 / 已故 胡希恕 / 已故 陈瑞春) + 火神派 (已故 李可 / 卢崇汉 / 历史 郑钦安 + 已故 吴佩衡) + 体质学派 (王琦 创立) + 络病学派 (吴以岭) + 醒脑开窍 (石学敏) + 态靶辨治 (仝小林) + 五脏相关 (邓铁涛) + 衡法 (颜德馨) — 每个学派都 有 院校教师页 + 协会专委会 + 学术继承人 + 出版社专著 四重一手呼应.

第三, **中西医结合 + 现代研究轨** 是 中医 figures 国际化最强的一段 — 屠呦呦 2015 诺贝尔生理学或医学奖 (青蒿素 + 双氢青蒿素 抗疟) 是 中医药对现代医学贡献的标志性事件, 一手在 nobelprize.org + NEJM + Nature + cacms.ac.cn 中药研究所; 张亭栋 (已故 2023) 三氧化二砷 (砒霜) 治疗 急性早幼粒细胞白血病 APL 是 另一项 中药现代化全球认可成果, 一手在 哈尔滨医科大学 + 国际血液学期刊 Blood; 陈可冀 (中国科学院院士 西苑医院 中西医结合心血管) + 吴以岭 (中国工程院院士 络病学 + 通心络) 是 中西医结合 临床+基础 双轨代表. 这一段 figures 的 一手 多 在 院士单位 + 国际期刊 publisher, surrogate 必走 'vendor docs (国际期刊 publisher)' 或 'vendor docs (CACMS 国务院直属 国家级研究机构 + 监管业务下属 国家中医药管理局)' literal.

第四, **反面案例 严格学术批评 不入人身攻击** — 中医这条线 反例 选取必须比 ICP 那条线 (滴滴 / 知网 企业反例) 更小心: 张悟本 2010 "绿豆万能" 央视 + 卫生部 + 国家中医药管理局 联合表态 是 标志性 非法行医伪养生 案例, 一手在 国家卫生部 (现 NHC) 2010 通报 + 同期 NATCM 公开表态; 胡万林 1990s "芒硝大师" + 2000 年 一审 + 2013 年 二次 因非法行医致死 入狱, 一手在 最高法 / 各级法院 判决文书 + 公安机关通报; 倪海厦 (已故 2012, 海外 大量追随者, 部分论点 — 例如 否定 现代医学 抗肿瘤治疗 / 主张某些晚期癌症 单靠 中药治愈 — 缺乏循证 + 与现代肿瘤学界共识 严重冲突), 一手批评在 学术界 + 国际期刊 评述, 不引 微信公众号 / 自媒体粉丝战, 严守 "对论点的学术批评, 不对个人的人格攻击" 边界. 已故 figures 一律标 "已故 (YYYY)", 在世 figures 一律标 "在世" 与 现任职务, 不混淆.

第五, **一手率结构性偏低 但 一手类型多元** — 中医 figures 一手 由 (1) NATCM 国医大师 + 学术继承人 公告 (verified_primary .gov.cn) + (2) 院校 .edu.cn 教师页 (verified_primary) + (3) 协会 cacm.org.cn / caam.org.cn (verified_primary .org.cn) + (4) CACMS .ac.cn 教师页 (surrogate_primary + 'vendor docs (CACMS 国务院直属 国家级研究机构 + 监管业务下属 国家中医药管理局)') + (5) 院士单位 中科院 cas.cn + 中国工程院 cae.cn (verified_primary .cn .gov 边界, 实务 走 'vendor docs (国家级研究机构 + 监管)' 保守) + (6) 国际期刊 publisher (surrogate_primary + 'vendor docs (国际期刊 publisher)') 六层 互补, 严禁 mp.weixin.qq.com (中医博主 + 名老中医二代传承 + 中药企业营销主战场) / zhihu.com (中医辩论极端立场) / 抖音 / 快手 / 微博 / baike.baidu.com — 任何 中医 KOL / 养生博主 / 自媒体账号 都 不进 figure 列表, 必须 用 协会 + 院校 + NATCM + CACMS + 国际期刊 替代.
