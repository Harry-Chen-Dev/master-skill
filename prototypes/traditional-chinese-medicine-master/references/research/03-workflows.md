# Track 03 — Workflows: Traditional Chinese Medicine Clinical SOPs

中医诊疗 (Traditional Chinese Medicine) 这一行业的 SOP 主体由 **五方文档** 共同定义, 缺一不可: (1) **千年古籍辨证论治体系** (黄帝内经 / 伤寒论 / 金匮要略 / 温病条辨 / 神农本草经 / 本草纲目 — ctext.org 古籍数字化平台 + 中国哲学书电子化计划 + 国家中医药管理局古籍平台), 这是 中医临床的 灵魂层 不可剥离; (2) **国家监管一手通知**, 集中在 国家中医药管理局 `natcm.gov.cn` (95 个单病种中医临床路径 + 中医优势病种诊疗方案 + 中医诊所基本标准 + 中医诊所备案管理暂行办法 + ZY/T 001 病案规范) + 国家卫健委 `nhc.gov.cn` + 国家药监局 `nmpa.gov.cn` (中药注射剂警示 + 古代经典名方目录 + 不良反应监测年报) + 全国人大 `flk.npc.gov.cn` 中医药法全文 + 国家药典委员会 `chp.org.cn` (中国药典一部强制性国家标准); (3) **院校规划教材 vendor docs** (中国中医药出版社 `tcmbook.cn` + 人民卫生出版社 `pmph.com` 新世纪 + 十四五 规划教材 — 中医基础理论 / 中医诊断学 / 中医内科学 / 中医外科学 / 中医妇科学 / 中医儿科学 / 中医骨伤科学 / 中药学 / 方剂学 / 针灸学 / 推拿学 / 各家学说 共 26 所中医药大学共用); (4) **名老中医师承传承 一手** (国医大师 + 全国名老中医 + 学术继承人 项目 — 主要在 院校 .edu.cn 教师页 + CACMS .ac.cn 研究所 + 三甲中医院 名医工作室); (5) **中西医结合 + 国际循证文献** (Cochrane Library + PubMed + NEJM 1993 马兜铃酸肾病开山文献 + WHO ICD-11 第 26 章 Traditional Medicine Module + WHO Benchmarks for Training in TCM). **入门 vs 资深路径 显式 diff** 体现在: 规培医师 + 标准临床路径 (NATCM 95 病种诊疗方案 + 院校教材常规辨证) 是入门, 资深路径必须含 `跳过` (省略某些低效步骤) + `优化` (改良关键环节) + `额外` (加入名老中医独有的辨证或处方思路), 体现 千年师承 + 名老中医个体化辨证 + 学派特色 (经方派 / 温病派 / 滋阴派 / 温补派 / 攻邪派 / 易水派 / 中西汇通派 / 火神派) 的 显式 diff.

**临床警示绝不软化** — 急症 (中风 / 急性心脑血管 / 急性腹痛 / 急性呼吸困难 / 急性中毒 / 严重创伤) **西医必须优先**, 中医作为协同补充, 不可以中医辨证耽误西医急救黄金时间窗 (中风 4.5h 静脉溶栓时间窗 / STEMI 90min 介入时间窗 / 创伤性休克 1h 黄金时间); 中药安全严控 — 含马兜铃酸药材 (关木通 / 广防己 / 青木香 等已被国家药监局取消药用标准) 严禁使用, 何首乌 / 雷公藤 / 朱砂 / 雄黄 / 大黄 / 黄药子 / 苦楝皮 / 川楝子 等 肝肾毒性药严控剂量与疗程; 儿童 / 孕妇 / 老人 / 哺乳期 / 肝肾功能不全 患者用药特别注意, 妊娠禁忌 (大戟 / 芫花 / 甘遂 / 牵牛 / 巴豆 / 商陆 / 麝香 / 三棱 / 莪术 / 水蛭 / 虻虫 等) 一律禁用; **假中医严格区分** — 张悟本 / 胡万林 / 刘逢军 类 反面 canon 在 SOP 第 10 + 11 段反复出现, 用于校准 警示烈度, 假冒中医专家 + 非法行医 + 神医偏方 + 玄学风水包装中医 不是 "宣传过头" 而是 主体多年不可执业 + 刑事追究 + 患者死亡损害赔偿 的实质后果, SOP 绝不能软化.

**% 比例 一律 inline 加 caveat 标注** (协会公开 / NATCM 通告 / 院校公开数据 三类之一); 数字 仅作 SOP 阶段时长 / 通过率 / 疗程的量级提示, 不作 绝对依据. 中医诊疗 工作流独特三点 — (a) **辨证论治灵魂** 不可剥离, 同病异治 + 异病同治 是中医本质; (b) **中西医结合必然**, 1956 起院校化即与西医并行 70 年, 不存在 "纯中医" 闭门造车; (c) **名老中医师承** 必须 显式 diff, 入门按 教材常规辨证, 资深按 名医个体化经验方 + 学派特色加减, 这是中医 SOP 与西医标准化诊疗的 结构性差异.

## Workflows

### SOP-1. 中医门诊 标准诊疗 SOP (问诊→四诊→辨证→拟法→处方→病案→随访)

**适应症 / 适用情境**: 各类中医门诊常见病, 包括内科 (感冒 / 咳嗽 / 胃痛 / 失眠 / 头痛 / 眩晕 / 心悸 / 腰痛 / 痹证) / 妇科月经病 / 儿科常见病 / 慢性虚损调理. 患者 病情非急危重 + 生命体征平稳 + 神志清楚能配合问诊.

**禁忌 / 不适用 + 替代路径**: 急危重症 (中风急性期 / 急性胸痛疑 STEMI / 急性腹痛疑外科急腹症 / 高热抽搐 / 大出血 / 休克) → 立即转 SOP-2 急症诊疗 SOP, 西医急救优先; 神志不清 / 不能配合问诊 → 走 望诊 + 切诊 + 家属代述; 患者明确表达 仅愿西医治疗 → 尊重患者意愿不勉强 (中医药法第三十六条 患者知情同意).

**入门路径 (规培医师 / 标准临床路径)**:
1. **问诊 (15-25 min)**: 按"十问歌"(一问寒热二问汗 + 三问头身四问便 + 五问饮食六问胸 + 七聋八渴俱当辨 + 九问旧病十问因 + 妇女尤必问经期 + 小儿当问麻疹斑) 完整询问主诉 + 现病史 + 既往史 + 个人史 + 家族史 + 月经/婚育/小儿喂养. 陷阱: 规培生最常漏问"二便"+ 月经史 + 服药史 (包括西药 + 中药 + 保健品).
2. **望诊 (5-8 min)**: 望神 (得神/少神/失神/假神) + 望色 (五色主病) + 望形态 (胖瘦/姿势) + **望舌** (舌质 + 舌苔 + 舌下络脉; 自然光下 + 伸舌 5s 内观察避免染苔) + 望排出物. 陷阱: 染苔 (咖啡 / 杨梅 / 蛋黄 / 橄榄) 误判 + 厚白苔 vs 厚腻苔区分.
3. **闻诊 (2-3 min)**: 听声音 (语声 / 呼吸 / 咳嗽 / 呕吐) + 嗅气味 (口气 / 汗 / 二便). 陷阱: 慢支咳嗽 vs 哮喘喘鸣音 vs 心衰湿啰音 西医听诊互补.
4. **切诊 (5-10 min)**: 脉诊 (寸关尺 三部九候 / 浮中沉 / 28 脉象 主常见 浮沉迟数虚实滑涩弦紧促结代) + 按诊 (腹诊 / 肌肤 / 手足). 陷阱: 入门生 凭"脉象一字"草率定证, 必须 脉症合参, 舍症从脉 或 舍脉从症 是高阶判断.
5. **辨证 (10-15 min)**: 按 八纲辨证 (阴阳/表里/寒热/虚实) + 病因辨证 (六淫/七情/饮食劳倦) + 气血津液辨证 + 脏腑辨证 (内科主) / 六经辨证 (伤寒主) / 卫气营血辨证 (温病主) / 三焦辨证 (温病湿温主) 综合, 写出 完整证型 (如 "肝郁脾虚 兼痰湿内蕴"). 陷阱: 单一辨证体系套全部病 错误, 必须 病机决定辨证方法选择.
6. **拟法 + 选方 + 加减 (10-15 min)**: 按 治则 (扶正 / 祛邪 / 标本缓急 / 三因制宜) → 选 主方 (《方剂学》360+ 方) → 君臣佐使 加减 (主证用君药 + 兼证加佐药) → 用法用量 (汤剂 / 颗粒 / 中成药 / 针灸 / 推拿). 陷阱: 不辨证套用 "感冒就用 板蓝根 + 双黄连" 是 中成药滥用典型错误.
7. **病案书写 (10-15 min)**: 按 ZY/T 001 中医病案书写规范 写 主诉 + 现病史 + 既往史 + 望闻问切 + 中医诊断 (病名 + 证型) + 西医诊断 + 治法 + 方药 (君臣佐使) + 医嘱 + 随访计划. 陷阱: 病案 缺中医辨证依据 + 缺方药君臣佐使 + 缺中西医双诊断, 是 NATCM + 卫健委 病案抽查 最常见整改项.
8. **随访 + 复诊 调方 (7-14 天)**: 嘱咐 服药方法 (饭前 / 饭后 / 空腹 / 睡前) + 煎服法 (先煎 / 后下 / 包煎 / 烊化 / 冲服) + 饮食禁忌 + 复诊时间 (一般 7 天 / 急症 3 天 / 慢性 14-30 天). 陷阱: 不告知煎药细节 → 患者自煎错误 → 疗效差.

**资深路径 (名老中医 / 学派特色)**:
- **跳过 详细问诊耗时**: 资深医师 跳过 大量逐项问诊 → 优化 望神 + 望面色 + 望舌 + 切脉 直观主导 (3 秒 望神 + 望面色 即定 阴阳虚实大局, 望舌 + 切脉 5 分钟定 脏腑病位) → 额外 凭 "首诊三秒法" (患者进门 走姿 + 神态 + 面色) 5 分钟内完成 辨证 + 立法 + 处方雏形.
- **优化 脉诊主导**: 经方派 (黄煌) 资深医师 优化 切诊为 "六经体质 + 方证" 直观判断 (桂枝体质 / 麻黄体质 / 柴胡体质 / 大黄体质), 跳过 脏腑辨证细分, 直接 方证相应 出经方原方 (桂枝汤 / 小柴胡汤 / 大柴胡汤).
- **额外 取舌象 + 面色 + 手相 + 腹诊**: 滋阴派 (朱丹溪传承) 额外 看 面颧红 + 舌质红绛少苔 + 脉细数 直定 阴虚火旺; 温补派 (张景岳传承) 额外 看 面色㿠白 + 舌淡胖齿痕 + 脉沉迟无力 直定 阳气不足; 火神派 (郑钦安传承) 额外 看 唇白 + 舌淡水滑 直定 阳虚阴盛.
- **额外 加入名老中医独有经验方**: 国医大师王琦 体质九分法 加减 + 国医大师邓铁涛 五脏相关学说 五脏同调 + 国医大师朱良春 虫类药用于顽痹 + 国医大师路志正 脾胃调理万病 — 各 学派 + 名老中医 自有 数十首 独门验方 不在常规教材.
- **跳过 院校标准方**: 资深名老中医 跳过 教材常规辨证选方 → 优化 自创经验方 (如邓铁涛 强肌健力饮治重症肌无力 / 仝小林 葛根芩连汤化裁治糖尿病 / 黄煌 葛根汤治颈椎病) 直接 出方.

**常见陷阱**:
- **陷阱 1: 不辨证套方** — "感冒就板蓝根 + 失眠就酸枣仁汤 + 胃痛就香砂养胃丸" 不辨证 = 不是中医, 是 中成药商业化误导.
- **陷阱 2: 单一辨证体系套全部病** — 八纲套伤寒 + 脏腑套温病 + 六经套杂病 — 必须 病机决定辨证方法选择, 多体系互补.
- **陷阱 3: 病案缺中医辨证依据** — NATCM + 卫健委 病案质控抽查最常见整改项; 必须 主诉 + 四诊 + 辨证 + 治法 + 方药 + 医嘱 完整链.
- **陷阱 4: 中药滥用没禁忌** — 妊娠禁忌 (大戟 / 芫花 / 甘遂 / 麝香 / 三棱 / 莪术) / 十八反 (乌头反半夏栝楼贝母白蔹白及; 甘草反甘遂大戟海藻芫花; 藜芦反人参沙参丹参玄参细辛芍药) / 十九畏 必须严守.
- **陷阱 5: 忽视西医诊断** — 中医病名 + 西医病名 双轨; 中医 "胸痹" 必须 排除 急性 STEMI; 中医 "中风" 必须 排除 脑出血 vs 脑梗 (CT/MRI 西医诊断为先).

**监管要点 + 安全警示**: NATCM + NHC 双重监管; 病案按 ZY/T 001 规范; 处方按 中药处方管理规定; 中医诊所备案制 + 中医执业医师执业证书强制; 急症必须中西医结合 + 西医优先 + 中医协同 (不可以中医耽误西医急救); 妊娠 / 儿童 / 老人 / 肝肾功能不全 患者特别注意 中药用药剂量与禁忌.

**evidence**: [T03-S001 / T03-S002 / T03-S011 / T03-S018 / T03-S022 / T03-S023 / T03-S033 / T03-S036 / T03-S049 / T03-S050]

### SOP-2. 急症 中医诊疗 SOP (西医优先 + 中医协同 + 醒脑开窍)

**适应症 / 适用情境**: 中医急症范畴 包括 中风急性期 (脑卒中) / 急性心脑血管事件 (胸痹心痛 / 厥证) / 高热神昏 (热入营血 / 暑厥) / 急性呕吐 / 急性腹泻 (霍乱 / 急性食物中毒) / 急性胸痛 / 急性腹痛 + 中医院急诊 + 综合医院中医科会诊.

**禁忌 / 不适用 + 替代路径**: 心搏骤停 → 立即 CPR + AED + 转 ICU, 中医不能延误; 大出血 / 创伤性休克 → 立即外科 + 输血 + 抗休克; 张力性气胸 → 立即胸腔闭式引流; 急性心梗 → 立即介入或溶栓 (DTB ≤ 90min 时间窗) — 任何 急危重症 西医优先 + 中医协同 不是 "中西医平等", 是 西医先救命 + 中医后协同恢复; 患者拒绝中医 → 尊重患者意愿不勉强.

**入门路径 (规培急诊 + 中医协同)**:
1. **生命体征评估 + 西医急救分诊 (1-3 min)**: ABC (Airway 气道 / Breathing 呼吸 / Circulation 循环) + 意识 (GCS) + 瞳孔 + 血压 + 心率 + 血氧 + 血糖 (床旁指血) — 与 西医急诊完全一致; 中风疑似立即 FAST (Face / Arm / Speech / Time) 评估 + 急诊 CT 排除脑出血. 陷阱: 中医医师 凭 "脉象 + 望神" 不做西医急救评估直接开中药 = 严重违规 + 患者死亡责任.
2. **西医急救启动 (立即并行)**: 中风 4.5h 时间窗内 启动 静脉溶栓评估 + 6h 内 介入取栓评估; STEMI 启动 双抗 + 介入 (DTB ≤ 90min) / 急诊溶栓 (DTN ≤ 30min); 高热抽搐 物理降温 + 抗惊厥; 急腹症 启动 外科评估; 失血性休克 启动 输血 + 抗休克 — **西医急救与中医协同绝不可二选一**. 陷阱: 中医院急诊 西医急救能力不足 必须 与综合医院建立绿色转诊通道 (NATCM 急诊中心建设要求).
3. **中医四诊快速评估 (3-5 min)**: 望神 + 望色 + 望舌 + 切脉 + 简化问诊 (主诉 + 既往史 + 用药史 + 过敏史) — 重点判断 闭证 vs 脱证 / 阳闭 vs 阴闭 / 实证 vs 虚证. 陷阱: 中风 闭证 (神昏 + 牙关紧闭 + 两手握固) vs 脱证 (神昏 + 目合口张 + 手撒尿遗 + 汗多肢冷) 处方完全相反, 闭证开窍 (安宫牛黄丸 / 紫雪丹 / 苏合香丸) / 脱证 回阳救逆 (参附汤 / 独参汤).
4. **中医急救协同 (与西医并行)**: 中风急性期 醒脑开窍针刺法 (石学敏 国医大师法 — 内关 + 人中 + 三阴交 + 极泉 + 尺泽 + 委中) + 安宫牛黄丸 (阳闭) 或 苏合香丸 (阴闭) 鼻饲, 与 西医静脉溶栓 / 介入取栓 并行; 高热神昏 紫雪丹 / 安宫牛黄丸 + 物理降温; 胸痹心痛 速效救心丸 / 复方丹参滴丸 含服 (但不能替代 硝酸甘油 + 阿司匹林 + 介入). 陷阱: 安宫牛黄丸 含 朱砂 + 雄黄 (含汞 + 砷), 不可长期大剂量, 急救 1-2 丸即可.
5. **病情观察 + 中西医联合调整 (持续)**: 每 30-60 min 评估 生命体征 + 神志 + 中医证候演变 (闭证转脱证警示) + 西药 + 中药相互作用 (华法林 + 丹参 + 当归 出血风险); 与 西医急诊医师 + 重症医学 联合制定治疗方案. 陷阱: 中药与西药相互作用 (银杏 + 阿司匹林 + 华法林 = 出血风险) 中医医师 必须查阅文献.
6. **病案记录 + 多学科会诊 + 转归判断 (持续)**: 病案 按 ZY/T 001 规范 + 急诊抢救记录 + 中医急救操作记录 (针刺时间 / 穴位 / 手法 / 留针 / 出针); 多学科会诊 (神经内科 + 心内科 + ICU + 中医). 陷阱: 中医急救操作 (针刺 / 中药鼻饲) 必须 病案 完整记录 + 患者 / 家属 知情同意签字.

**资深路径 (名老中医急症 + 学派特色)**:
- **跳过 标准教材辨证**: 资深急诊中医师 跳过 详细六经辨证 → 优化 直接 "闭证 vs 脱证 / 阳闭 vs 阴闭" 二选一 决定 醒脑开窍 vs 回阳救逆 → 额外 中风急性期 直接 醒脑开窍针刺 (石学敏国医大师 + 天津中医药大学 第一附属医院 全国推广法) 1-2 min 内出针开始治疗.
- **优化 针刺 + 中药 + 西医 三轨并行**: 资深医师 优化 治疗启动顺序 — 针刺 (1 min 内开始, 醒脑开窍 / 急救穴位) + 中药鼻饲 (3-5 min 内安宫牛黄丸 / 苏合香丸) + 西医溶栓评估 (并行) — 三轨并行 不串行.
- **额外 火针 / 放血 / 雷火灸**: 资深医师 额外 加入 火针 (顽固肿痛 / 阴疽) / 三棱针放血 (中暑 / 高热 / 局部瘀血) / 雷火灸 (寒证 / 阳虚 / 风湿). 陷阱: 火针 + 放血 必须 严格 无菌操作 + 知情同意.
- **跳过 院校标准方**: 经方派资深医师 急症 跳过 后世温病方 → 直接 经方原方 (大柴胡汤 + 大承气汤治急腹症 / 白虎汤 + 安宫牛黄丸治高热神昏 / 四逆汤治休克阳脱) — 量大力宏 急症显效.

**常见陷阱**:
- **陷阱 1: 中医医师不做西医急救评估** — 中风开窍 + 高热神昏 + 急性胸痛 都必须 西医急救评估为先 + 中医协同, 凭脉象草率开中药 = 严重违规 + 患者死亡责任.
- **陷阱 2: 闭证 vs 脱证 误判** — 中风 闭证 vs 脱证 处方完全相反; 闭证开窍 / 脱证回阳; 错用 后果严重.
- **陷阱 3: 安宫牛黄丸 长期大剂量** — 含朱砂 (HgS) + 雄黄 (As2S3) 汞 + 砷重金属, 急救 1-2 丸即可, 长期服用 重金属中毒.
- **陷阱 4: 中药与西药相互作用忽视** — 银杏 + 阿司匹林 / 丹参 + 华法林 / 麻黄 + MAOI / 甘草 + 利尿剂 — 必须 用药史完整查 + 药师审核.
- **陷阱 5: 中医急救操作病案缺失** — 针刺时间 / 穴位 / 手法 / 中药鼻饲 必须 完整病案记录 + 知情同意签字.

**监管要点 + 安全警示**: 急诊抢救必须 西医优先 + 中医协同; NATCM 急诊中心建设要求 中医院急诊与综合医院建立绿色转诊通道; 中风时间窗 (4.5h 静脉溶栓 / 6h 介入取栓) / STEMI 时间窗 (DTB ≤ 90min) 绝不能因中医辨证耽误; 安宫牛黄丸 / 紫雪丹 含朱砂雄黄 急救短期使用; 中药与西药相互作用 必查; 病案 + 知情同意 必须规范; **急症西医优先 + 中医协同 不可软化**.

### SOP-3. 慢性内科 中医诊疗 SOP (冠心病 / 高血压 / 糖尿病 / COPD / 哮喘 / 慢肾炎 / 慢性肝病 / 风湿病 / 抑郁焦虑 — 长期管理 + 名老中医经验方)

**适应症 / 适用情境**: 慢性内科常见病长期管理 — 心血管 (冠心病稳定期 / 高血压 / 心力衰竭代偿期) / 内分泌 (2 型糖尿病 / 甲亢甲减 / 肥胖代谢综合征) / 呼吸 (COPD 稳定期 / 哮喘缓解期 / 肺间质病变) / 肾内 (慢性肾炎 / 慢性肾脏病 CKD 1-4 期 / IgA 肾病) / 消化 (慢性胃炎 / GERD / 肝硬化代偿期 / 脂肪肝) / 风湿 (类风湿关节炎 / 痛风 / 系统性红斑狼疮稳定期) / 神经精神 (抑郁症轻中度 / 焦虑症 / 失眠 / 帕金森).

**禁忌 / 不适用 + 替代路径**: 急性失代偿 (急性心衰 / 急性肾衰 / 哮喘重度急性发作 / 类风湿急性活动 / SLE 危象 / 重度抑郁 自杀风险) → 转 SOP-2 急症 + 西医优先; 终末期 (CKD 5 期需透析 / 肝功能衰竭 / 心衰 NYHA IV 级) → 中医协同 + 姑息医学.

**入门路径 (规培内科 + 中医长期管理)**:
1. **首诊 全面评估 + 双轨诊断 (40-60 min)**: 西医病名 (ICD-10/11) + 西医分期分级 (NYHA / Child-Pugh / CKD 分期 / GINA / GOLD / DAS-28) + 中医病名 (胸痹 / 消渴 / 哮病 / 喘证 / 水肿 / 鼓胀 / 痹证 / 郁证) + 中医证型 (心气虚 / 心阳虚 / 心血瘀阻 / 痰浊闭阻 / 阴虚火旺 / 肝肾阴虚 等); 完整 四诊 + 西医辅查 (心电图 / 心超 / 血糖 + HbA1c / 肌酐 + eGFR / 肺功能 / 风湿四项 + 抗 CCP / 汉密尔顿抑郁焦虑量表). 陷阱: 慢性病管理 中医忽视西医分期分级 + 西医忽视中医证型 — 必须 双轨整合.
2. **长期管理方案 制定 (30-45 min)**: 西药 维持 (降压 / 降糖 / 调脂 / 抗血小板 / 免疫抑制 / 吸入支扩) + 中药 调治 (主方 + 加减 + 用药周期一般 4-12 周一调整) + 非药物 (针灸 / 推拿 / 八段锦 / 太极 / 食疗 + 调摄). 陷阱: 中药 与 西药相互作用 必须查 (人参 + 华法林 / 麻黄 + MAOI / 甘草 + 噻嗪类利尿剂导致低钾).
3. **中药调治 长程方案 (4-12 周 / 调方)**: 按 中医证型选 主方 (冠心病 痰瘀闭阻 用 瓜蒌薤白半夏汤 + 血府逐瘀汤合方 / 高血压 肝阳上亢 用 天麻钩藤饮 / 2 型糖尿病 阴虚燥热 用 玉女煎或消渴方 / COPD 肺脾肾虚 用 玉屏风散 + 六君子汤 + 金匮肾气丸 / 慢性肾炎 脾肾阳虚 用 真武汤 + 实脾饮 / 风湿病 寒湿痹阻 用 乌头汤 / 抑郁症 肝郁气滞 用 柴胡疏肝散 / 逍遥散) — 4-12 周 一调整. 陷阱: 慢性病 中药不能长期不调方, 必须 4-12 周一评估 + 调整, 否则 证变方不变 = 不是中医.
4. **针灸 / 推拿 / 食疗 + 调摄 整合 (持续)**: 针灸 (冠心病 内关 + 神门 + 心俞 + 厥阴俞 / 高血压 太冲 + 风池 + 三阴交 / 糖尿病 胰俞 + 脾俞 + 足三里 / COPD 肺俞 + 膏肓 + 定喘) + 食疗 (糖尿病 苦瓜 + 山药 + 黄连 / 高血压 芹菜 + 决明子 + 菊花) + 调摄 (情志 + 季节 + 起居). 陷阱: 食疗只是 辅助, 不能 替代降糖降压药.
5. **病情评估 + 调方 (每 4-12 周)**: 西医指标 (血压 / HbA1c / 肌酐 / 肺功能 / DAS-28 / 量表) + 中医证候演变 (寒热 / 虚实 / 痰瘀 变化) + 患者自评 (生活质量 / 主诉) + 复诊调方. 陷阱: 西医指标改善 但 中医证候未变 → 仍 须调方; 西医指标恶化 → 立即 联合西医专科调整.
6. **多学科会诊 + 长期随访 (持续)**: 与 西医专科 (心内 / 内分泌 / 肾内 / 呼吸 / 风湿 / 精神) 定期会诊; 长期随访 (1-3 月) 评估 治疗效果 + 并发症预防 + 患者教育. 陷阱: 慢性病 患者教育 (服药依从 + 饮食 + 运动 + 情志) 是 长期管理的核心, 不是 附加.

**资深路径 (国医大师经验 + 学派特色 + 个体化辨证)**:
- **跳过 教材常规辨证**: 资深中医师 跳过 教材证型套用 → 优化 名老中医独有 病机认识 (国医大师仝小林 糖尿病 "态靶因果" 辨治 — 跳过 阴虚燥热常规 → 直接 "郁热虚损" 四态 + 现代药理"靶药" 葛根 + 知母 + 黄连 + 生地) → 额外 经方化裁 葛根芩连汤 + 大柴胡汤治糖尿病代谢综合征.
- **优化 五脏相关学说**: 国医大师邓铁涛 五脏相关学说 优化 心血管病治疗 — 跳过 单纯活血化瘀常规 → 直接 心脾肾同调 (补益心气 + 调理脾胃 + 温补肾阳) → 额外 邓老经验方 "强心饮" (太子参 + 麦冬 + 五味子 + 法半夏 + 茯苓 + 橘红 + 田七 + 三七).
- **优化 脾胃调治万病**: 国医大师路志正 脾胃调治万病 优化 — 跳过 单纯辨脏腑 → 直接 调脾胃为核心 → 额外 路老经验方 "调脾胃方" (太子参 + 白术 + 茯苓 + 陈皮 + 半夏 + 苍术 + 厚朴 + 砂仁 + 木香 + 鸡内金) 治疗 慢性消化系统疾病.
- **额外 虫类药顽痹**: 国医大师朱良春 风湿病 额外 加入 虫类药 (蜈蚣 + 全蝎 + 地龙 + 僵蚕 + 乌梢蛇 + 蕲蛇 + 土鳖虫) 治疗 顽固性类风湿 + 强直性脊柱炎 + 痛风 — 远超 院校教材常规.
- **额外 火神派扶阳**: 火神派 (郑钦安 / 卢崇汉 / 李可 传承) 资深医师 额外 大剂量四逆汤 + 附子 (30-100g 先煎 2h) + 干姜 + 肉桂 + 鹿茸 — 治疗 阳虚体质慢性病 (心衰 / 慢肾衰 / 慢性疲劳综合征) — 但 附子毒性 + 心律失常风险 必须 严控.
- **跳过 单一学派局限**: 资深名医 跳过 单一学派 → 优化 多学派融汇 (经方 + 时方 + 温病 + 滋阴 + 温补 灵活组合) → 额外 个体化辨证 + 体质辨识 (王琦 体质九分法) 加减.

**常见陷阱**:
- **陷阱 1: 慢性病中药长期不调方** — 必须 4-12 周一调, 否则 证变方不变 = 不是中医; 中医灵魂在 辨证论治 + 随证调方.
- **陷阱 2: 中药与西药相互作用忽视** — 人参 + 华法林 / 麻黄 + MAOI / 甘草 + 噻嗪类 / 银杏 + 阿司匹林 — 必须 查阅 PubMed + Cochrane.
- **陷阱 3: 食疗替代药物治疗** — 苦瓜不能替代二甲双胍 + 芹菜不能替代降压药 — 食疗是辅助, 不是替代.
- **陷阱 4: 火神派大剂量附子未严控** — 附子 30-100g 必须 先煎 2h + 配伍干姜炙甘草 解毒 + 心电监护; 误用 + 急性附子中毒 + 心律失常 + 死亡.
- **陷阱 5: 中医忽视西医分期分级** — CKD 5 期 + 心衰 NYHA IV 级 + 严重肝衰 不再适合积极中药调治, 应 协同姑息医学.

**监管要点 + 安全警示**: 慢性病管理必须 中西医结合 + 双轨诊断 + 中药 4-12 周调方; 中药与西药相互作用 必查; 火神派大剂量附子 严格炮制 + 先煎 + 配伍 + 心电监护; 妊娠 / 儿童 / 老人 / 肝肾不全 患者 中药剂量调整; 长期随访 + 患者教育; 中药 4-12 周不调方 + 食疗替代药物 + 大剂量附子未严控 = 中医临床三大误区.

### SOP-4. 妇科 中医诊疗 SOP (月经病 / 带下 / 不孕 / 围绝经 / 产后)

**适应症 / 适用情境**: 中医妇科 五大类 — 月经病 (月经先期 / 后期 / 先后无定期 / 量多 / 量少 / 闭经 / 痛经 / 崩漏 / 经断前后诸证) + 带下病 (黄白赤带) + 妊娠病 (恶阻 / 胎漏 / 胎动不安 / 滑胎 / 子肿 / 子晕) + 产后病 (产后血晕 / 产后发热 / 产后腹痛 / 产后恶露不绝 / 产后缺乳 / 产后抑郁) + 妇科杂病 (不孕 / 围绝经期综合征 / 子宫脱垂 / 慢性盆腔炎 / 子宫肌瘤 / 子宫内膜异位症).

**禁忌 / 不适用 + 替代路径**: 急性失血 / 失血性休克 → 立即西医输血 + 急诊宫腔操作; 异位妊娠 / 卵巢黄体破裂 → 立即外科; 胎盘早剥 + 前置胎盘出血 → 立即产科急救; 重度子痫前期 / 子痫 → 立即西医急救; 恶性肿瘤 (宫颈癌 / 子宫内膜癌 / 卵巢癌 / 乳腺癌) → 西医手术 + 化疗 + 放疗 优先, 中医协同.

**入门路径 (规培妇科 + 中医妇科辨证)**:
1. **十问 + 妇科专项问诊 (15-25 min)**: 十问歌 + 月经史 (初潮 / 周期 / 经期 / 经量 / 经色 / 经质 / 经期不适 + LMP) + 婚育史 (孕产 / 流产 / 哺乳) + 性生活史 + 避孕史 + 生殖系统手术史. 陷阱: 妇科辨证 月经史 是 核心, 月经 周期 / 量 / 色 / 质 + 经期不适 直接 决定 寒热虚实辨证.
2. **望闻问切 + 妇科查体 + 西医辅查 (20-30 min)**: 望神色 + 望面色 + 望舌 + 切脉 + 妇科常规查体 (外阴 / 阴道 / 宫颈 / 双合诊) + 西医辅查 (B 超 + 性激素六项 + 甲状腺功能 + 妇科肿瘤标志物 + HPV + TCT + 必要时 MRI / 宫腔镜 / 腹腔镜). 陷阱: 中医妇科师 拒绝做西医妇科查体 + 拒绝看 B 超 — 错失 子宫肌瘤 / 内异症 / 多囊卵巢综合征 PCOS 等器质性病变.
3. **中医妇科 辨证 (15-20 min)**: 按 脏腑 (肝郁 / 脾虚 / 肾虚) + 气血 (气虚 / 血虚 / 气滞 / 血瘀) + 寒热 (寒凝 / 热盛) + 痰湿 综合辨证 — 月经先期 多 血热 / 气虚 + 月经后期 多 血寒 / 血虚 / 气滞 + 痛经 多 寒凝 / 血瘀 / 气滞 / 肾虚 + 闭经 多 肝肾不足 / 气血亏虚 / 痰湿 + 不孕 多 肾虚 / 肝郁 / 痰湿 / 血瘀 + 围绝经期 多 肾阴虚 / 肾阳虚 / 肾阴阳两虚.
4. **拟法 + 选方 + 加减 (10-15 min)**: 主方按 中医妇科学 教材 — 痛经 (温经汤 / 少腹逐瘀汤 / 调肝汤 / 益肾调经汤) / 月经先期 (清经散 / 两地汤 / 补中益气汤) / 月经后期 (温经汤 / 桃红四物汤 / 大营煎 / 当归补血汤) / 闭经 (归肾丸 / 苍附导痰丸 / 桃红四物汤) / 崩漏 (固冲汤 / 安冲汤 / 清热固经汤) / 不孕 (毓麟珠 / 开郁种玉汤 / 启宫丸 / 少腹逐瘀汤) / 围绝经期 (二仙汤 / 左归丸 + 右归丸 / 知柏地黄丸).
5. **针灸 / 推拿 + 食疗 + 调摄 (持续)**: 针灸 (痛经 三阴交 + 关元 + 中极 + 子宫穴 / 月经病 关元 + 气海 + 三阴交 + 血海 / 不孕 关元 + 中极 + 子宫 + 三阴交 + 命门) + 艾灸 (寒证 + 虚证) + 食疗 (痛经 红糖姜茶 + 乌鸡白凤丸 / 围绝经 百合莲子) + 调摄 (情志 + 起居 + 节饮食).
6. **中西医结合 + 长期随访 (4-12 周 一调)**: 与 西医妇科 联合 — 多囊卵巢综合征 中药调经 + 西药促排卵 / 子宫内膜异位症 中药活血化瘀 + 西药假绝经 / 不孕 中药调理 + 西医辅助生殖 IVF / 围绝经期 中药 + 西医激素替代 HRT (个体化). 陷阱: 妊娠期 中药严守 妊娠禁忌 (大戟 / 芫花 / 甘遂 / 牵牛 / 巴豆 / 商陆 / 麝香 / 三棱 / 莪术 / 水蛭 / 虻虫 / 桃仁 / 红花 / 大黄 / 番泻叶 慎用).

**资深路径 (名医妇科经验 + 学派特色)**:
- **跳过 教材标准方**: 资深妇科名医 跳过 教材常规方 → 优化 自创经验方 (国医大师夏桂成 调周法 治不孕 / 国医大师班秀文 调经种子方 治不孕 / 罗元恺 滋肾育胎丸 治先兆流产) → 额外 个体化辨证.
- **优化 中医周期疗法 (调周法)**: 资深医师 优化 月经周期分四期 — 经后期 (滋阴养血) + 经间期 (滋阴助阳促排卵) + 经前期 (温肾助阳) + 行经期 (活血化瘀) — 跳过 单一周期固定方, 额外 按月经周期 4 期 + 不同证型 + 不同方剂分期调治, 是 现代中医妇科 调周法 核心.
- **额外 中西医结合辅助生殖**: 资深医师 额外 配合 IVF — 取卵前 中药 调经 + 滋肾养血 + 取卵后 中药 + 针灸 提高胚胎着床率 + 妊娠后 中药 安胎 — Cochrane 已有 SR 支持 针灸 + 中药 提高 IVF 成功率.
- **优化 内异症 + 多囊 个体化辨治**: 资深医师 优化 内异症 痛经 + 不孕 — 跳过 单一活血化瘀 → 直接 "化瘀 + 软坚 + 散结 + 调经" 四联 (桂枝茯苓丸 + 桃红四物汤 + 三棱 + 莪术 + 山慈姑 + 鳖甲); 多囊 跳过 单一补肾 → 直接 "化痰 + 祛湿 + 活血 + 调经" 四联 (苍附导痰丸 + 桂枝茯苓丸 + 当归芍药散).

**常见陷阱**:
- **陷阱 1: 拒绝西医妇科查体 + 不看 B 超** — 错失 子宫肌瘤 + 内异症 + 多囊 + 宫颈癌 + 卵巢癌 等 器质性病变.
- **陷阱 2: 妊娠期忽视妊娠禁忌** — 大戟 + 芫花 + 甘遂 + 麝香 + 三棱 + 莪术 + 水蛭 + 虻虫 等 严禁; 桃仁 + 红花 + 大黄 + 番泻叶 慎用.
- **陷阱 3: 围绝经期单一治法** — 围绝经期 涉及 肾阴虚 + 肾阳虚 + 心肾不交 + 肝郁气滞 + 痰瘀互结 多证型, 必须 个体化辨证.
- **陷阱 4: 不孕单一治法** — 不孕涉及 肾虚 + 肝郁 + 痰湿 + 血瘀 多种病机 + 男方因素 + 输卵管 + 卵巢储备 — 必须 中西医结合 + 男女同治.
- **陷阱 5: 中药调经替代西医辅助生殖** — 高龄不孕 + 卵巢储备低下 + 输卵管堵塞 必须 西医辅助生殖优先, 中药协同, 不可纯中药耽误.

**监管要点 + 安全警示**: 妇科中医必须 中西医结合 + 西医妇科查体 + B 超 必做; 妊娠期 中药严守 妊娠禁忌; 围绝经期 个体化辨证; 不孕 男女同治 + 中西医结合 + 辅助生殖; 高龄不孕 + 严重器质性病变 西医优先; 中医妇科 经验方 必须 病案记录 + 知情同意.

### SOP-5. 儿科 中医诊疗 SOP (感冒 / 咳嗽 / 厌食 / 遗尿 / 多动症 + 小儿推拿)

**适应症 / 适用情境**: 中医儿科常见病 — 肺系 (感冒 / 咳嗽 / 肺炎喘嗽 / 哮病) + 脾系 (厌食 / 积滞 / 泄泻 / 便秘 / 疳证) + 心肝系 (惊风 / 多动症 ADHD / 抽动症 Tourette / 自闭谱系障碍) + 肾系 (遗尿 / 五迟五软 / 性早熟) + 时行病 (麻疹 / 风疹 / 水痘 / 手足口病). 儿科有效年龄 0-14 岁.

**禁忌 / 不适用 + 替代路径**: 新生儿期重症 (新生儿黄疸重度 / 新生儿肺炎重度 / 新生儿败血症) → 西医新生儿科 + NICU; 急性传染病高传染期 (麻疹 / 水痘 / 手足口病重症) → 西医隔离 + 抗病毒 + 对症; 急性中毒 / 急性意外伤害 → 西医急救; 川崎病 / 急性肾炎 / 紫癜性肾炎 重症 → 西医优先 + 中医协同; 重症感染 (脑膜炎 / 脓毒症) → 西医抗生素 + ICU.

**入门路径 (规培儿科 + 中医儿科辨证)**:
1. **儿科特殊问诊 + 家长配合 (10-15 min)**: 患儿主诉 (家长代述为主, 3 岁以上可直接询问) + 现病史 + 既往史 (出生史 / 喂养史 / 既往疾病 / 预防接种 / 过敏史) + 家族史 + 二便 + 饮食 + 睡眠. 陷阱: 儿科问诊 必须 家长全程陪伴 + 与患儿建立信任 (玩具 + 温和语调 + 蹲下平视).
2. **儿科望诊 + 闻诊 + 切诊 + 西医查体 (10-15 min)**: 望神色 (儿科神色诊断比成人更直接) + 望面色 + 望苗窍 (鼻 / 口 / 眼 / 耳 / 二阴) + 望舌 + 望指纹 (3 岁以下用 — 三关测轻重 + 红紫辨寒热) + 听啼哭 + 听咳嗽 + 切诊 (一指定三关 — 因儿童寸口短小) + 西医查体 (体温 / 呼吸 / 心率 / 听诊 / 咽部 / 颈淋巴结 / 腹部 / 神经系统). 陷阱: 儿科脉诊 3 岁以下 必须 看指纹 + 一指定三关 + 西医听诊.
3. **儿科辨证 + 双轨诊断 (10-15 min)**: 中医儿科特点 "脏腑娇嫩 + 形气未充 + 易虚易实 + 易寒易热" + 辨证 (脏腑 + 气血 + 寒热虚实 + 痰瘀); 西医双轨诊断 + ICD-10/11; 注意 儿科 特殊证型 — 风寒咳嗽 + 风热咳嗽 + 痰湿咳嗽 + 痰热咳嗽 + 阴虚燥咳; 厌食 脾失健运 + 脾胃气虚 + 脾胃阴虚; 遗尿 下元虚寒 + 肺脾气虚 + 肝经郁热; 多动症 肝肾阴虚 + 心脾两虚 + 痰火扰心.
4. **儿科 中药 + 小剂量 + 矫味 (10-15 min)**: 儿科 中药剂量 一般成人 1/2-1/3 (1 岁以下 1/4-1/5, 1-3 岁 1/3, 3-7 岁 1/2, 7-14 岁 2/3-1); 矫味 (蜂蜜 1 岁以上 / 冰糖 / 红糖) 提高 口服依从; 选 儿科专用方 (小儿七星茶 / 王氏保赤丸 / 健儿清解液 / 小儿豉翘清热颗粒 / 小儿肺热咳喘口服液) + 经方化裁 (桂枝汤 + 麻杏石甘汤 + 银翘散 + 桑菊饮). 陷阱: 1 岁以下 严禁 蜂蜜 (肉毒杆菌孢子风险); 儿科 严禁含朱砂 + 雄黄 + 马兜铃酸 中成药 (婴幼儿肝肾未发育完全).
5. **小儿推拿 (5-15 min / 次)**: 小儿推拿 独立体系 — 五经 (脾经 / 肝经 / 心经 / 肺经 / 肾经) + 八卦 + 六腑 + 三关 + 天河水 + 二人上马 + 七节骨 + 龟尾 + 捏脊; 操作 (推 / 揉 / 拿 / 运 / 掐 / 捏) + 介质 (滑石粉 / 婴儿油 / 蛋清). 陷阱: 小儿推拿 必须 短时 (5-15 min) + 力度轻 (婴幼儿如蚂蚁爬行) + 方向严格 (补 vs 泻 方向相反).
6. **儿科长期随访 + 家长教育 (持续)**: 儿科疾病 多 急性 + 反复; 长期随访 + 家长教育 (喂养 / 起居 / 预防 / 早期识别危重信号) + 体质调理 (王琦体质九分法 + 小儿特殊体质). 陷阱: 家长 中药煎服细节 + 服药方法 + 剂量 必须 详细书面交代.

**资深路径 (儿科名医 + 小儿推拿流派)**:
- **跳过 教材标准方**: 资深儿科中医师 跳过 教材常规方 → 优化 自创经验方 (国医大师汪受传 + 江育仁 等 江苏儿科派 / 山东张奇文 山东儿科派 / 上海董廷瑶 沪派儿科) → 额外 病机独到见解 + 经验方.
- **优化 小儿推拿主导**: 资深医师 优化 小儿推拿 为 主治法 (1-6 岁儿童 + 慢性反复感冒咳嗽 + 厌食 + 遗尿 + 多动) 跳过 中药 — 单纯小儿推拿 1-2 周 1 疗程 显效, 是 国医大师张素芳 推拿派 + 山东齐鲁推拿 主流; 额外 季节性预防保健推拿 (春季疏肝 / 夏季清心 / 秋季养肺 / 冬季补肾).
- **优化 捏脊 + 摩腹 + 揉中脘**: 资深医师 优化 小儿厌食 + 反复感冒 — 跳过 复杂多穴推拿 → 直接 捏脊 (从龟尾到大椎, 3-5 遍) + 摩腹 (顺时针 100 次) + 揉中脘 (2-3 min) + 揉足三里 (1-2 min), 5 min 内完成日常保健.
- **额外 中药外敷 + 穴位贴敷**: 资深医师 额外 三伏贴 (冬病夏治 — 大椎 + 肺俞 + 膏肓 + 定喘 治哮喘 + 反复感冒) + 三九贴 (夏病冬治) + 中药脐贴 (丁桂儿脐贴 治泄泻) — 远比 内服中药 患儿 + 家长依从性高.
- **跳过 ADHD / 抽动症单一治法**: 资深医师 跳过 单一中药或西药 → 优化 中西医结合 (西药哌甲酯 + 中药调肝肾 + 针灸 + 推拿 + 行为治疗 多模式) → 额外 王季儒 + 王孟英 经验方 化裁.

**常见陷阱**:
- **陷阱 1: 儿科剂量按成人** — 儿科 中药剂量 必须 按年龄 / 体重 严格调整 (1 岁 1/4-1/5; 1-3 岁 1/3; 3-7 岁 1/2; 7-14 岁 2/3-1).
- **陷阱 2: 婴幼儿用含朱砂 + 雄黄 + 马兜铃酸中成药** — 婴幼儿肝肾未发育完全, 朱砂 (汞) + 雄黄 (砷) + 马兜铃酸 严禁; 安宫牛黄丸 + 至宝丹 + 紫雪丹 婴幼儿 严禁长期.
- **陷阱 3: 1 岁以下用蜂蜜** — 肉毒杆菌孢子风险, 1 岁以下蜂蜜严禁.
- **陷阱 4: 小儿推拿力度过重** — 婴幼儿推拿 力度 如蚂蚁爬行, 力度过重 + 推拿方向错 + 时间过长 都 适得其反.
- **陷阱 5: 反复感冒咳嗽 滥用抗生素 / 中成药** — 儿科 反复感冒 多 体质问题, 不是 抗生素能解决 + 不是 中成药能解决, 必须 体质调理 + 调摄.

**监管要点 + 安全警示**: 儿科中医必须 严格年龄剂量 + 儿科专用方 + 矫味提高依从 + 家长教育; 婴幼儿严禁 朱砂 + 雄黄 + 马兜铃酸 + 蜂蜜 (1 岁以下); 小儿推拿 短时 + 轻力 + 方向正确; 重症 / 急症 / 传染病 西医优先; 儿科病案 + 知情同意 必须规范; **儿科特别注意 不可成人剂量套用 + 不可滥用毒性中药**.

**evidence**: [T03-S001 / T03-S002 / T03-S005 / T03-S008 / T03-S010 / T03-S011 / T03-S025 / T03-S033 / T03-S034 / T03-S050]

### SOP-6. 骨伤科 中医诊疗 SOP (颈椎 / 腰椎 / 肩周 / 膝骨关节炎 / 急性扭伤 / 慢性劳损 — 推拿 + 正骨 + 针灸 + 中药)

**适应症 / 适用情境**: 中医骨伤科 常见病 — 颈椎病 (神经根型 / 椎动脉型 / 脊髓型 / 交感型) + 腰椎病 (腰肌劳损 / 腰椎间盘突出 / 腰椎管狭窄 / 急性腰扭伤) + 肩周炎 (粘连期 / 冻结期 / 解冻期) + 膝骨关节炎 (KOA Kellgren 1-3 级) + 急性软组织损伤 (扭伤 / 挫伤) + 慢性劳损 (肌筋膜炎 / 网球肘 / 鼠标手 / 跟痛症) + 风湿痹证.

**禁忌 / 不适用 + 替代路径**: 骨折 + 关节脱位 → 必须 西医 X 光 / CT / MRI 确诊 + 复位固定; 脊柱肿瘤 / 转移 / 结核 / 感染 → 西医优先; 脊髓型颈椎病 + 椎管狭窄 严重 + 神经损害进行性加重 → 西医外科手术评估; 骨质疏松严重 → 推拿 + 正骨 慎重 (易致骨折); 急性炎症期 (红肿热痛) → 暂停推拿 + 优先消炎; 出血倾向 / 抗凝治疗 → 推拿 + 针刺 慎重.

**入门路径 (规培骨伤 + 中医骨伤辨证)**:
1. **问诊 + 病史 (10-15 min)**: 主诉 (疼痛部位 / 性质 / 程度 / 加重缓解因素) + 现病史 (外伤史 + 劳损史 + 起病急缓) + 既往史 + 职业 + 运动习惯 + 影像学结果. 陷阱: 骨伤科 必须 区分 急性外伤 vs 慢性劳损, 处方完全不同.
2. **望诊 + 触诊 + 西医专科查体 (15-25 min)**: 望姿势 + 步态 + 局部畸形 + 肿胀 + 皮色; 触诊 (压痛点 + 条索 + 结节 + 关节活动度 + 肌力 + 神经反射) + 西医专科查体 (颈椎 — Spurling / 椎间孔挤压 / 臂丛牵拉; 腰椎 — 直腿抬高 / 加强试验 / 4 字征 / 跟臀试验; 肩关节 — Neer / Hawkins / 落臂试验; 膝关节 — Lachman / McMurray / 髌骨研磨) + 必查影像 (X 光 + MRI + CT 必要时). 陷阱: 骨伤科 拒绝看影像 = 拒绝鉴别诊断 → 错失 骨折 + 肿瘤 + 结核 + 椎管严重狭窄.
3. **中医骨伤辨证 + 双轨诊断 (10-15 min)**: 中医病名 (筋伤 / 骨伤 / 痹证 / 项强 / 腰痛 / 肩痹 / 鹤膝风) + 中医证型 (气滞血瘀 / 寒湿痹阻 / 湿热痹阻 / 肝肾不足 / 气血亏虚 / 风寒湿邪) + 西医诊断 (颈椎病分型 / 腰椎间盘突出节段 / KOA 分级). 陷阱: 中医证型 + 西医分型 双轨, 不可偏废.
4. **推拿 + 正骨 + 针灸 + 拔罐 + 中药 综合治疗 (30-60 min)**: 推拿 (滚 + 一指禅推 + 揉 + 按 + 拿 + 搓 + 摇 + 抖 + 弹拨 + 拨筋 + 整复) + 正骨 (颈椎旋转扳法 / 腰椎斜扳 / 提拉旋转 — 必须 严格规范) + 针灸 (颈椎 风池 + 颈夹脊 + 大椎 + 肩井; 腰椎 肾俞 + 大肠俞 + 委中 + 阿是穴; 膝 阳陵泉 + 阴陵泉 + 内外膝眼 + 鹤顶 + 血海) + 拔罐 (留罐 / 走罐 / 闪罐) + 中药 (内服 + 外敷 + 熏洗 + 膏药). 陷阱: 颈椎扳法 + 腰椎扳法 必须 严格规范 + 避免暴力 — 颈椎扳法 暴力 + 脊髓损伤 + 截瘫 案例屡见报道.
5. **中药内服 + 外用 (4-12 周一调)**: 内服 — 急性扭伤 复元活血汤 + 桃红四物汤; 慢性劳损 + 痹证 独活寄生汤 + 蠲痹汤 + 身痛逐瘀汤; 肝肾不足 六味地黄丸 + 左归丸 + 右归丸; 外用 — 跌打损伤 红药水 + 云南白药 + 七厘散; 风寒湿痹 风湿膏 + 麝香壮骨膏 + 关节止痛膏. 陷阱: 慢性劳损 中药 + 推拿 + 针灸 综合, 必须 长疗程 (4-12 周) + 定期评估.
6. **功能锻炼 + 防复发指导 (持续)**: 颈椎 (米字操 + 颈肩操 + 八段锦) + 腰椎 (五点支撑 + 燕飞 + 平板支撑 + 桥式运动) + 肩关节 (爬墙 + 钟摆 + 弹力带) + 膝 (股四头肌训练 + 直腿抬高); 工作姿势 + 运动量调整 + 防潮防寒. 陷阱: 不做功能锻炼 + 只做被动治疗 = 复发率高.

**资深路径 (骨伤名医 + 推拿流派 + 正骨派)**:
- **跳过 标准辨证**: 资深骨伤名医 跳过 教材常规辨证 → 优化 触诊主导 (筋出槽 + 骨错缝 直观判断) → 额外 凭手感 直接 摸出 阳性反应点 (条索 + 结节 + 错位) 出 针对性手法.
- **优化 林如高 + 林家正骨**: 资深医师 优化 林如高 (国医大师 / 福建林家正骨派) 经验 — 跳过 复杂多手法 → 直接 "摸接端提 + 推拿按摩 + 摇摆屈伸" 八字诀 + 整复 + 中药 + 夹板 — 福建林家正骨 全国推广.
- **优化 张氏正骨 + 平乐正骨**: 河南平乐 (郭氏正骨 + 平乐郭氏正骨派) 资深医师 优化 骨折正骨 — 跳过 标准西医石膏 → 直接 中医手法整复 + 小夹板 + 中药 + 功能锻炼 — 平乐正骨 国家级非物质文化遗产 + 国家正骨 SOP 主流之一.
- **优化 推拿派**: 资深医师 优化 — 上海一指禅推拿派 (王松山 / 钱福卿 / 丁季峰 滚法) 优化 内科推拿 (失眠 / 高血压 / 胃痛); 山东齐鲁推拿 (张素芳) 优化 小儿推拿; 长春袁氏正骨 + 北京罗有明 (双桥老太太) + 葛家正骨 — 各派 + 师承 + 独门手法.
- **额外 针刀疗法 + 银质针 + 浮针**: 资深医师 额外 针刀疗法 (朱汉章创立 — 介于针灸与外科手术之间, 治筋膜粘连 / 颈椎病 / 网球肘 / 腰椎间盘突出) + 银质针 (软组织松解) + 浮针 (符仲华创立 — 治筋膜疼痛 + 即时显效).
- **跳过 急性期推拿**: 资深医师 跳过 急性期推拿 → 优化 急性期 中药 + 冰敷 + 制动 + 针刺 远端取穴, 急性期过后 (3-7 天) 再 推拿 + 正骨.

**常见陷阱**:
- **陷阱 1: 颈椎暴力扳法** — 脊髓损伤 + 截瘫 案例屡见报道; 必须 严格规范 + 评估椎管 + 排除脊髓型颈椎病再扳; 老年 + 骨质疏松 + 椎动脉狭窄 严禁颈椎扳法.
- **陷阱 2: 急性期推拿加重损伤** — 急性扭伤 + 急性炎症期 + 急性椎间盘突出 严禁推拿 + 必须 冰敷 + 制动 + 中药内服 + 针刺远端取穴.
- **陷阱 3: 不看影像盲目正骨** — 骨折 + 肿瘤 + 结核 + 转移 + 椎管严重狭窄 严禁正骨手法 + 必须 西医影像确诊.
- **陷阱 4: 慢性病只做被动治疗** — 不做 功能锻炼 + 工作姿势调整 = 复发率高; 必须 主动锻炼 + 防复发指导.
- **陷阱 5: 中药外用过敏 + 灼伤** — 中药膏药 + 熏洗 + 烫疗 + 火针 必须 询问过敏史 + 控制温度 + 观察皮肤反应.

**监管要点 + 安全警示**: 骨伤科必须 中西医结合 + X 光 / MRI 必查; 颈椎扳法 + 腰椎扳法 严格规范 + 避免暴力; 急性期严禁推拿; 老年 + 骨质疏松 + 椎动脉狭窄 推拿正骨慎重; 针刀疗法 + 银质针 + 浮针 必须 持证 + 知情同意 + 无菌操作; 中药外用过敏 + 灼伤 注意; **暴力扳法 + 急性期推拿 + 不看影像盲目正骨 = 骨伤科三大严重错误**.

### SOP-7. 针灸临床 SOP (经络辨证 → 选穴 → 手法 → 留针 → 出针 + 禁忌)

**适应症 / 适用情境**: 针灸 全面适应症 — 痛证 (头痛 / 颈肩腰腿痛 / 痛经 / 偏头痛 / 三叉神经痛) + 神志病 (失眠 / 抑郁 / 焦虑 / 中风后遗症) + 内科 (高血压 / 糖尿病 / 哮喘 / 胃痛) + 妇科 (月经病 / 不孕 / 围绝经期) + 儿科 (遗尿 / 厌食) + 五官 (耳鸣 / 鼻炎 / 咽炎) + 美容 (面瘫 / 痤疮) + 戒断综合征 (戒烟 / 戒毒 / 减肥). WHO 已认可 针灸有效病种 100+ 类.

**禁忌 / 不适用 + 替代路径**: 出血倾向 / 抗凝治疗 严禁深刺 + 浅刺慎重; 严重心衰 / 极度虚弱 / 严重糖尿病 + 周围神经病变 + 皮肤感染 + 局部肿瘤 / 溃疡 + 妊娠 (合谷 / 三阴交 / 至阴 + 腰骶部 + 腹部 严禁; 后期可选用胎位不正 + 引产) + 精神病发作期 + 醉酒 / 极度饥饿 / 极度疲劳 + 婴幼儿 (头部 + 心脏 + 大血管 + 眼球 + 内脏深部) + 急危重症 (中风超急性期 + STEMI + 大出血 西医优先).

**入门路径 (规培针灸 + 标准选穴)**:
1. **辨证 + 经络诊察 (10-15 min)**: 中医辨证 (脏腑 + 气血 + 寒热虚实) + 经络辨证 (循经压痛 + 经络敏感点 + 阿是穴 + 反应点) + 西医诊断. 陷阱: 针灸 不辨证 + 不诊察经络 = 不是中医针灸, 是 神经反射点针刺.
2. **选穴 (5-10 min)**: 按 配穴原则 — 局部取穴 + 邻近取穴 + 远端取穴 + 循经取穴 + 辨证取穴 + 经验取穴; 常用 主穴 (合谷 + 太冲 = 四关 + 内关 + 足三里 + 三阴交 + 关元) + 病种主穴 (头痛 百会 + 风池 + 太阳 + 列缺 / 失眠 神门 + 三阴交 + 安眠 / 中风 内关 + 人中 + 三阴交 + 极泉 + 尺泽 + 委中 醒脑开窍) + 阿是穴. 陷阱: 选穴 过多 (一次 > 20 穴) + 配穴无章法 = 经验不足.
3. **体位 + 消毒 + 持针 (5 min)**: 体位 (仰卧 / 俯卧 / 侧卧 / 坐位 — 患者舒适 + 取穴方便 + 易于操作); 消毒 (穴位皮肤 + 医生手指 + 针具 — 75% 酒精 / 碘伏); 持针 (左手押手定穴 + 右手刺手持针, 双手协作). 陷阱: 消毒不规范 + 重复使用一次性针 = 感染 + 传染病 (HIV + HBV + HCV) 风险.
4. **进针 + 行针 + 得气 (5-15 min)**: 进针 (单手 / 双手 / 管针 / 速刺); 进针深度 (按穴位 + 患者胖瘦 + 病情); 行针 手法 (提插 + 捻转 + 弹 + 刮 + 摇 + 飞 + 颤) + 补泻手法 (徐疾补泻 + 提插补泻 + 捻转补泻 + 迎随补泻 + 呼吸补泻 + 开阖补泻 + 平补平泻); 得气 (患者酸麻胀重 + 医师手下沉紧). 陷阱: 不得气 = 无效 + 针感过强 = 晕针 + 内脏 / 神经损伤.
5. **留针 + 灸法 + 电针 + 火针 + 水针 (15-30 min)**: 留针 (15-30 min, 一般 20 min) + 间歇行针; 配合 灸法 (艾炷灸 + 艾条灸 + 隔姜灸 + 隔附子饼灸 + 温针灸) + 电针 (G6805 + 韩氏穴位刺激仪 — 频率 + 强度 + 时间) + 火针 (顽痹 + 阴疽) + 水针 (穴位注射). 陷阱: 留针太长 (> 60 min) 老年 + 虚弱 + 晕针风险; 火针 + 水针 严格无菌.
6. **出针 + 按压 + 观察 + 嘱咐 (5 min)**: 出针 (与进针相反顺序 + 干棉球按压防出血) + 计数 (避免漏针) + 观察 (晕针 + 出血 + 血肿 + 不适); 嘱咐 (针后 30 min 不剧烈活动 + 不洗澡 + 注意保暖). 陷阱: 漏针 + 滞针 + 弯针 + 折针 是 针灸医师 严重失误.
7. **疗程 + 评估 + 调整 (持续)**: 一般 每日 / 隔日 1 次 + 10 次 1 疗程 + 疗程间隔 5-7 天; 每疗程 + 评估 + 调整选穴 + 调整手法 + 调整疗程. 陷阱: 疗效不佳 + 不评估调整 + 长期无效针刺 = 浪费患者时间金钱.

**资深路径 (针灸大家 + 学术流派)**:
- **跳过 教材标准取穴**: 资深针灸名医 跳过 教材常规取穴 → 优化 自创经验取穴 (国医大师石学敏 醒脑开窍针刺法 / 国医大师贺普仁 三通法 [微通 + 温通 + 强通] / 国医大师程莘农 三才针法 / 国医大师郭诚杰 透穴针法) → 额外 经验穴 + 独门组方.
- **优化 醒脑开窍针刺法 治中风**: 石学敏 (国医大师 / 天津中医药大学第一附属医院) 醒脑开窍针刺法 — 跳过 传统中风针灸 (百会 + 风池 + 风府 + 肩髃 + 曲池 + 足三里) → 直接 主穴 内关 + 人中 + 三阴交 + 极泉 + 尺泽 + 委中 — 全国推广 中风急性期 必用法 + 进入 NATCM 临床路径.
- **优化 火针 + 三棱针放血**: 贺普仁 三通法 资深医师 优化 — 微通 (毫针) + 温通 (火针 + 灸) + 强通 (三棱针放血) — 跳过 单一毫针 → 三通灵活组合; 额外 火针治顽痹 + 三棱针放血治急性扭伤 + 中暑 + 高热.
- **额外 透穴 + 巨刺 + 缪刺**: 资深医师 额外 透穴 (一针透多穴 — 如阳陵泉透阴陵泉 + 太阳透率谷 + 地仓透颊车) + 巨刺 (左病右治 + 右病左治) + 缪刺 (络脉病 — 远端取穴) + 远部取穴 + 平衡针 (王文远创立).
- **优化 浮针 + 腹针 + 头针**: 资深医师 优化 — 浮针 (符仲华 — 治筋膜疼痛, 即时显效) + 腹针 (薄智云 — 神龟图; 腹部全息) + 头针 (焦顺发 + 朱明清 — 大脑皮质投射区) + 眼针 (彭静山 — 八卦区) + 耳针 (王不留行 + 磁珠 + 电耳针) + 手针 + 足针 + 鼻针 — 全息针法 创新.
- **跳过 单一针刺**: 资深医师 跳过 单一毫针 → 优化 针 + 灸 + 推拿 + 中药 + 拔罐 + 刮痧 综合, 提高 临床疗效.

**常见陷阱**:
- **陷阱 1: 不辨证 + 不诊察经络** — 神经反射点针刺 = 不是中医针灸, 失去 中医灵魂.
- **陷阱 2: 消毒不规范 + 重复用针** — 感染 + 传染病 (HIV + HBV + HCV) 风险 + 严重违规.
- **陷阱 3: 危险穴位深刺** — 风池 + 风府 + 哑门 (延髓) + 缺盆 (锁骨下 + 胸膜) + 期门 + 章门 (肝脾) + 中脘 + 神阙 (内脏) 严格控制深度 + 角度 + 避免内脏损伤.
- **陷阱 4: 妊娠期针灸禁忌穴** — 合谷 + 三阴交 + 至阴 + 腰骶 + 腹部 严禁针刺 + 防流产.
- **陷阱 5: 晕针处理不当** — 晕针 (头晕 + 出汗 + 面色苍白 + 心慌 + 恶心) 必须 立即出针 + 平卧 + 保暖 + 饮温糖水 + 严重者 急救.

**监管要点 + 安全警示**: 针灸医师 必须 持证 (执业医师 + 中医专业); 消毒规范 + 一次性针具 强制; 危险穴位 严格控制深度 + 角度; 妊娠 + 出血倾向 + 极度虚弱 + 严重糖尿病 + 婴幼儿 严格禁忌; 火针 + 三棱针放血 + 针刀 必须 持证 + 无菌; 病案 + 知情同意 必规范; **针灸感染 + 危险穴位损伤 + 妊娠禁忌违反 = 针灸三大事故根源**.

### SOP-8. 推拿临床 SOP (评估 → 体位 → 手法 → 反应 + 禁忌 + 配合)

**适应症 / 适用情境**: 推拿 适应症 — 软组织损伤 (颈肩腰腿痛 + 急性扭伤 + 慢性劳损 + 肌筋膜炎) + 关节疾病 (颈椎病 + 腰椎间盘突出 + 肩周炎 + 膝骨关节炎) + 内科 (失眠 + 高血压 + 胃痛 + 便秘 + 偏头痛) + 妇科 (痛经 + 月经病) + 小儿 (感冒 + 咳嗽 + 厌食 + 遗尿 + 多动) + 保健 (亚健康 + 疲劳 + 美容).

**禁忌 / 不适用 + 替代路径**: 急性创伤 + 骨折未愈合 + 关节脱位未复位 + 严重骨质疏松 (椎体骨折风险) + 严重心衰 + 严重出血倾向 / 抗凝 + 急性传染病 + 局部皮肤感染 / 溃疡 / 烫伤 + 局部肿瘤 / 转移 + 妊娠 (腰骶部 + 腹部 + 合谷 + 三阴交 严禁) + 醉酒 + 极度饥饿 + 极度疲劳 + 婴幼儿 (手法 必须 极轻) + 急危重症 (中风急性期 + STEMI + 大出血 西医优先).

**入门路径 (规培推拿 + 标准手法)**:
1. **病情评估 + 西医诊断 (10-15 min)**: 主诉 + 现病史 + 既往史 + 影像学结果; 西医专科查体 + 必查 X 光 / MRI 排除 骨折 + 肿瘤 + 严重椎管狭窄; 中医辨证 + 经络诊察 + 阳性反应点 (压痛 + 条索 + 结节). 陷阱: 不看影像 + 不做西医查体 + 盲目推拿 = 错失 严重器质性病变.
2. **体位 + 暴露 + 介质 (5 min)**: 体位 (俯卧 + 仰卧 + 侧卧 + 坐位 — 患者舒适 + 易操作 + 充分暴露); 介质 (滑石粉 + 按摩油 + 凡士林 + 麝香风湿油 + 红花油 — 减少摩擦 + 提高疗效) + 室温 + 隐私保护. 陷阱: 暴露不足 + 体位不舒适 + 介质过敏 + 隐私保护不足 = 患者投诉 + 疗效差.
3. **手法选择 + 顺序 + 力度 + 时间 (30-60 min)**: 手法 — 滚法 (滚动主治法) + 一指禅推 (一指禅推动) + 揉 (旋转揉动) + 按 (按压) + 摩 (抚摩) + 拿 (拿捏) + 搓 (搓动) + 摇 (关节摇动) + 抖 (抖动) + 拨 (弹拨) + 击 (拍打) + 捏 (捏挤) + 捻 (捻转) + 推 (推动) + 弹 (弹拨); 顺序 (轻 → 重 → 轻; 远 → 近 → 远; 周围 → 局部; 由表及里); 力度 (轻 — 头面 + 婴幼儿 + 老年 + 虚弱; 中 — 一般 + 慢性; 重 — 强壮成年 + 急性扭伤; 渗透 — 由表及里); 时间 (一般 15-30 min; 局部 5-10 min; 全身 30-60 min). 陷阱: 力度过重 + 时间过长 + 暴力手法 = 患者损伤 + 投诉.
4. **整复手法 + 关节运动 (5-15 min)**: 颈椎 (颈部摇法 + 颈部扳法 + 颈部拔伸) + 腰椎 (腰部斜扳 + 腰部旋转扳 + 腰部拔伸 + 后伸扳) + 肩 (肩部摇法 + 肩部扳法 + 上举摇) + 髋 (髋部摇法 + 髋部扳法) + 膝 (膝关节摇法 + 屈伸扳法) + 踝 (踝部摇法 + 踝部背屈扳法). 陷阱: 颈椎 + 腰椎扳法 必须 严格规范 + 避免暴力 + 排除禁忌; 颈椎扳法 暴力 + 脊髓损伤 + 截瘫 案例屡见报道.
5. **配合 中药 + 针灸 + 拔罐 + 灸 (持续)**: 推拿 + 中药内服 + 外敷 + 熏洗 + 膏药 (跌打损伤膏 / 麝香壮骨膏 / 关节止痛膏) + 针灸 (推拿后 + 即时针灸 + 增效) + 拔罐 (留罐 + 走罐 + 闪罐 + 刺络拔罐) + 艾灸 (寒证 + 虚证) — 综合治疗 提高疗效. 陷阱: 单一推拿 + 不配合其它 = 慢性病疗效有限.
6. **治疗反应 + 注意事项 (持续)**: 推拿后 部分患者 局部酸痛 + 疲倦 (一般 1-2 天缓解, 不是不良反应); 严重不良反应 (推拿后 加重疼痛 + 麻木 + 无力 + 大小便异常) 立即停止推拿 + 西医检查 + 排除 脊髓损伤; 注意 推拿后 30 min 内 不剧烈活动 + 不洗澡 + 注意保暖. 陷阱: 推拿后 严重不良反应 没识别 + 延误诊治 = 严重后果.

**资深路径 (推拿大家 + 学术流派)**:
- **跳过 教材常规手法**: 资深推拿名医 跳过 教材常规手法序列 → 优化 自创手法组方 (国医大师张素芳 小儿推拿 / 国医大师刘柏龄 长春 / 罗有明 北京双桥老太太 / 丁季峰 上海一指禅 + 滚法) → 额外 经验手法 + 独门绝活.
- **优化 一指禅推拿 + 滚法**: 上海一指禅 + 滚法派 (王松山 + 钱福卿 + 丁季峰 + 朱春霆) 资深医师 优化 — 一指禅推为主治法 + 配合滚法 + 内功推拿 — 跳过 多手法堆叠 → 直接 一指禅推 + 渗透 (透皮 + 透肉 + 透骨) — 治 内科 (失眠 + 高血压 + 胃痛 + 便秘) + 显效.
- **优化 林氏正骨推拿**: 福建林家 (林如高 / 林子顺 / 林达明) 优化 + 北京罗氏 (罗有明) + 河南平乐郭氏 + 长春刘氏 (刘柏龄) + 山东齐鲁张氏 (张素芳 小儿推拿) — 各流派 + 师承 + 独门正骨 + 独门推拿.
- **额外 内功推拿 + 易筋经 + 八段锦**: 资深医师 额外 内功推拿 (调息 + 调心 + 调身 + 气功配合手法) + 教患者 易筋经 + 八段锦 + 五禽戏 + 太极 — 推拿 + 功法 双修, 提高 长期疗效.
- **优化 经络推拿 + 子午流注**: 资深医师 优化 — 经络推拿 (按 十四经循行 推拿 整条经络) + 子午流注 (按 时辰 选 相应经络脏腑 推拿 — 寅时肺 + 卯时大肠 + 辰时胃) — 比 单纯局部推拿 + 全身效果更佳.
- **跳过 急性期推拿**: 资深医师 跳过 急性期推拿 → 优化 急性期 中药 + 冰敷 + 制动 + 针刺远端取穴, 急性期过后 (3-7 天) 再 推拿 + 整复.

**常见陷阱**:
- **陷阱 1: 颈椎暴力扳法 + 脊髓损伤** — 案例屡见报道; 严格规范 + 排除脊髓型颈椎病 + 排除椎动脉狭窄 + 排除严重骨质疏松 再扳.
- **陷阱 2: 不看影像盲目推拿** — 骨折 + 肿瘤 + 结核 + 转移 严禁推拿; 必须 X 光 / MRI 排除.
- **陷阱 3: 急性期推拿加重损伤** — 急性扭伤 + 急性炎症期 + 急性椎间盘突出 严禁推拿; 必须 冰敷 + 制动 + 中药 + 远端针刺.
- **陷阱 4: 力度过重 + 时间过长** — 患者损伤 + 投诉 + 适得其反; 必须 按 个体差异 + 病情 + 年龄 + 体质 调整.
- **陷阱 5: 推拿后严重不良反应没识别** — 推拿后 加重疼痛 + 麻木 + 无力 + 大小便异常 立即停止 + 西医检查 + 排除脊髓损伤.

**监管要点 + 安全警示**: 推拿医师 必须 持证 (执业医师 + 中医专业 / 推拿专业 / 康复治疗师); 颈椎 + 腰椎扳法 严格规范; 急性期 + 严重器质性病变 严禁推拿; 妊娠期 + 婴幼儿 + 老年 + 骨质疏松 + 出血倾向 谨慎; 病案 + 知情同意 必规范; **暴力扳法 + 急性期推拿 + 不看影像盲目推拿 = 推拿三大事故根源; 严重不良反应识别延误 = 严重后果**.

### SOP-9. 中药处方 临床 SOP (辨证 → 立法 → 选方 → 加减 → 君臣佐使 → 配伍禁忌 → 用法用量 + 煎服)

**适应症 / 适用情境**: 所有 中医临床 + 中药处方 + 汤剂 + 颗粒 + 中成药 + 院内制剂 + 经方 + 时方 + 验方 + 自拟方. 处方者 必须 执业中医师 + 师承 + 确有专长 三类 之一持证.

**禁忌 / 不适用 + 替代路径**: 假冒中医 / 无证行医 (张悟本 / 胡万林 类) 严禁开处方 — 触犯 执业医师法 + 中医药法 + 刑法非法行医罪; 妊娠期 严守 妊娠禁忌; 哺乳期 严控 + 评估对乳儿影响; 婴幼儿 严控 朱砂 + 雄黄 + 马兜铃酸; 肝肾功能不全 严控 何首乌 + 雷公藤 + 大黄 + 黄药子 + 朱砂 + 雄黄; 急危重症 西医优先; 含马兜铃酸药 (关木通 + 广防己 + 青木香 + 天仙藤 + 寻骨风 + 朱砂莲 等已被国家药监局取消药用标准) 严禁.

**入门路径 (规培医师 + 标准处方流程)**:
1. **辨证 (10-20 min)**: 按 SOP-1 中医门诊标准 SOP 完成 四诊 + 八纲辨证 + 病因辨证 + 气血津液辨证 + 脏腑辨证 / 六经辨证 / 卫气营血辨证 / 三焦辨证, 得出 完整证型 (病位 + 病性 + 病势). 陷阱: 不辨证 + 套方 + 验方 + 经验方 = 不是中医, 是 草药.
2. **立法 (5-10 min)**: 按 治则 (扶正 + 祛邪 + 调整阴阳 + 因人因时因地制宜) + 治法 (汗 + 吐 + 下 + 和 + 温 + 清 + 消 + 补 — 八法) + 具体治法 (解表 + 清热 + 泻下 + 和解 + 温里 + 补益 + 安神 + 开窍 + 固涩 + 理气 + 理血 + 治风 + 治燥 + 祛湿 + 祛痰 + 消导 + 驱虫 + 涌吐); 法随证立. 陷阱: 法不对证 + 处方再好也无效.
3. **选方 (5-10 min)**: 按 治法 选 主方 (《方剂学》新世纪规划教材 360+ 方 + 经方 113 + 262 = 375 方 + 时方 / 后世方 / 验方); 经方派 优先 张仲景 经方原方 + 时方派 优先 后世名方 + 学派 优先 学派代表方. 陷阱: 选方 教条 + 不灵活加减 = 不是中医.
4. **加减 + 君臣佐使 (10-15 min)**: 按 主证 + 兼证 + 体质 + 年龄 + 性别 + 季节 加减 — 加 (针对未涵盖的兼证) + 减 (针对未涉及的症) + 替换 (相似功能 + 不同性能 + 替换); 君药 (主治主证 + 用量最大 + 一般 1-2 味) + 臣药 (辅助君药 + 治次要证 + 一般 2-3 味) + 佐药 (制约君臣 + 反佐 + 引经 + 调和) + 使药 (引经 + 调和 — 一般 1 味). 陷阱: 君臣佐使 不分 + 用量乱开 + 不是处方, 是 草药堆.
5. **配伍禁忌 严守 (5 min)**: 严守 "十八反" (本草明言十八反 — 半蒌贝蔹及攻乌; 藻戟遂芫俱战草; 诸参辛芍叛藜芦; 翻译: 乌头反 半夏 + 栝楼 + 贝母 + 白蔹 + 白及; 甘草反 甘遂 + 大戟 + 海藻 + 芫花; 藜芦反 人参 + 沙参 + 丹参 + 玄参 + 细辛 + 芍药) + "十九畏" (硫黄畏朴硝 + 水银畏砒霜 + 狼毒畏密陀僧 + 巴豆畏牵牛 + 丁香畏郁金 + 牙硝畏三棱 + 川乌草乌畏犀角 + 人参畏五灵脂 + 官桂畏石脂); 妊娠禁忌 (大戟 + 芫花 + 甘遂 + 牵牛 + 巴豆 + 商陆 + 麝香 + 三棱 + 莪术 + 水蛭 + 虻虫); 妊娠慎用 (桃仁 + 红花 + 大黄 + 番泻叶 + 干姜 + 肉桂 + 附子 + 牛膝 + 川芎). 陷阱: 配伍禁忌违反 + 严重医疗事故 + 法律责任.
6. **用量 + 用法 (5-10 min)**: 用量 (按 中国药典 一部 规定剂量 + 按 主辅药 + 按 患者年龄 + 体质 + 病情; 一般 6-10g 9-15g 10-15g 15-30g; 急症重病 加大 + 慢性轻病 减小; 毒性药 严控 — 附子 3-15g + 制川乌制草乌 1.5-3g 先煎; 妊娠 + 儿童 + 老人 减量); 用法 (煎药法 — 先煎 [矿石 + 介壳 + 毒性药 30-60 min] + 后下 [芳香 + 易挥发 5-10 min] + 包煎 [粉末 + 细籽 + 绒毛] + 烊化 [胶类] + 冲服 [贵重 + 难溶] + 另煎 [贵重]); 服法 (温服 + 凉服 + 顿服 + 分服 + 频服; 饭前 + 饭后 + 空腹 + 睡前). 陷阱: 用量超量 + 毒性药未先煎 + 煎服法错误 = 中毒 + 无效.
7. **处方书写 + 调剂 + 复诊 (5-10 min)**: 处方按 中药处方管理规定 — 姓名 + 性别 + 年龄 + 诊断 + 中医病名 + 中医证型 + 药品名称 (正名 + 拉丁名 必要) + 剂量 + 数量 + 用法 + 医师签字 + 日期; 调剂 (中药师 复核 + 称量 + 包装 + 煎药指导); 复诊 7-14 天 调方. 陷阱: 处方不规范 + 中医师签字缺失 + 中药师调剂不规范 = NATCM + NMPA 监管处罚.

- **跳过 教材标准方**: 资深名医 跳过 教材常规方 → 优化 经方原方 + 名医经验方 + 自创方 (国医大师邓铁涛 强心饮 / 国医大师朱良春 益肾蠲痹丸 / 国医大师路志正 调脾胃方 / 国医大师贺普仁 火针组方) → 额外 数十年经验积累.

**常见陷阱**:
- **陷阱 1: 不辨证套方** — 失眠就酸枣仁汤 + 胃痛就香砂养胃丸 + 感冒就板蓝根 = 不是中医, 是 草药.

### SOP-10. 中医诊所 备案 + 监管 合规 SOP (中医药法 + 备案管理办法 + 资质 + 病案 + 处方 + 院感 + 监管 + 应急)

2. **执业中医师 + 卫技人员 配置 (1-2 月)**: 至少 1 名 执业中医师 (执业满 5 年 + 独立承担临床工作能力) + 中药师 (大型诊所) + 护士 / 助理 (按需). 陷阱: 中医师 必须 执业满 5 年 + 注册地址变更 + 多点执业备案.

### SOP-11. 中医师资格 + 师承 + 确有专长 SOP (执业医师 + 师承考核 + 确有专长 + 继续教育 + 学术继承人)

**适应症 / 适用情境**: 申请 中医执业资格 — 四种主要路径 — (1) 执业医师 (中医学专业本科 + 5 年制 / 七年制 / 八年制) → 1 年规培 → 国家医师资格考试; (2) 中医专长医师 (中医药法 第十五条 师承学习 + 确有专长 实践 + 考核合格); (3) 学术继承人 (国家级名老中医 + 全国名老中医药专家学术经验继承工作项目); (4) 海外执业 (NCCAOM 美国 + CTCMA 加拿大 + AHPRA 澳洲).

**入门路径 (执业医师 + 标准路径)**:

### SOP-12. 中成药 NMPA 注册 + 古代经典名方 SOP (新药 / 古代经典名方简化注册 / 院内制剂备案)

**适应症 / 适用情境**: 中成药 / 中药新药 / 古代经典名方制剂 / 院内制剂 + NMPA 注册 + 备案 + 上市后监管. 三大路径 — (1) 中药新药注册 (新药申报 + 临床试验 + 注册) + (2) 古代经典名方制剂 简化注册 (NATCM + NMPA 古代经典名方目录 — 第一批 100 首 2018 + 第二批儿科 + 第三批) + (3) 医疗机构 院内制剂备案 (省级 NMPA + 院内调剂使用).

**禁忌 / 不适用 + 替代路径**: 不属于中医药 / 中药范畴 → 走 化学药 / 生物制品 / 医疗器械 NMPA 注册; 仅在 单一医疗机构 使用 → 走 院内制剂备案 (不能跨机构使用); 中药保健品 → 走 国家市场监督管理总局 保健食品注册 (蓝帽子); 仅 食品 → 走 食品生产许可.

**入门路径 (中药新药 NMPA 注册 + 标准路径)**:
1. **临床前研究 (2-5 年)**: 药理 + 药效 + 毒理 + 药代动力学 + 质量标准 + 工艺研究 + 稳定性研究 + 注册分类申请 (1.1 / 1.2 / 1.3 / 1.4 类 — 中药新药分类). 陷阱: 中药新药 化学成分复杂 + 质量标准 + 工艺稳定性 难; 必须 严格 ICH + 国家药典 一部.
2. **临床试验申请 IND (3-12 月)**: 提交 NMPA 药品审评中心 CDE — 临床试验申请 IND (Investigational New Drug Application) + 60 工作日 默认许可 (2018 改革后 60 日默认许可制); 通过后 启动 临床试验. 陷阱: IND 申请 60 日内 CDE 不发函 = 默认许可; 60 日内 CDE 提出意见 = 必须 回应 + 修改.
3. **临床试验 I / II / III 期 (3-7 年)**: I 期 (健康志愿者 + 安全性 + 剂量爬升 + 20-100 例) + II 期 (患者 + 初步疗效 + 剂量优化 + 100-300 例) + III 期 (患者 + 确证疗效 + 多中心 + 300-3000 例) — 严格 GCP + 伦理委员会 + 知情同意 + 临床试验登记 (ClinicalTrials.gov + ChiCTR). 陷阱: 中药临床试验 安慰剂对照 + 双盲 + 随机 难度高; 必须 与 临床流行病学专家 + 统计学专家 合作.
4. **新药申请 NDA (1-3 年)**: 完成 临床试验 + 提交 NDA (New Drug Application) — 全套数据 (临床前 + 临床 + 质量 + 工艺 + 稳定性 + 包装 + 标签 + 说明书) + CDE 审评 (12-24 月) + 现场核查 + 注册检验 + 通过后 颁发 药品注册证. 陷阱: NDA 审评 任何缺陷 + 补充资料 = 审评时钟暂停 + 周期延长.
5. **上市后监管 + IV 期 + 不良反应监测 (持续)**: 取得 药品注册证 + 上市销售 + IV 期临床 + 不良反应监测 (ADR 报告) + 国家药品不良反应监测年报 + 定期安全性更新报告 (PSUR) + 上市后变更 (注册标准 + 工艺 + 说明书 + 适应症 + 注册地点). 陷阱: 上市后 严重不良反应 + 国家药监局 可责令 限制使用 + 暂停 + 撤销注册.
6. **再注册 + 续期 (每 5 年)**: 药品注册证 有效期 5 年 + 期满前 6 月 申请再注册 + 提交 5 年使用情况 + 不良反应 + 质量稳定性 + 临床有效性 + CDE 审评 + 续期 5 年. 陷阱: 漏再注册 = 注册失效 + 不能再生产销售.

### SOP-13. 中医循证 + RCT + Cochrane SR 检索 SOP (临床医师 + 研究者 + 教学)

**适应症 / 适用情境**: 中医循证医学实践 — 临床医师 (基于证据的临床决策 + 临床指南 + 临床路径) + 研究者 (RCT 设计 + 系统评价 SR + Meta 分析) + 教学 (循证医学课程 + 文献综述 + 临床见习). 中医循证 是 中医现代化 + 国际化 + 监管 + 国际期刊发表 的 必由之路.

**禁忌 / 不适用 + 替代路径**: 反对循证 / 反对现代医学整合 → 中医不可能闭门造车 + 必须循证; 个案 + 经验论 + 名医说了算 → 不能 替代 RCT + SR 高级别证据; 拒绝学习 PubMed + Cochrane + GRADE → 中医临床医师 + 研究者 + 教学者 必备技能.

**入门路径 (规培医师 + 标准循证)**:
1. **循证医学基础 (持续学习)**: 学习 循证医学 (EBM Evidence-Based Medicine) 基本概念 + 5A 步骤 (Ask + Acquire + Appraise + Apply + Audit) + 5 级证据等级 (系统评价 SR + Meta 分析 > RCT > 队列 > 病例对照 > 病例报告 > 专家意见) + GRADE 证据质量评级 + 推荐强度. 陷阱: 中医医师 + 不学循证 + 个案经验论 = 不被国际医学界认可.
2. **临床问题 PICO 构建 (5-10 min / 问题)**: 按 PICO 框架 — Patient (患者 + 病情 + 体质 + 证型) + Intervention (干预 + 中药 + 针灸 + 推拿 + 中西医结合) + Comparison (对照 + 安慰剂 + 西药 + 其它中医治法) + Outcome (结局 + 主要 + 次要 + 安全 + 生活质量); 转 临床问题 → 可检索问题. 陷阱: 临床问题模糊 + 不可检索 = 循证起点错误.
3. **数据库检索 (15-30 min / 问题)**: 国际 — PubMed (国际医学文献) + Cochrane Library (国际系统评价) + EMBASE + Web of Science + ClinicalTrials.gov + ChiCTR (中国临床试验注册); 中文 — CNKI (中国知网) + 万方 + 维普 + SinoMed; 中医专用 — Chinese Medicine 期刊 + Journal of Traditional Chinese Medicine + Journal of Ethnopharmacology; 检索词 — MeSH 主题词 + 自由词 + 布尔逻辑 + 文献类型 + 时间限制. 陷阱: 检索 不规范 + 不全面 = 偏倚 + 错失关键文献.
4. **文献评价 + 质量评价 (1-3 h / 文献)**: 文献评价 — Cochrane Risk of Bias 工具 (RoB 2.0) + Jadad 量表 + AMSTAR 2 (SR 评价) + GRADE 证据等级评价; 中医 RCT 特殊评价 — 中医辨证 + 证型纳入 + 标准化 + 操作 + 安慰剂设计 + 盲法 + 退出 + 失访. 陷阱: 中医 RCT 安慰剂设计 + 盲法 + 中医辨证标准化 难; 必须 严谨评价.
5. **证据综合 + 应用临床 (30-60 min / 决策)**: 综合 高质量 SR + RCT + 国际指南 + 国内 NATCM 中医临床路径 + 患者价值观 + 临床判断 → 临床决策; 结合 中医辨证论治 灵魂 + 现代循证证据 — 不是 二者对立, 而是 二者整合. 陷阱: 把 循证医学 = 西医化 + 把 中医辨证 = 拒绝循证 = 双重错误.
6. **临床实践审计 + 反馈 (持续)**: 临床决策实施 后 + 评估患者反应 + 持续学习 + 改进; 加入 病案 + 团队讨论 + 学术分享. 陷阱: 不审计 + 不反馈 = 循证形式化.

**资深路径 (研究者 + 学术领袖 + 教学者)**:
- **跳过 仅作临床应用**: 资深研究者 跳过 仅作临床应用 → 优化 设计 + 实施 + 发表 中医 RCT (国家级课题 + 国家自然科学基金 + 国家重点研发计划 + 行业基金) + 系统评价 (Cochrane SR + 国际期刊 SR) + 真实世界研究 RWS (基于电子病历 + 大数据 + AI) → 额外 引领 学术方向 + 制定 临床指南.
- **优化 病证结合 + 真实世界 RWS**: 资深研究者 优化 — 跳过 单纯 RCT → 病证结合研究 (西医病 + 中医证 双轨纳排) + 真实世界研究 RWS (基于 三甲中医院 电子病历 + 大数据 + AI 自然语言处理 NLP) — 是 中医现代化 + 国际化 + 监管认可 + 真实世界证据 RWE 最新趋势 (国家药监局 + FDA + EMA 已认可 RWE 注册).
- **优化 Cochrane SR + 国际期刊**: 资深研究者 优化 — 跳过 国内期刊 → 直接 Cochrane SR + Lancet + BMJ + NEJM + JAMA + Ann Intern Med + Chinese Medicine (BMC) + Journal of Ethnopharmacology + Phytomedicine + Acupuncture Med + Complementary Therapies in Medicine 国际期刊发表 — 国际同行评议 + 高 IF + 国际影响力.
- **额外 学术领袖 + 制定指南**: 资深研究者 额外 — 牵头 制定 中华中医药学会 团体标准 + NATCM 中医临床路径 + WHO benchmarks + 国际中医药学会指南 — 从 文献使用者 → 文献制定者.
- **优化 临床流行病学方法**: 资深研究者 优化 — 学习 临床流行病学 (clinical epidemiology) + 循证医学 + 卫生经济学 + 统计学 + 流行病学 + 卫生政策 + 卫生服务研究 — 多学科融汇 + 创新方法.
- **跳过 单一西医 EBM 框架**: 资深研究者 跳过 单一西医 EBM 框架 → 优化 中医特色 EBM 框架 (病证结合 + 整体观 + 个体化 + 真实世界 + 系统科学 + 中医临床流行病学) → 额外 中医现代化 + 国际化 + 监管认可的中医循证体系.

**常见陷阱**:
- **陷阱 1: 中医医师不学循证 + 个案经验论** — 不被国际医学界认可 + 中医现代化阻碍.
- **陷阱 2: 中医 RCT 安慰剂 + 盲法 + 中医辨证标准化困难** — 必须 严谨设计 + 临床流行病学专家合作; 不是 拒绝 RCT, 是 严谨 RCT.

**监管要点 + 安全警示**: 中医临床医师 + 研究者 + 教学者 必备 循证医学 + 临床流行病学 + GCP + GCP + 国际期刊发表能力; 中医 RCT 严谨 + 病证结合 + 真实世界 + Cochrane SR; NATCM + 国家自然科学基金 + 国家重点研发计划 大力支持; 国际期刊发表 + 国际指南制定 + 国际监管认可 = 中医现代化 + 国际化必由之路; **不学循证 + 拒绝 RCT + 个案经验论 = 中医现代化三大阻碍**.

### SOP-14. 中西医结合 整合诊疗 SOP (危重 + 围手术期 + 肿瘤 + 终末期 中西医协同)

**适应症 / 适用情境**: 中西医结合 适应症 — 危重急症 (中风 + STEMI + 急性肾衰 + 急性肝衰 + 脓毒症 + ARDS) + 围手术期 (术前调理 + 术后康复 + 术后并发症) + 肿瘤 (放化疗增效减毒 + 术后康复 + 姑息治疗 + 晚期姑息) + 终末期 (心衰 + 肾衰 + 肝衰 + 慢病终末期姑息) + 慢性病多系统受累 (糖尿病并发症 + 高血压并发症 + 风湿病多脏器). 1956 起中国 中医西医并行 70 年, 中西医结合 已是 国家战略 + 临床主流.

**禁忌 / 不适用 + 替代路径**: 不存在 纯中医 / 纯西医 闭门造车 — 中西医结合 是 必然; 但 不是 简单 中药 + 西药 叠加, 是 整合诊疗 + 优势互补 + 协同增效; 中药 + 西药 严重相互作用 (如 华法林 + 丹参 + 银杏 = 大出血 / 麻黄 + MAOI = 高血压危象) → 严控 + 调整 + 监护.

**入门路径 (规培医师 + 标准中西医整合)**:
1. **双轨诊断 (20-40 min)**: 西医病名 (ICD-10/11) + 西医分期分级 + 西医辅查 + 西医诊断标准 + 中医病名 (《中医内科学》 + 古籍病名) + 中医证型 (脏腑 + 气血 + 寒热虚实 + 痰瘀) + 中医辨证依据 — 双轨整合 + 互相补充. 陷阱: 仅有西医诊断 + 缺中医辨证 = 不是中医; 仅有中医辨证 + 缺西医诊断 = 不是中西医结合.
2. **危重评估 + 西医优先 + 中医协同 (按急症程度)**: 急危重症 (中风 + STEMI + 脓毒症 + ARDS + 大出血 + 休克) → 西医急救优先 (时间窗内 + 介入 + 溶栓 + 抗生素 + 升压 + 输血 + 通气 + ICU) + 中医协同 (醒脑开窍 + 中药鼻饲 + 针灸); 非危重 → 中西医并行 + 优势互补. 陷阱: 急危重症 不可以 中医辨证 耽误 西医急救; 必须 西医优先.
3. **中医辨证 + 治法 + 协同方案 (20-30 min)**: 按 中医辨证 制定 治法 + 选方 + 加减; 与 西医方案 协同 — 围手术期 (术前 益气养血 + 增强体质 / 术后 益气补血 + 活血化瘀 + 健脾和胃 + 缓解便秘 + 缓解疲劳); 肿瘤 (放疗 滋阴 + 化疗 益气 + 升白 + 缓解呕吐 + 升血小板 + 缓解神经毒性 + 增免疫); 终末期 (姑息 + 缓解症状 + 改善生活质量).
4. **中药与西药相互作用 严控 (持续)**: 严控 — 抗凝 (华法林 + 丹参 + 当归 + 银杏 + 川芎 + 红花 = 出血) / 降糖 (磺脲类 + 黄连 + 苦瓜 + 葛根 = 低血糖) / 降压 (ACEI / ARB + 麻黄 = 高血压 / 利血平 + 麻黄 / 钙通道阻滞剂 + 麻黄 = 心律失常) / 抗抑郁 (MAOI + 麻黄 = 高血压危象) / 利尿 (噻嗪类 + 甘草 = 低钾) / 化疗 (顺铂 + 何首乌 = 肝损 / 紫杉醇 + 雷公藤 = 肝损 / 阿霉素 + 何首乌 = 肝心毒性) / 免疫抑制 (环孢素 + 茯苓 + 黄连 = 影响代谢). 陷阱: 中药与西药相互作用 必须 完整查 PubMed + Cochrane + 临床药师 + 慎重处方.
5. **多学科会诊 MDT + 联合制定方案 (持续)**: 复杂病例 + 多学科会诊 (西医 + 中医 + 药学 + 营养 + 康复 + 心理) + 联合制定 整合诊疗方案 + 患者 + 家属 知情同意 + 长期随访 + 联合调整. 陷阱: 单科决策 + 缺多学科会诊 + 不是 整合诊疗.
6. **疗效评估 + 调整方案 (4-12 周)**: 西医指标 (实验室 + 影像 + 生命体征 + 量表) + 中医证候演变 + 患者自评 (生活质量 + KPS + ECOG + EORTC QLQ) + 联合 调整方案. 陷阱: 仅评西医 + 不评中医 + 不评生活质量 + 不是 整合.

**资深路径 (中西医结合大家 + 学术领袖)**:
- **跳过 单纯中医或西医**: 资深中西医结合医师 跳过 单纯中医 / 单纯西医 → 优化 中西医深度整合 (国医大师陈可冀 中西医结合心血管 + 病证结合 + 活血化瘀 / 国医大师张伯礼 中医脑病 + 中医急症 + COVID-19 三方三药 / 国医大师沈自尹 中西医结合补肾 + 现代肾本质研究) → 额外 临床 + 科研 + 转化 + 国际化.
- **优化 病证结合 + 现代药理**: 资深中西医结合大家 优化 — 跳过 单纯病或证 → 病证结合 (西医病 + 中医证 + 现代药理 三位一体) + 现代药理 (中药有效成分 + 靶点 + 通路 + 作用机制) → 额外 中医现代化 + 国际化 + 监管认可.
- **优化 围手术期 ERAS + 中医加速康复**: 资深医师 优化 — ERAS (Enhanced Recovery After Surgery 加速康复外科) + 中医加速康复 (术前 益气养血 + 术后 益气补血 + 活血化瘀 + 健脾和胃 + 针灸 + 推拿 + 中药外敷) — 国际 ERAS 与 中医加速康复 整合.
- **优化 肿瘤 中西医结合 + 免疫治疗增效减毒**: 资深医师 优化 — 跳过 单纯中药抗癌 → 中医协同 西医手术 + 放疗 + 化疗 + 靶向 + 免疫 (PD-1 / PD-L1 / CAR-T) — 中医 增效减毒 + 改善生活质量 + 延长生存; 额外 国家级中西医结合肿瘤研究.
- **额外 三方三药 COVID-19 中西医结合模式**: 资深医师 额外 — 张伯礼 + 仝小林 + 黄璐琦 + 王辰 等领衔 COVID-19 三方三药 (清肺排毒汤 + 化湿败毒方 + 宣肺败毒方 + 金花清感颗粒 + 连花清瘟胶囊 + 血必净注射液) — 是 中西医结合救治急性传染病 全球范例 + 入选 国家级中医药防治新冠肺炎方案.
- **跳过 中西医对立**: 资深医师 跳过 中西医对立 → 优化 中西医平等 + 整合 + 优势互补 + 协同增效 → 额外 引领 国家中西医结合学科建设 + 国际化 + 全球传统医学整合.

**常见陷阱**:
- **陷阱 1: 急危重症 中医辨证耽误西医急救** — 中风 + STEMI + 脓毒症 + 大出血 西医优先 + 中医协同; 不可以 中医辨证 耽误 时间窗.
- **陷阱 2: 中药与西药相互作用忽视** — 必须 完整查 PubMed + Cochrane + 临床药师审核 + 慎重处方.
- **陷阱 3: 缺多学科会诊 MDT** — 复杂病例 单科决策 + 不是 整合诊疗; 必须 MDT.
- **陷阱 4: 中西医对立 + 闭门造车** — 中医 + 西医 二选一 是 1900 错误; 整合 + 优势互补 才是 中西医结合本质.
- **陷阱 5: 仅疗效评估西医指标** — 中医证候演变 + 生活质量 + 患者自评 必须 同时评估.

**监管要点 + 安全警示**: 中西医结合 是 国家战略 + 临床主流 + 1956 起 70 年; 急危重症 西医优先 + 中医协同; 中药与西药相互作用 严控; 多学科会诊 MDT + 联合方案; 围手术期 ERAS + 加速康复; 肿瘤 中西医结合 + 免疫治疗增效减毒; COVID-19 三方三药全球范例; **中医辨证耽误西医急救 + 中西药相互作用忽视 + 中西医对立 = 中西医结合三大严重错误**.

### SOP-15. 治未病 + 调摄 SOP (体质辨识 + 调养 + 季节 + 情志)

**适应症 / 适用情境**: 治未病 三大层面 — (1) 未病先防 (健康人 + 亚健康 + 高危人群 — 体质辨识 + 调养 + 预防); (2) 既病防变 (慢性病 + 既病人群 — 防 病情进展 + 防并发症); (3) 瘥后防复 (康复期 + 巩固疗效 + 防复发). 治未病思想 源于《黄帝内经·素问·四气调神大论》"圣人不治已病治未病, 不治已乱治未乱", 是 中医独特预防医学 + 健康管理体系.

**禁忌 / 不适用 + 替代路径**: 已患急危重症 → 走 SOP-2 急症 + 西医优先; 已患重症 + 严重器质性病变 → 走 SOP-3 慢性内科 + 中西医结合; 治未病 不是 包治百病 + 不能替代 急症 + 重症治疗.

**入门路径 (健康管理师 + 标准治未病)**:
1. **体质辨识 (15-25 min)**: 按 国医大师王琦 中医体质九分法 (《中医体质学》新世纪规划教材 + 中医体质分类与判定 ZYYXH/T 157-2009 中华中医药学会团体标准) — 平和质 (健康基准) + 气虚质 (易疲倦 + 易感冒) + 阳虚质 (畏寒 + 手脚冰凉) + 阴虚质 (口干 + 便秘 + 失眠) + 痰湿质 (肥胖 + 黏腻) + 湿热质 (口苦 + 痤疮 + 大便黏滞) + 血瘀质 (面色晦暗 + 痛经 + 静脉曲张) + 气郁质 (情志不舒 + 易抑郁) + 特禀质 (过敏 + 哮喘) — 用 标准化体质辨识量表 + 60 项问卷 评分判定. 陷阱: 体质 不是单一固定, 一般 主体质 + 兼夹体质 (如 平和 + 气虚兼夹) — 必须 综合判定.
2. **健康风险评估 + 中西医结合 (15-20 min)**: 中医体质风险 (气虚易感染 + 阴虚易燥热 + 痰湿易代谢综合征 + 血瘀易心脑血管 + 气郁易抑郁) + 西医健康风险 (家族史 + BMI + 血压 + 血糖 + 血脂 + 肿瘤标志物 + 心血管风险评分 + 糖尿病风险评分) + 综合评估 + 个体化健康规划. 陷阱: 中医体质评估 + 西医风险评估 必须 整合, 不能 偏废.
3. **个体化调养方案 (20-30 min)**: 按 体质 + 风险 制定 调养方案 — 饮食调摄 (五味五脏 + 五色五脏 + 体质对应食物) + 起居 (作息 + 卧具 + 着衣 + 起居规律) + 情志 (七情调摄 + 怒喜思悲恐惊忧 + 中医五志相胜) + 运动 (八段锦 + 五禽戏 + 太极 + 易筋经 + 散步 + 慢跑 + 按体质强度) + 中药调养 (体质对应 经典养生方 + 六味地黄丸 / 玉屏风散 / 香砂六君子丸 / 杞菊地黄丸 / 逍遥丸) + 针灸 / 推拿保健.
4. **季节调摄 + 四时养生 (持续)**: 春 (生发 + 疏肝 + 适度运动 + 多食绿色 + 戒怒) + 夏 (心 + 长养 + 适度出汗 + 多食苦味 + 避暑) + 长夏 (脾 + 化湿 + 健脾 + 多食黄色) + 秋 (肺 + 收敛 + 滋阴润燥 + 多食白色 + 防秋燥) + 冬 (肾 + 闭藏 + 温补 + 多食黑色 + 早睡晚起). 陷阱: 季节调摄 不是 固定教条, 必须 按 体质 + 地域 + 气候 调整.
5. **情志调摄 + 心身整合 (持续)**: 七情过激 — 怒伤肝 + 喜伤心 + 思伤脾 + 悲伤肺 + 恐伤肾; 五志相胜 — 怒胜思 + 思胜恐 + 恐胜喜 + 喜胜悲 + 悲胜怒; 现代 + 心理咨询 + 正念冥想 + 心理治疗 + 中医情志调摄 整合; 与 SOP-3 抑郁焦虑 中医治疗 联动. 陷阱: 严重抑郁焦虑 + 自杀风险 必须 精神科 + 西医优先, 治未病情志调摄 仅作 轻度调摄 + 协同.
6. **定期评估 + 调整方案 (每 3-12 月)**: 定期 体质再评估 + 健康指标复查 + 生活方式评估 + 调整调养方案; 与 SOP-1 中医门诊 + SOP-3 慢性内科 联动 — 已病转既病防变 SOP. 陷阱: 调养 不是 一次性 + 必须 长期管理 + 定期评估.

**资深路径 (名医治未病 + 中医养生大家)**:
- **跳过 标准体质量表**: 资深治未病大家 跳过 标准体质量表 → 优化 望神 + 望色 + 望舌 + 切脉 + 触诊 直观体质 + 经验判定 (5 分钟内 + 体质 + 风险 + 调养方案 雏形) → 额外 个体化深度.
- **优化 治未病专科 + 中医健康管理中心**: 资深治未病大家 优化 — 跳过 普通门诊 → 治未病专科 + 中医健康管理中心 (一站式 — 体质辨识 + 健康风险评估 + 中医辨证 + 个体化调养 + 长期随访 + 数字化管理) — 现 NATCM 推动 三甲中医院 + 综合医院 + 社区 治未病专科建设.
- **优化 季节性预防保健 + 三伏贴 + 三九贴**: 资深治未病大家 优化 — 跳过 单一中药调养 → 季节性预防保健 (三伏贴 [大椎 + 肺俞 + 膏肓 + 定喘 治哮喘 + 反复感冒 + 鼻炎] / 三九贴 [冬病冬治 + 巩固疗效] / 春秋膏方 + 冬令膏方) — 是 中医 季节性预防 + 传统养生 + 现代健康管理 整合.
- **额外 中医康复 + 慢病管理**: 资深治未病大家 额外 — 既病防变 (慢性病 — 高血压 + 糖尿病 + 慢肾炎 + 慢性肝病 + 风湿) + 瘥后防复 (中风后 + 心梗后 + 肿瘤术后 + 化疗放疗后 + 围绝经 + 老年虚弱) — 治未病 + 中医康复 + 慢病管理 整合.
- **优化 中医心理 + 正念 + 情志调摄**: 资深治未病大家 优化 — 跳过 单纯西医心理 → 中医心理 + 正念冥想 + 中医情志调摄 + 五志相胜 + 音乐疗法 + 艺术疗法 — 中医心身整合 + 现代心理学 + 正念冥想 整合.
- **跳过 单一调养**: 资深治未病大家 跳过 单一调养方法 → 优化 全方位调养 (饮食 + 起居 + 情志 + 运动 + 中药 + 针灸 + 推拿 + 八段锦 + 五禽戏 + 太极 + 易筋经 + 季节 + 地域 + 体质) → 额外 个体化深度健康管理.

**常见陷阱**:
- **陷阱 1: 体质辨识仅评单一主体质** — 实际 多 主体质 + 兼夹体质, 必须 综合判定 + 针对主兼夹分别调理.
- **陷阱 2: 季节调摄教条** — 不按 体质 + 地域 + 气候 调整 + 教条 = 适得其反.
- **陷阱 3: 治未病替代急症 + 重症治疗** — 治未病 不是 包治百病, 急症 + 重症 必须 西医优先 + 中西医结合.
- **陷阱 4: 严重抑郁焦虑 + 自杀风险 仅做情志调摄** — 必须 精神科 + 西医优先 + 治未病情志调摄 仅作协同.
- **陷阱 5: 中药养生方滥用** — 六味地黄丸 + 玉屏风散 + 逍遥丸 等中药养生方 必须 辨证使用; 不辨证滥用 = 不是中医, 是 草药保健品.

**监管要点 + 安全警示**: 治未病 思想 源于 内经 + 国医大师王琦 中医体质学 + ZYYXH/T 157-2009 体质判定 + NATCM 治未病专科建设; 体质辨识 必须 标准化 + 综合判定; 健康管理 中西医结合; 季节调摄 + 情志调摄 + 个体化; 中药养生方 必须 辨证使用; 严重抑郁焦虑 + 自杀风险 + 重症 西医优先; **体质辨识不准 + 季节调摄教条 + 治未病替代急症重症 + 中药养生方滥用 = 治未病四大误区**.
