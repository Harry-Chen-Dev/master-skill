## 诚实边界（creative-craft figures 维度结构性说明，先读这段）

网文「大神」的公开发声生态有三个结构性特征，直接影响本 track 的一手率与女频厚度：

1. **最高价值一手 = 北大「破壁者」长访谈系列**。北大网络文学研究论坛（邵燕君主持）2025-2026 选 12 位经典性作家做超长访谈（含个人阅读史 / 文学观形成 / 代表作创作 / 网文史细节 / 自我评价），完整版将收入北大社《创作者说：网络文学代表性作家访谈录》。访谈里**作家本人的直接引语 = 一手 voice**，但发布 host 多为腾讯新闻 / 澎湃 / 中国作家网 / 解放日报转载页，verifier 判 `secondary`（unknown host），按 manifest 「只能升级 surrogate_primary」规则统一标 `surrogate_primary`（作家原话，转载 host）。中国作家网（中国作协官网）转载版最权威。

2. **男频大神访谈充足，女频大神结构性低调**。男频（唐家三少 / 天蚕土豆 / 猫腻 / 烽火戏诸侯 / 爱潜水的乌贼 / 愤怒的香蕉 / 今何在 / 杀虫队队员）公开长访谈 / 直播多。女频晋江系大神（priest / 墨香铜臭 / 尾鱼 / Twentine / 丁墨）极度低调、几乎不出镜，长访谈稀少 —— **唯一例外是 priest 有一篇澎湃深度对话**（已收入本 track）。墨香铜臭仅有零散转述（魔道祖师完结访谈散见知乎，黑名单，不采）。**故本 track 女频维度偏薄属诚实事实，仅 1 位女频 retain，已在 Phase 2 接口标注。**

3. **平台写作课 / 培训讲师（橙瓜、星魂、墨星）软文水分极高**，「黄金三章」「三大爽点」等方法论虽流传广，但系统输出多在知乎 / 公众号（黑名单），原创讲师极少有可追溯一手长访谈。本 track **不收纯培训讲师**；方法论术语沉到 T06 glossary，方法论本身由大神访谈里的原话承载（如猫腻的节奏论、香蕉的「欲望→情绪→需求」、乌贼的「先世界观后升级」）。

→ 一手率结构性偏低 + verified_primary 多为学术 figure（邵燕君 / 欧阳友权，.edu.cn / .org.cn）+ 作家原话走 surrogate_primary，符合 creative-craft 行业特征，未洗分。

### 1. 猫腻

- **One-liner**: 「以爽文写情怀」的代言人——把网文的节奏工业（小高潮/平复/余韵）和文学性企图缝合，被学术界视为网文「经典化」最有力个案。(evidence: [T01-S001, T01-S002])
- **核心身份**: 起点中文网白金男频作家；代表「文学性网文」一派。
- **代表作品**: 《庆余年》《将夜》《间客》《择天记》《大道朝天》。
- **值得读/听/看的 3 件事**:
  - 📖 「破壁者」访谈·猫腻（中国作家网官网版）: https://www.chinawriter.com.cn/n1/2025/0228/c404024-40428143.html (collected 2026-06-20)
  - 🎙️ 「破壁者」访谈·猫腻（腾讯新闻全文，含节奏方法论原话）: https://news.qq.com/rain/a/20250227A06UNW00 (collected 2026-06-20)
  - 🎬 ⚠️ 未找到稳定 30min+ 视频长 talk（猫腻极少出镜）
- **核心思想关键词**: 以爽文写情怀 / 节奏（小高潮-平复-余韵）/ 看对方怎么输 / 细纲先行 / 文学性。
- **voice_samples**:
  - 同业/方法论样本: 「'自己赢'不是快感的来源，'看对方怎么输'才是快感的来源。」(source: T01-S001, 原话)
  - 同业/方法论样本: 「小高潮、平复期和余韵。高潮越小，平复期越短。几个小高潮后要有大高潮，大高潮前要挖好扣子。」(source: T01-S001, 原话)
  - 创作实践样本: 「我不把情节想明白，就没法把手放在键盘上。我必须把情节想明白了才能去实现它。」(source: T01-S001, 原话)
  - 公开/文学讨论样本: 「这是一个怯懦的人写的一本无畏的大书。」(source: T01-S001, 原话)
- **sub_skill_candidate**: `true` — 思想体系自洽（节奏论+情怀论）、影响力前 5、长访谈材料足，若用户 focus 男频文学性写作可独立蒸馏。

- **争议/批评**: 被批「太监/烂尾」倾向、后期作品节奏拖沓；「文青」标签也被部分纯爽文读者视为不够爽。
- **来源**: [Primary/surrogate] T01-S001, T01-S002 | [Reference] T04-S008（canon track 同一访谈）。
- **可信度自评**: high — 两个独立 host 转载同一长访谈，原话密度高。

- **One-liner**: 体系化世界观+升级流的天花板，「先设定世界观，再考虑怎么升级」是其方法论核心，《诡秘之主》把克苏鲁/塔罗序列融进网文升级骨架。(evidence: [T01-S005, T01-S006])
- **核心身份**: 起点白金男频；世界观流/克苏鲁神秘学流代表。
- **代表作品**: 《诡秘之主》《奥术神座》《一世之尊》《光明纪元》《宿命之环（诡秘2）》。
- **值得读/听/看的 3 件事**:
  - 📖 「破壁者」访谈·乌贼（中国作家网官网版）: https://www.chinawriter.com.cn/n1/2025/0605/c404024-40494534.html (collected 2026-06-20)

  - 🎬 ⚠️ 未找到稳定长视频
- **核心思想关键词**: 世界观先行 / 升级体系设计 / 表达（创作第一关键词）/ 强矛盾强刺激 / 准备期素材搜集。
- **voice_samples**:
  - 同业/方法论样本: 「我先做的是世界观，再考虑怎么升级，再考虑升级的表现。」(source: T01-S005, 原话)
  - 创作实践样本: 「我先把22张大牌拿来，看哪些可以用，哪些需要修改。改完后再把它们对应的旧日名称、象征含义做好，然后从上往下推导出整个序列途径。」(source: T01-S005, 原话)
  - 同业/方法论样本: 「网文需要强矛盾和强刺激，这是最核心的。」(source: T01-S005, 原话)
  - 创作实践样本: 「前面写的两三百万字的内容，都在约束着结局，每一个字都可能对结局产生影响。」(source: T01-S005, 原话)

- **dual_role**: 无。
- **最近12个月动态**: 「破壁者」访谈 2025-06；《宿命之环》（诡秘2）连载中，反复谈「第一本已揭世界观，解密元素失效后如何续吸引力」。
- **争议/批评**: 《诡秘2》被部分读者批节奏/新鲜感不及前作；世界观过于庞大致中段拖节奏。
- **来源**: [Primary/surrogate] T01-S005, T01-S006 | [Reference] T04-S010。
- **可信度自评**: high。
