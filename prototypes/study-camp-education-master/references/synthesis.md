**贯穿全 OS 的第一性矛盾**：**教育属性 vs 旅游属性**。「只游不学 / 游而不学」是行业耻辱线（intake 点明，T06 标为「极高频」外行破绽，2024-11 文旅部专门发文治理伪研学）。本 OS 的几乎每个心智模型、playbook、反模式都是这条矛盾在不同环节的投影。读本文件时，先记住这条线，再看每节如何在「合规 / 课程 / 安全 / 商业 / 主题 / 口碑」六个面上把它具体化。

**跨 track 未解决矛盾（供 Phase 2.5 用户留意）**：
- 王晓燕跨源任职署衔不一致（中国教育科学研究院副研究员 vs 教育部教育发展研究中心基础教育研究室副主任）——T01 已如实并列，不影响其框架采用。
- 「研学旅行」存在**两套权威定义并存**且未抹平：教育部口径（教基一〔2016〕8号，落点「校外**教育**活动」）vs 旧 LB/T 054-2016 旅游口径（国家旅游局，落点「教育**旅游**活动」）。这是教育派 vs 旅游派之争的语言层硬证据，保留不统一（见 §7）。

## 1. 心智模型 (7 个)

> 经三重验证（跨场景复现 / 生成力 / 排他性）。✅✅✅=心智模型；✅✅⚠️=行业放大版心智模型（强制捕捉放大程度）。候选 C8/C9/C10/C11 经验证降级为 playbook（见 §2），C12 是工具栈结构性结论（见 §3）。

### 1.1 「研学是课程不是行程」——教育属性优先

- **跨 figure 共识**（figures: 赵蔚 / 李璟晖 / 王晓燕 / 刘婷 / 梁晓燕）：≥3 位独立 figure 公开同声。
- **一句话**：研学/营地的本体是**课程**（有学习目标—实施—评价闭环），不是**行程**（景点+讲解词）；从业者用「教育本质」作为对抗旅游化、获客化、速成化的立场锚与对客叙事。
- **它说的是**：行业第一性矛盾的「正方」立场。判断一个产品姓「教」还是姓「旅」，看它有没有「有计划组织 + 研究性学习」两大要件（教基一〔2016〕8号）；缺了就是「带学生出游」而非研学。这条几乎是 figures 群像的最高频共识——赵蔚「教育初心 / 从教育本身去思考」、李璟晖「营地是表、教育是本、活动是核心」、刘婷「教育初心」、王晓燕「研学≠说走就走的旅行」、梁晓燕「教育的本质是慢」，学术派与运营派在此罕见同声。
- **应用方式**：任何产品/方案评审、对客解释、监管自证时，先回到「这是不是一门课」。被质疑「就是旅游」时，回到学习目标与评价证据自查教育属性占比。营地>研学>游学的教育属性梯度可作定位参照。
- **局限**：① 这是**立场不是配方**——光喊「教育属性优先」不解决「怎么把它做成证据」（那是 §1.5 + playbook 2.2 的活）。② 教育派内部对「教育」是什么仍分裂（学院派课程标准 vs 营地现场体验，见 §7），本模型不裁决该分歧。③ 纯 C 端轻量营的「教育性」呈现方式与 B 端校采不同（C 端靠体验+口碑可视化，B 端靠对照表+课标），不可一刀切。
- **evidence**: [T01-S003, T01-S015, T04-S012, T06-S001]（跨 figures + canon + glossary；放大度：教育部 vs 旅游局双口径定义 T06-S004 是其语言层硬证据）

### 1.2 体验—反思闭环（Kolb 圈）——「经验无反思即旅游」

- **跨 figure 共识**（figures: 王晓燕 / 段玉山 / 李璟晖）：学院派与营地实践派同点「反思/复盘」为命门。
- **一句话**：学习不是「去过/动手过」就发生，而要走完「具体经验→反思观察→抽象概念化→主动实践」的圈；**「反思/复盘」环节的有无，是研学区别于旅游的命门工序**。
- **它说的是**：Kolb 体验学习圈 + Dewey「并非一切经验都有教育性（要有连续性+交互）」的本土落地。行业把它简化为「行前-行中-行后」三段骨架，但**最易被砍掉的恰是「行后反思+成果」**——砍掉就退化成「玩过就忘」的旅游。这是把第一性矛盾从「立场」（§1.1）下沉到「可操作工序」的那一层。
- **应用方式**：设计任一研学/营期活动，确保每个体验环节都闭合 Kolb 圈；行后「反思+成果产出」缺失即返工。审方案时，先找「反思/复盘」在哪、是不是缩水成口头问「今天开不开心」。
- **局限**：（含放大度⚠️）Kolb/Dewey 是**通用教育学理论**，本身无行业排他性——**放大程度**在于：研学营地把「反思环节缺失」直接等同「产品质量不合格/沦为旅游」，赋予它一票否决式的鉴别权重，这是别的体验场景（如企业团建）不会给的。另：行业常把 Kolb 简化掉「四种学习风格」的个体差异维度（同一营期对不同孩子效果不同），是简化代价。
- **evidence**: [T04-S006, T04-S002, T03-W-S020]（canon Kolb+Dewey + workflows W3 命门 + 王晓燕专访）

### 1.3 「安全是 1，后面才是 0」——安全一票否决 / 风控前置门

- **跨 figure 共识**（figures: 李璟晖 / 王晓燕 / 赵蔚）：政策派与运营派罕见完全同频。
- **一句话**：安全/风控是研学营地的第一性约束，是「1」；体验、课程、规模、利润都是这个 1 后面的「0」——1 没了，后面多少 0 都没用。
- **它说的是**：本行业**罕见地让政策派与运营派完全同频**的硬约束。李璟晖明述「安全是第一重要的……没有安全这个一，后面多少零都没有用」；王晓燕「活动有方案、行前有备案、应急有预案」（三案口诀）；赵蔚把「风险评估」直接纳入营地方案。它是一道**前置门**：三案+三层险+配比(研学指导1:30/随团1:20/营辅按年龄分级)+对标 LB/T 054-2025 任一不达标，无论课程多好、SaaS 多先进，都不准成行（与 §2 playbook、§3 决策树 D 同构互证）。
- **应用方式**：任何活动成行前必跑安全前置门（贯穿行前-行中-行后）。它的优先级高于一切其他考虑——选型、设计、获客之前先过这一关。出事即重大法律与舆情风险，无可挽回性极高（涉未成年人）。
- **局限**：① 安全是**否决项不是加分项**——做到 100 分不会让产品变好，只是「准入」；过度安全化（如砍掉一切有挑战性的户外项目）会反噬体验与教育性（户外/冒险教育的 Hahn 一脉强调「挑战与反思」，见 §7）。② 具体配比**数值层在衰减**：LB/T 054-2025 刚换版（1:30 指导/1:20 随团），引用旧标 1:20 带队即过期；地方更严（常德 1:15、营地 ACA 低龄 1:6）。原则稳定 5+ 年，数值需随标/地标刷新。
- **evidence**: [T01-S015, T03-W-S020, T02-S009]（李璟晖原话 + 王晓燕三案 + 君合三层险合规；canon#16 / 决策树 D 互证）

### 1.4 逆向设计——「先定学习结果与评价，再排活动与景点」

- **跨 figure 共识**（figures: 段玉山 / 朱洪秋 / 王晓燕）：课程标准派的共同设计起点。
- **一句话**：先定「期望学习结果 + 评价证据」，景点和活动只是达成目标的**资源**（能砍则砍）；对治研学「先定去哪几个点、再硬给每站编个学习目标」的两张皮通病。
- **它说的是**：UbD 逆向设计（先结果→再评价→后活动）+ 布鲁姆可测目标（把「参观 X」改写成「能分析/能创造 Y」）的本土应用。是「资深 vs 入门」课程设计的最大分水岭：入门从「先排去哪」出发，资深从目标与评价反推。LB/T 092-2025 官方七步课程设计流程把「目标设计」放在「内容/实施设计」之前，是这条的国标级背书。
- **应用方式**：写研学手册/课程包时，先用布鲁姆动词定可测目标 + 定评价标准，再选能达成它的资源；项目式课用 PBL Gold Standard 七要素核对（尤其「真实性+公开成果+学生话语权」三项研学最易缺）。
- **局限**：（含放大度⚠️）UbD/OBE 是**通用课程设计法**，无行业独占——**放大程度**在于：研学营地因「景点诱惑」（现成的、好卖的、家长爱拍的）特别容易倒置设计顺序，所以这条在本行业的纠偏价值被放大。另：逆向设计偏「设计前置」，对临场带教反思帮助小（要配 Kolb §1.2）；短时一日营/低龄段用全套 UbD/PBL 偏重，可降级用更轻的 5E/Kolb。
- **evidence**: [T04-S026, T04-S021, T03-W-S016]（UbD canon#8 + 段玉山研学课程标准/课程评价 + workflows W3 资深 optimize + PBL）

### 1.5 「标准化骨架 + 在地化血肉」——资深的真正分野

- **跨 figure 共识**（figures: 赵蔚 / 李璟晖 / 王学辉）：营地实践派资深路径的共同母题。
- **一句话**：研学营地的资深 ≠「少做步骤」，而是**把可标准化的部分模板化复用（合规/安全预案/课程框架/报批材料/方案库）以提速骨架，把省下的全部精力押到不可标准化的高价值血肉（安全实地核查、教育性深度、在地资源差异化、口碑/转介绍）——骨架用模板，血肉绝不偷工**。
- **它说的是**：直接对应 intake 核心矛盾「标准化课程 vs 在地化资源」。入门 SOP 平均 6-9 步、几乎不敢跳；资深差异的主轴不是省步骤，而是**分清哪些能模板化、哪些是命门**。段玉山课程标准/朱洪秋三阶段四环节/张赫赫自然体验师培训代表「把体验做成可复制骨架」；而优质研学高度依赖在地独特资源（本地特有物种/物候、本地革命史细节），这是血肉，模板化即同质化死。
- **应用方式**：判断一个机构/方案是否「资深」——看它有没有标准模板库（合规/安全/课程/方案），同时看它在「安全实地核查、在地资源、反思深度、复购」这几处有没有偷工。自己做事时：骨架抄标准提速，血肉死磕在地与体验。
- **局限**：① 这条**预设了「有成熟标准可抄」**——但本行业标准刚换版（LB/T 054-2025），且垂直 SaaS 不成熟，「骨架」的可模板化程度低于成熟行业，骨架本身也要持续维护。② 「血肉绝不偷工」的临场 know-how（破冰话术/复盘仪式/私域转化细节）大量沉淀在黑名单渠道，本研究一手薄弱（见 §8），本模型给的是「骨架/血肉之分」的判断框架，不是血肉的具体配方。
- **evidence**: [T03-W-S006, T01-S003, T01-S017]（workflows 入门vs资深差距提炼[W1/W5 资深 skip/optimize/add] + 赵蔚营地研发/李璟晖自营营地的资深路径；直接对应 intake 核心矛盾「标准化课程 vs 在地化资源」）

### 1.6 重资产·教育理想 vs 轻资产·商业规模——尚未解开的方程

- **跨 figure 共识**（figures: 王学辉 / 赵蔚 / 李璟晖 / 葛文伟）：两条路线代表 + 资本侧旁证。
- **一句话**：研学营地的盈利模型有一道**结构性张力**：重资产+教育理想（自建营地/研发中心，客单价高、规模慢、叫好难叫座）vs 轻资产+商业规模（B 端集采走量，规模最大但毛利薄、连续亏损）——两条路线**都还没解开「教育理想与商业可持续」这道方程**。
- **它说的是**：商业模式层的共同母题，也是 figures 谱系的真实分歧。赵蔚（启行，自建北京/北戴河/南戴河/北海道营地+研发中心）、李璟晖（游美，4 个自营营地）= 重资产慢生意；王学辉（世纪明德，B 端校长渠道集采）= 规模最大但**连续四年亏损**（T05 财报硬证据，唯一可公开拉取的研学龙头财务序列）。葛文伟从资本侧旁证：「教育不是 winner-take-all 的生意」「争夺学生时间而非市场份额」。
- **应用方式**：做战略/投资判断时，先认清自己在哪条路线，别幻想「既重资产做出教育理想、又轻资产快速规模化盈利」——这道方程行业龙头都没解开。评估一个研学营地项目的可持续性时，把「教育属性强度」与「单位经济/盈利模型」放在张力关系里看，而非孤立看增长。
- **局限**：① 数字带 caveat——世纪明德连续亏损受疫情等不可抗力叠加，不能简单归因于「轻资产路线必亏」；游美转介绍率约 35% 是单机构数据非行业均值，不作 benchmark。② 这是**结构性观察不是解法**——本 OS 不声称知道怎么解这道方程（行业本身没解），它的价值是让人**别低估这道张力**、别把通用增长道理套上来。③ 市场规模各报告口径严重打架（1469亿/1791亿/1656亿/946亿2018口径各源），禁跨报告拼趋势线。
- **evidence**: [T01-S004, T05-S016, T01-S011]（王学辉 B 端走量 + 世纪明德 2024 年报连续亏损 + 葛文伟资本视角；赵蔚/李璟晖重资产对照 T01-S003/S017）

### 1.7 合规资格前置闸门——「先批后招 / 先名录后投标」

- **跨 figure 共识**（figures: 王晓燕 / 赵蔚 / 李璟晖）：政策派「三案/备案」与运营派 B 端实务同指。
- **一句话**：合规资格（报批通过 / 进入服务机构名录）是商业动作（招生收钱 / 投标抢单）的**硬前置闸门**——顺序反了必返工或废标：先收费再补报批=方案被驳要退费+信誉受损；没进名录就投标=资格审查直接废标。
- **它说的是**：政策驱动型 B 端服务业的特有节奏。研学是「先批后招」（向教育部门提交活动方案+安全预案+应急预案+家长告知同意书，审核通过方可成行、方可招生）；B 端采购是「先名录后投标」（先进省级研学服务机构名录——如黑龙江原则上每 2 年遴选 1 次、有效期 2 年、有除名机制——才有投标权）。这是 §1.3 安全前置门之外的**第二道前置门**（合规资格门），二者共同构成「做这门生意之前先过两道闸」。
- **应用方式**：启动任何招生/投标前，先核「报批通过了吗 / 名录资格在不在」。把名录资格+资质+过往合规业绩做成「随时可投」的资格包，公告一出即响应。中标后**不得转委托给未进名录的机构**（转包即违约+可能被除名）。
- **局限**：① 主要适用 **B 端（学校/教育局采购）**——纯 C 端家庭付费夏令营无教育局备案义务，这道闸主要是「安全/保险/营辅比」（§1.3）而非「报批/名录」。② 名录/黑白名单/廉政制度**正在收紧且 12 个月内大概率再出细则**（high decay），具体遴选周期/除名条件以各省最新办法为准。③ 研学采购涉财政经费+未成年人，廉政是持续红线（评标含家长代表占比≥60%〔个别地区门槛，比例各地不一，以当地政府采购办法为准，source: W-S007〕即制度性防腐），合规≠只过纸面闸门。
- **evidence**: [T02-S009, T02-S004, T03-W-S006, T03-W-S007]（君合合规/资质长稿 + T/CATS 研学指导师资质标准 + 常德报批全流程 + 黑龙江名录遴选/除名机制；决策树 D 互证）

## 2. 标准 Playbook (10 条决策启发式)

> 形如「**如果 X**，则 Y」。来源：从心智模型降级的候选（C8/C9/C10/C11）+ workflows 跨 SOP 反复出现的步骤模式 + figures 的决策标准。句式规整以便 Phase 3 cli_writer.py 抽取聚类。

1. **如果三案（活动方案/安全预案/应急预案）+ 三层险栈（校方责任险+旅行社责任险+人身意外险）+ 人员配比（研学指导≥1:30、随团服务≥1:20、营辅按年龄分级、每团≥1 专职安全员）+ 对标 LB/T 054-2025 任一项不达标**，则无论课程多漂亮、SaaS 多先进、报价多有竞争力，都**不准成行 / 不准投标**——这是先于一切设计与选型的硬前置门。
   - 案例：「只买一份意外险就出团」是最常见致命错（三层险赔付主体错位）；师生比套班级 1:40 是安全红线违规；无应急预案不准成行。(evidence: T02-S009, T02-S005, W-S006)

2. **如果一个方案没有可测学习目标 + 没有反思/复盘环节 + 没有评价或公开成果**，则它是旅游不是研学——立项即主动附「本项目对应哪条课标 / 综合实践活动方式（考察探究/社会服务/设计制作/职业体验）」的**对照表**自证教育性，否则过不了监管核查、家长验收、B 端评标。
   - 案例：方案=旅行社线路换皮被教育局认定「伪研学」；研学手册做成盖章打卡本（无任务/记录/评价表）验收不过。(evidence: W-S004, W-S011)

3. **如果是新基地 / 换季 / 雨季 / 新增高风险项目（涉水/高海拔/交通节点）**，则绝不凭图册或招商话术决策，必**实地踩点 + 标准核查清单逐项核**（安全设施/容量/食宿/应急/医疗 + 满员峰值动线 + 雨备室内替代场地）；仅长期合作、近期带过且无变更的成熟基地可凭既有踩点报告+电话复核跳过重踩。
   - 案例：凭招商图册签约→现场食宿/安全/容量缩水行中崩盘；旺季冲量塞满→集合/如厕/用餐动线崩，安全体验双输。(evidence: W-S008, W-S006)

4. **如果还没拿到合规资格（B 端报批未通过 / 未进服务机构名录）**，则先过这道闸再启动商业动作——**先批后招、先名录后投标**，绝不先收费再补报批、绝不无名录硬投标、绝不中标后转包给名录外机构。
   - 案例：先招生收费后报批被驳→退费+信誉受损；无名录资格投标被资格审查废标；转包名录外机构→违约+被除名。(evidence: W-S006, W-S007)

5. **如果机构营期少、预算有限、刚起步（单一夏令营级）**，则默认用**通用工具拼装**（微信小程序+金数据/腾讯收集表做报名+健康信息+监护授权+微信支付；需线上课/分销再叠加小鹅通或有赞教育）；只有当成体系运营、多营期多渠道、需安全档案/营辅排班/位置追踪时，才上垂直研学 SaaS（旅科云等），且采购前必试用+索取真实客户案例核验（行业自评垂直 SaaS「炒概念、实用性弱」，vendor 自报市占需独立核验）。
   - 案例：营期少时一上来买价格不透明的垂直 SaaS=ROI 低偏重；用通用教培系统（校管家）硬套纯营期制夏令营=无营期/线路/安全模块、错位反更贵。(evidence: T02-S019, T02-S010, T02-S012, T02-S014)

6. **如果在招聘或培训研学导师 / 营地辅导员**，则按**研学指导师四维能力（态度/知识/技能/发展）+ 课程设计能力**筛选，而不是用导游或带队老师的标准——导师管学习目标与课程，导游管行程交通，安全员管安全，三者分列配置不可一人兼；营辅需培训+背景审查。
   - 案例：现场只有导游讲解、无研学导师组织任务→学生「游而不学」家长差评；把 T/CATS 指导师证当「国家职业资格准入证」宣传=合规破绽（实为团体标准+测评）。(evidence: T04-S030, W-S003, T02-S004)

7. **如果在判定一个场所能不能用 / 怎么对标**，则先分清它是**「基地」还是「营地」**：基地=依托现有资源的单一主题场所（如博物馆），对标 LB/T 054-2025 服务侧；营地=含食宿、可承载多日营期的综合体，对标 T/CATS 002 设施侧 + ACA 营辅比分级；并核验等级真伪（国家级研学基地/营地仅 2017+2018 两批，之后不再统一命名，市面大量是省/市/机构自封）。
   - 案例：把普通景区/酒店当合规营地；把机构自封或低级别基地当国家级宣传被核查打脸。(evidence: T04-S031, T06-S008, W-S008)

8. **如果在选工具**，则按**「获客 / 交易分销 / 履约 / 课程设计 / 合规」五层分别选型，不混用**：抖音/视频号/小红书+美团/携程做获客与分销，小鹅通/有赞做交易，垂直 SaaS 或通用拼装做履约，Kolb/PBL/UbD/5E 做课程设计，LB/T 054-2025+三层险+配比做合规——一个工具想全包必有短板。**分销渠道 ≠ 课程工具**：用 OTA/本地生活的旅游逻辑做产品最易滑向「只游不学」。
   - 案例：把携程/美团/抖音团购当「研学课程工具」→重旅游轻教育触碰口碑红线；只靠小鹅通做线下履约→无营期排期/营员档案。(evidence: T02-S030, T02-S016, T02-S014)

9. **如果用 AI 研学方案/手册/线路生成器**，则只当**草稿层**，必用 Kolb/PBL/UbD 人工把关教育性与安全后才成品——AI 生成易堆砌景点、缺反思与评价（典型「只游不学」），当成品=把不合规的产品卖得更快。
   - 案例：AI 一键生成行程课时表直接卖，无行后反思与可测评价。(evidence: T02-S025)

10. **如果一期营/一个项目办得满意度高**，则立即把口碑转成**复购与转介绍引擎**（闭营典礼设家长参与+成果可晒+老带新优惠+营辅持续维系群），而不是让它停在偶然——C 端单位经济的命门是把单期口碑沉淀成复购资产，否则下期又从零买量、获客成本压垮利润。
    - 案例：游美营地公开转介绍率约 35%（单机构数据，非行业均值）；办得好但不收反馈不设转介绍→下期重新买量。(evidence: W-S019)

## 3. 工具栈与选型决策树

> 本节是对 Track 02 的**集成 + 一致性 sanity-check**，非重新提炼。详细工具卡片见 `references/research/02-tools.md`。

**结构性结论（C12，必须随工具栈一起说）**：本行业「工具」**不是软件为主，而是广义运营工具栈**，且**缺成熟的全行业垂直 SaaS，主力靠通用工具拼装 + 标准/框架当 checklist**。这是真实结论，非调研不足——垂直 SaaS（旅科云/朗睿/好多研学）真实存在，但主要面向 B 端政府/监管侧「一网通办」，C 端机构渗透低、价格不透明、行业自评「炒概念、实用性弱、售后难保障」。Phase 3 写 SKILL.md 时**必须在工具栈节明说这一结构性特征**，不要假装行业有成熟软件生态。

### 三层结构（消化 Track 02）

- **必备层（11 个，主体是「非软件工具」）**：① 合规/政策清单 6——LB/T 054-2025《研学旅游服务要求》(+091出境/092课程线路) / 教基一〔2016〕8号+综合实践活动纲要 / T/CATS 001(指导师)+002(基地营地)；② 课程设计框架 4——Kolb 体验学习圈 / PBL Gold Standard 七要素 / UbD 逆向设计+布鲁姆 / 5E 模型；③ 安全工具 3——三层保险栈 / 师生比·营辅比配比表 / 应急预案模板；④ 真正的「软件必备」仅 1——微信小程序+金数据/腾讯收集表的通用拼装底座。
- **场景特化层（10 个）**：垂直研学 SaaS（旅科云/朗睿/好多研学一网通办）/ 通用交易分销（小鹅通/有赞教育/校管家）/ 获客（抖音/视频号+美团/携程）/ 科技器材（Makeblock/乐高 SPIKE/思创优学无人机）/ 自然识别（iNaturalist/Seek/形色）。
- **新兴层（3 个，experimental，high decay）**：研学 AI 方案生成器（yxueai）/ 中经网络 AI 线路系统 / 视频号-抖音 AI 投流私域打法。

### 选型决策树（8 节点 + 1 前置门，消化 Track 02）

- **决策树 D（前置门，先于一切软件选型）**：合规与安全工具到位了吗？对标 LB/T 054-2025（地方查地标如北京 3.0）→ 三层险栈齐 → 师生比/营辅比按年龄分级达标 → 应急预案按本线路实地定制 → 课程用了 Kolb/PBL/UbD/5E 任一且有「行后反思+成果」。**这门不过，A/B/C 任何软件都只是把不合规的产品卖得更快。**（= §1.3 + §1.7 + playbook 1/4 在工具层的落地）
- **A（报名+收款+轻运营）**：A1 极轻→小程序+金数据/腾讯收集表+微信支付；A2 要线上课/分销→叠加小鹅通（偏内容/私域/裂变）或有赞（偏交易/分销佣金）。
- **B（成体系运营）**：B1 机构侧→垂直研学 SaaS 旅科云（差异点=安全监管位置追踪+电子档案+合规模块）；B2 已机构化有固定班型→校管家做教务财务底座+有赞/小鹅通做营期交易；B3 你是教育局/监管方→研学一网通办政府平台（监管侧，不解决机构招生）。
- **C（子赛道课程器材）**：C1 科技研学→Makeblock/乐高 SPIKE/思创优学无人机+5E 框架；C2 自然教育→iNaturalist+Seek+形色+LNT 七原则+LY/T 3329-2022。

### Sanity-check 摘要

- ✅ 必备层 11 个 ≥ 3（但结构性特征：非软件工具为主体，已如实标）。
- ✅ 选型决策树 8 节点（A1/A2/B1/B2/B3/C1/C2/D），每节点挂证据。
- ✅ 避坑清单 9 条 ≥ 5（见 §6 反模式，已整合 Track 02 避坑）。
- ⚠️ 薄弱：垂直 SaaS 真实市占/用户数/价格几乎全缺可信第三方数据（vendor 自报为主，逐条挂 caveat）；C 端营地一体化运营系统本土几乎空白（CampMinder/Campwise 是美国成熟产品，无 zh-CN 在地对标）。

## 4. 工作流 / Pipeline

> 本节集成 Track 03 的 7 个 SOP（完整 12 字段卡片 + 失败模式见 `references/research/03-workflows.md`）。每个都给「入门做法 → 资深路径（**跳过** / **优化** / **额外**）」。资深 ≠ 少做安全，而是 §1.5「标准化骨架提速、在地化血肉不偷工」。decay：high 2（W2/W7）· medium 2（W1/W5）· low 3（W3/W4/W6）。

### W1 一次完整研学旅行项目全流程（B 端校采 end-to-end，medium decay）

入门：需求对接→主题立项→课程设计→踩点选基地→安全评估+三层险→报批备案→招生→行前会→双师带队→研学评价→复盘（6-9 步几乎不敢跳）。
资深：**跳过**成熟基地的重复踩点与从零写方案（调标准模板库）；**优化**立项到报批用方案/手册/预案模板库从 2-3 周压到 3-5 天、用逆向设计先定评价再排活动；**额外**教育性自证对照表（对客/对教育局）、转介绍引擎。失败模式：先招生收费后补报批被驳回退费（W-S006）。

### W2 C 端夏令营/营地营期全流程（产品→获客→执行→复购，high decay）

入门：产品设计→营期排期→获客→小程序报名→营前准备→营期执行（破冰→课程→每日复盘→闭营）→家长反馈→续报。
资深：**跳过**厚研学手册（C 端轻量营靠体验+口碑，但安全/营辅比绝不跳）；**优化**私域+短视频本地生活获客、营辅比与物资用配比表模板；**额外**营前全要素压力测试、未遂事件复盘归档、转介绍/口碑可视化引擎。失败模式：旺季塞满致集合/用餐/如厕动线崩，安全体验双输（W-S008）。

### W3 课程设计 SOP（Kolb 圈 + PBL/5E + LB/T 092 官方七步，low decay）

入门：先选去哪→给每站编学习目标（两张皮通病）。
资深：**跳过**低龄/一日营的全套 UbD/PBL（降级用更轻的 5E/Kolb）；**优化**用布鲁姆动词定可测目标、逆向设计先定评价证据；**额外**每个体验环节闭合 Kolb 圈、强制「行后反思+成果产出」环节（缺失即返工）。失败模式：研学手册做成盖章打卡本（无任务/记录/评价）验收不过（W-S011）。

### W4 安全风控 SOP（行业生命线·最高优先，贯穿全程，low decay）

入门：买一份意外险+派老师带队即出团。
资深：**跳过**——无可跳（安全是 §1.3 的「1」，零容忍）；**优化**三案做成一车一案/一队一案标准模板 + 定期演练；**额外**每团≥1 专职安全员不兼岗、三层险栈（校方责任险+旅行社责任险+人身意外险，出境加全球救援）、未遂事件归档。失败模式：只买一份意外险出团→赔付主体错位致命（W-S006）。

### W5 选研学基地/营地 SOP（对标 LB/T 054-2025 + T/CATS 002，medium decay）

入门：看招商图册/朋友推荐就签约。
资深：**跳过**——选基地不可跳实地核查（红线）；**优化**对照标准做核查清单模板、主题资源匹配度评分；**额外**安全设施一票否决项前置、容量与食宿动线实测、淡旺季容量条款。失败模式：凭图册签约→现场食宿/安全/容量缩水行中崩盘（W-S008）。

### W6 主题特化设计流程（自然/红色/科技各有命门，low decay）

入门：套通用研学模板换个主题名。
资深：**跳过**与主题无关的通用环节；**优化**自然教育归口林草+LNT、红色研学重党史脉络+仪式感、科技研学用 5E/PBL 任务式；**额外**子赛道合规（自然教育 LY/T 3329、出境游学 LB/T 091-2025）、在地一手资源深挖。失败模式：红色研学做成「到此一游+拍照」无党史脉络、被认定形式主义。

### W7 B 端政府采购/招投标/履约全流程（含合规与廉政红线，high decay）

入门：有方案就投标、低价中标。
资深：**跳过**无名录资格的标（先进名录再投，否则资格审查废标）；**优化**方案库/报价模型模板化；**额外**廉政合规自查（评标家长代表占比、财政经费留痕）、履约不缩水承诺。失败模式：低价中标后履约缩水砍师资/安全（W-S006）；转包名录外机构被除名（W-S007）。

### 出团前置安全合规 checklist（跨 W1/W4/W5/W7 复用，最高优先）

资质核验→配比（研学指导≥1:30/随团≥1:20/营辅按 ACA 年龄分级/每团≥1 专职安全员）→三案齐备→三层险栈→食宿交通医疗核查→**报批前置**（先批后招/先名录后投标）→双师带队→研学评价→复盘归档。资深**优化**：全部做成可勾选模板库；**额外**：全要素预演 + 未遂事件归档。任一不达标，方案再好、SaaS 再先进都不准成行（与 §3 决策树 D 互证）。

### 近期工作流变化（三股外部驱动力 + AI 横切）

- **政策—标准驱动**：LB/T 054/092-2025 换版（2025-05），合同/手册/培训引用旧标号 054-2016 即合规破绽。
- **法规/监管驱动**：文旅部 2024-11 治理伪研学 + 名录/廉政收紧，「教育性自证」从加分项变隐性必答。
- **平台/渠道驱动（C 端，最易变）**：获客转视频号/抖音/美团本地生活，平台规则月度变。
- **AI 横切**：课程/行程/手册/营销 AI 生成（仅草稿，影响「资料生成」子环节非主干）。
- ⚠️ 薄弱：C 端临场运营层（破冰话术/复盘仪式/私域转化 SOP）一手沉淀在黑名单渠道，本研究无一手，用标准条款+运营者专访兜底；红色/科技研学流程细节靠政策+框架推导。

## 6. 质量基准 + 反模式

### 什么算「好」（5 条具体可验证基准）

- **教育性可被第三方看见**：方案有「课标/综合实践活动方式对照表」、研学手册有任务单+观察记录+评价表（不是盖章打卡本）、有可展示的公开成果（调查报告/自然笔记/路演）——监管核查、家长验收、B 端评标都能一击命中。(来源: W-S004/W-S011, canon#13/#14)
- **Kolb 圈闭合**：每个体验环节都有「反思观察+抽象概念化」，行后有反思+成果产出环节——而非把反思缩水成口头问「今天开不开心」。(来源: 段玉山课程评价 T04-S021, W-S020)
- **学习目标可测**：用布鲁姆动词写成「能分析/能创造 Y」（结果），而非「参观 X」（活动）；目标先于活动设计。(来源: T04-S026/S032)
- **安全闭环达标且可追溯**：三案齐+三层险+配比按年龄分级+全程专职安全员（不兼岗）+ 演练记录 + 含未遂事件的复盘归档。(来源: W-S003/S013, ACA T02-S005)
- **C 端口碑成资产**：转介绍/复购被做成引擎（闭营成果可晒+老带新+持续维系），而非偶然——单位经济可摊薄获客成本。(来源: W-S019，游美 35% 转介绍为单机构参照非 benchmark)

### 反模式（外行 / 入门常犯，整合 T06 outsider-tell + T03 失败模式 + T02 避坑）

- **只游不学 / 游而不学**（耻辱线）——为什么错：研学/营地是挂在国家必修课的校外**教育**活动，无目标/反思/评价就是旅游，2024-11 文旅部专门发文治理伪研学。
- **报批前置做反**（先招生收费再补报批）——为什么错：方案被驳要退费+信誉受损；合规资格是商业动作的硬前置。
- **三层险只买一层**（以为一份意外险够）——为什么错：校方/旅行社责任险与人身意外险赔付主体不同，缺层即保障缺口，出事赔付错位。
- **师生比/营辅比套班级 1:40**——为什么错：营地按 ACA 年龄分级最严 1:6，超标是最常见安全红线违规。
- **双师退化为单导游**——为什么错：只有导游讲解无研学导师组织任务，学习目标落空，「游而不学」。
- **应急预案照抄通用模板不演练**——为什么错：真出极端天气/事故时无人会用，照抄模板=形同虚设。
- **景点堆砌、目标后补**（Kolb 断环 / 先定景点再凑目标）——为什么错：目标与活动两张皮，有体验没反思，玩过就忘。
- **把 OTA/本地生活平台当课程工具**——为什么错：携程/美团/抖音是分销获客渠道、重旅游轻教育，用其逻辑做产品=滑向只游不学；渠道≠课程设计工具。
- **把 T/CATS 指导师证当国家职业资格宣传**——为什么错：它是团体标准+测评，非国家准入资格，夸大=合规破绽。
- **AI 生成方案直接当成品**——为什么错：AI 易堆砌景点、缺反思评价（只游不学），必须当草稿+人工把关。
- **混淆基地/营地、把自封基地当国家级**——为什么错：国家级仅 2017+2018 两批，普通景区/酒店≠合规营地。
- **低价中标后履约缩水 / 转包名录外机构**——为什么错：研学采购是竞争性磋商非最低价中标，低价中标=砍师资/安全/保险埋雷，转包即违约被除名。
- **自然教育做成「公园认花」/ 红色做成「景点打卡」/ 科技做成「玩具展示」**——为什么错：各子赛道命门（自然=方法+LNT 伦理、红色=思政课程化+仪式、科技=5E 的 Explain/Elaborate 建构迁移）被跳过。
- **还在引用 LB/T 054-2016**——为什么错：2025-05-19 已废止，引旧标号=合规知识过期破绽。

### 主要流派与当代代表（≥2，保留分歧）

#### 流派 A：教育派 / 学院派课程标准
- 奠基：杜威（经验论）→ 泰勒（课程原理）；本土学术化锚点段玉山《研学旅行课程标准》系列（2019）、朱洪秋三阶段四环节（2017）。
- 当前代表：段玉山（华东师大，地理学科本位、课程标准论文）、王晓燕（中国教育科学研究院，政策—实施解读）、朱洪秋（泰勒式三阶段四环节）。
- 核心主张：研学是「课程」不是「行程」，自上而下课程化、纳入教学计划、有目标—实施—评价闭环。

#### 流派 B：营地现场实践派
- 奠基：Kurt Hahn/Outward Bound（户外冒险）+ ACA（营地标准）；本土系统引入者赵蔚（启行营地研发中心）。
- 当前代表：赵蔚（启行）、李璟晖（游美「营地是表、教育是本、活动是核心」）、王学辉/青青部落。
- 核心主张：自下而上从营地/现场真实体验出发，「没有围墙的学校/教室」，重体验设计与运营。

#### 流派 C：公益 / 自然教育派
- 奠基：Louv 自然缺失症（2005）；本土思想源头梁晓燕（自然之友联合发起人，「参与式/动手/互动」环境教育范式 + 「教育的本质是慢」）。
- 当前代表：张赫赫·蚊滋滋（自然之友—盖娅，自然体验师培训标准化）。归口国家林草系（LY/T 3329-2022），与教育/文旅口径的研学交叉但不同治理来源。
- 核心主张：重建人与自然联结，公益/理念驱动，人才培养标准化；价值观上游但体量小、付费转化弱。

#### 流派 D：资本 / 商业规模派
- 当前代表：王学辉（世纪明德，B 端集采走量）；外部参照系葛文伟/钟日昕（多鲸资本，「非 winner-take-all」「争夺学生时间」「与时间做朋友」）。
- 核心主张：从市场结构/集中度/单位经济看赛道，规模化与盈利模型优先（但承认教育本质难被技术改变）。

### 流派间正在吵什么（保留不调和）

1. **教育属性 vs 旅游属性的配比（第一性矛盾）**——监管/教育派要「教育主导」，市场化机构靠旅游化体验与成本控制求商业可持续。语言层硬证据：同一「研学旅行」有教育部口径（校外**教育**活动 T06-S001）vs 旧国家旅游局口径（教育**旅游**活动 T06-S004）两套权威定义并存。**未抹平，至今校正中。**
2. **学院派课程标准 vs 营地现场实践派**——A 派被批「过度学科化/程序化」（朱洪秋三阶段四环节被批「泰勒模式的研学翻版，对生成性体验包容不足」）；B 派被批「弱学术、家长成果焦虑难安放」（纯美式营地弱学术导向 vs 中国家长对学术成果的诉求）。
3. **重资产·教育理想 vs 轻资产·商业规模**——赵蔚/李璟晖（自建营地，叫好难叫座）vs 王学辉（B 端走量，规模最大但连续亏损）。**两条路线都没解开「教育理想与商业可持续」方程**（= §1.6）。
4. **自由 vs 结构**——夏山式自由教育理念 vs 营地/研学的目标导向与安全结构化，营地实践中持续拉扯（户外教育的 Hahn 一脉也强调「挑战需配反思与社会服务」，反对纯探险刺激）。
5. **本土理论原创性不足（学界自省）**——中国研学正典高度移植西方（杜威/Kolb/PBL/UbD），本土原创模型（三阶段四环节等）数量与影响力有限，操作知识主要由政策/标准/职业标准托底，而非本土学术奠基专著。

**治理来源三分（祖师爷三谱系，对应流派分治）**：研学归口教育部+文旅部（教育学+旅游业双祖师爷，已现双口径之争）；营地教育靠行业联盟 CCEA 自律（美式营地+户外祖师爷）；自然教育归口国家林草系 LY/T 标准（环境教育祖师爷）。三套治理来源是流派分裂的制度硬件层。

## 8. 诚实边界

> 重写实。本行业一手率结构性偏低且**轨间分化显著**，figures 维度偏冷，标准刚换版——这些不是调研不力，是行业结构使然，但必须如实告知。

- **信息截止 2026-06-03。合规/工具/获客模块衰减最快**（建议每 3-6 月 update 大师 study-camp-education）：标准刚换版（LB/T 054/091/092-2025，2025-05 实施，配套省级管理办法/地标/团标 12 个月内大概率再出细则，概率 >40%〔本蒸馏主观估计、非实测数据，仅作衰减提示〕）；C 端获客打法随抖音/视频号/美团平台规则月度变；研学指导师是否纳入国家职业资格目录存政策不确定性；过去 12 个月内有修订/新发的标准法规 ≥4 个（强时效衰减信号）。

- **一手率结构性偏低，且轨间分化大**：政策/标准/学术轨高（workflows 81% verified+surrogate、glossary 64%、canon 61-68%——因流程/定义被写进政府文件与团标）；**figures 仅约 35%、sources 仅约 15%、运营/获客层偏低**。根因：zh-CN 一手运营内容（运营复盘、课程方法论原话、破冰/复盘/私域 SOP）大量沉淀在微信公众号/知乎/视频号——本 pipeline 信源黑名单，一律不取用，大量启用 surrogate_primary（政策原文/团标/协会官档/律所合规长稿/政府平台/vendor 官档）兜底。

- **figures 维度偏冷（冷僻协议生效）**：retained 仅 11 位（< 15 floor），**无人达 sub_skill_candidate=true，无 high 可信度**（最高 medium，含 2 位 medium-low）。行业以「政策制定者+机构运营者+学术研究者」为主，**缺个人思想领袖型长访谈**；中文研学/营地**无一档持续运营的从业者长访谈播客**（podcast 维度近乎空白，唯一亮点张赫赫盖娅星球未转写）。故本 OS 以「群像+流派分歧+共识心智模型」方式提炼，**不是「大师个人 OS」**，也不宜对任一人做深度个人思想框架蒸馏。

- **voice 维度薄弱（voice_confidence: medium）**：7/11 figures 取得 ≥1 段可引用原话；王学辉、钟日昕、朱洪秋**完全无可引用原话**，段玉山为学术语体无对话原话。§5 对话样本库的「同业私下版」部分靠转述/推断重构（私下 register 大量在黑名单渠道），已逐段标 原话/转述/推断。SKILL.md §5 风格输出中，**「客户版/监管专业版」有真行业语料支撑，「同业私下版」部分靠 LLM 重构而非一手语料**。

- **红色研学 / 科技研学细节靠推导**：两条子赛道在 figures 维度**结构性缺标志性个人**（红色=政策驱动、代表人物分散于地方协会/高校；科技=果壳类不属本 scope）。其流程/打法主要靠政策文件（教基一8号育人导向、大思政课）+ 通用框架（5E/PBL）+ 标准推导，**一手运营细节薄弱**。出境游学同理依赖政策与汇率/地缘，材料偏旧。

- **数字一律带 caveat，禁拼趋势线**：市场规模各报告口径严重打架——研学市场有「1469 亿」(前瞻)、「2024 达 1791 亿/+21.9%」(艾媒)、「2024 达 1656 亿/+12%」(观知海内)、「泛游学营地 2018=946 亿」(艾瑞，疫情前口径)多个版本，引用任何数字必须挂具体来源+年份+口径，**禁止跨报告拼接趋势线**。游美转介绍率约 35% 是单机构数据非行业均值，不作 benchmark；世纪明德连续四年亏损是反例如实记（受疫情等不可抗力叠加，不简单归因「轻资产必亏」）。LB/T 054-2025 具体配比数值（1:30 指导/1:20 随团）以新标为准，地方更严（常德 1:15、营地 ACA 低龄 1:6）。

- **本土学术 canon 厚度偏薄**：中国研学正典高度移植西方理论，本土 textbook 多为「编著教材」而非奠基专著，被 ≥3 独立来源点过的「真·必读」本土书目偏少；无稳定公开 MOOC（课程多为收费/线下/机构内训）。操作知识由政策/标准/职业标准托底。

- **master skill 不能替代真实从业经验**。涉及具体雇佣 / 投资 / 法律 / 报批合规决策时，必须找真人 + 查最新标准原文（文旅部 zwgk.mct.gov.cn / 教育部 moe.gov.cn / 当地教育局办法）确认，尤其因标准刚换版、地方差异大。

## 9. Agentic Protocol — 研究维度

> 标准复杂度（7 心智模型 → 7 维度，未超 ×2）。每个维度从心智模型反推「首先关注什么」，凝结成可被搜索动作触发的研究方式。互斥性已自检（见文末候选清单），7 维各产出不同结论。Phase 3 cli_writer.py 会把本节直接编排进 cli/protocol/agentic.sh，「看什么/在哪看/输出格式」是脚本必填字段。

### 9.1 政策与标准合规（最新标准号 + 报批 + 名录）
- **看什么**：现行研学标准的**最新版本号与实施日期**（重点确认 LB/T 054-2025 已替代 054-2016，是否再有更新版/配套省级办法）；目标地教育局《研学实践管理办法》的报批—备案—审批流程与时间节点；目标省份的研学服务机构名录遴选周期与是否在册；近 12 个月新发的监管文件（如文旅部治理伪研学类）。
- **在哪看**：文旅部 zwgk.mct.gov.cn（行业标准公告/通知）、教育部 moe.gov.cn、目标省市教育局/文旅局 .gov.cn 官网（报批办法/名录公告）、全国团体标准信息平台 ttbz.org.cn（T/CATS）、国家数字标准馆 ndls.org.cn。**不查百度百科/微信/知乎**。
- **输出格式**：「现行标准：LB/T 054-{版本}（实施 {日期}）+ 出境 091 + 课程线路 092；目标地报批流程 = {步骤+时间节点}；名录资格 = {在册/需申请，周期 {N} 年}；近期监管变化 = {有/无 + 文件名日期}。」

### 9.2 学习目标与课程设计（逆向设计 + Kolb + PBL/5E）
- **看什么**：目标学段/主题对应的**可测学习目标**怎么写（布鲁姆动词层级）；该主题适配哪个课程框架（多日项目→PBL Gold Standard 七要素；单次探究→5E；通用→Kolb 圈+UbD）；LB/T 092-2025 官方七步课程设计流程与四类手册规范；如何嵌入「反思+公开成果」环节。
- **在哪看**：LB/T 092-2025 条目（文旅部 zwgk.mct.gov.cn）、PBLWorks pblworks.org（Gold Standard 七要素）、BSCS 5E（.edu 托管 PDF）、段玉山《研学旅行课程标准》系列（核心期刊）、本 skill canon 模块。**不依赖 AI 生成器输出当结论**。
- **输出格式**：「推荐框架 = {Kolb/PBL/5E/UbD 哪个+理由}；可测目标示例 = 「能{布鲁姆动词}…」×N；反思/成果环节 = {具体设计}；是否闭合 Kolb 圈 = {是/否+缺口}。」

### 9.3 安全风控与配比（三案 + 三层险 + 营辅比）
- **看什么**：本次活动类型对应的**人员配比数值**（研学指导/随团/营辅按年龄分级/安全员，以 LB/T 054-2025 + 地标 + ACA 为准）；三案（活动/安全/应急预案）是否需一车一案+演练；三层险栈投保主体与保额（出境加全球救援险）；高风险环节（涉水/高海拔/交通）的专项处置。
- **在哪看**：LB/T 054-2025 安全管理章（文旅部）、目标地地标（如北京未成年人研学服务规范）、ACA acacamps.org（营辅比年龄分级）、君合等律所合规长稿（三层险责任划分）、本 skill 工具栈安全三件套。
- **输出格式**：「配比要求 = 研学指导 ≥{X}、随团 ≥{Y}、营辅 {年龄段:比例}、安全员 {N}/团；三案 = {齐/缺哪项}；三层险 = {校方/旅行社/意外 投保主体齐否+出境救援}；高风险专项 = {有/无}。」

### 9.4 基地 / 营地选型（基地≠营地 + 资源匹配 + 安全设施）
- **看什么**：候选场所是**「基地」还是「营地」**（对标 T/CATS 002 / LB/T 054 / ACA 哪套）；其等级认定**真伪**（是否国家级 2017/2018 两批之一，还是省/市/自封）；资源能否支撑课程学习目标（主题对口）；安全设施核查（CCTV/疏散/医疗/周边隐患）、满员容量与雨备替代场地。
- **在哪看**：T/CATS 002 条目（国家数字标准馆/团标平台）、教育部 2017/2018 两批国家级名单（moe.gov.cn 原文核验等级）、目标省市研学基地名录、实地踩点报告（不可凭 vendor/招商图册）、ACA 标准。
- **输出格式**：「场所类型 = 基地/营地（对标 {标准}）；等级 = {国家级哪批/省市级/自封，已核验}；资源匹配 = {支撑/不支撑学习目标};安全设施 = {核查清单结果}；容量/雨备 = {满员动线+室内替代}。」

### 9.5 商业模式与获客（B 端校采名录投标 vs C 端私域短视频转介绍）
- **看什么**：这是 **B 端（学校/教育局采购）还是 C 端（家庭付费）**生意，对应完全不同打法——B 端走名录资格+竞争性磋商+教育性方案竞争（非最低价）；C 端走视频号/抖音/小红书种草→小程序/企微私域→美团/携程分销+转介绍引擎；以及对应的工具分层与单位经济（重资产 vs 轻资产张力）。
- **在哪看**：目标地政府采购/招标公告平台（B 端真实标的+竞争性磋商规则）、多鲸《素质教育趋势报告》营地研学专章（市场结构/客单价/集中度，注意口径 caveat）、世纪明德新三板财报（龙头盈利模型反例）、抖音/美团本地生活规则页（C 端获客）。
- **输出格式**：「生意类型 = B/C 端；B 端 = {名录资格+采购方式+教育性卖点}；C 端 = {获客渠道+私域闭环+转介绍机制}；资产模型 = 重/轻 + 单位经济关键变量；数据口径 caveat = {引用了哪份报告哪年}。」

### 9.6 主题特化（自然 / 红色 / 科技 / 出境 各自命门 + 子赛道合规）
- **看什么**：子赛道**专属命门与合规**——自然教育（在地物种/物候 + LNT 七原则 + LY/T 3329-2022 林草系归口）；红色研学（党史脉络思政课程化 + 仪式感 + 对接大思政课，B 端为主）；科技研学（5E 的 Explain/Elaborate 建构迁移 + 可展示作品 + STEAM 器材）；出境游学（LB/T 091-2025 + 教育部赴境外研学指南 + 汇率/地缘）。
- **在哪看**：自然—国家林草局 forestry.gov.cn（LY/T 3329 解读）+ lnt.org；红色—教基一8号育人导向+地方爱国主义教育基地名单；科技—PBLWorks/BSCS 5E+器材 vendor 官页；出境—教育部 jsj.moe.gov.cn 赴境外研学指南+LB/T 091。
- **输出格式**：「子赛道 = {自然/红色/科技/出境}；命门 = {该子赛道专属方法}；专属合规 = {LNT/思政/作品化/出境险与指南}；在地差异化资源 = {本地独特资源点}。」

### 9.7 质量与口碑（教育性可视化 + 反思闭环 + 评价对接 + 复购）
- **看什么**：产品的**教育性证据是否可被第三方看见**（对照表/手册质量/公开成果）；Kolb 圈是否闭合（反思不缩水）；学习评价是否对接学分/综合素质评价；C 端口碑是否成复购/转介绍资产；对照行业反模式自查（只游不学/打卡/玩具展示/手册=纪念册）。
- **在哪看**：本 skill §6 质量基准+反模式、段玉山课程评价/表现性评价（核心期刊）、UbD 评价证据观、运营者公开案例（如游美转介绍模型，作参照非 benchmark）、目标地综合素质评价对接政策。
- **输出格式**：「教育性证据 = {对照表/手册/成果 有无}；Kolb 闭合 = {是/否};评价对接 = {学分/综合素质评价 是否};口碑资产 = {复购/转介绍机制有无}；反模式自查 = {命中哪几条需修正}。」
