# Track 03 — Workflows / SOP / Pipeline：医疗美容 / 注射美容（locale=zh-CN）

> 本行的工作流横跨「临床医疗链」与「机构经营链」两条主线。第一性背景：**医疗本质 + 强监管 + 消费属性三重叠加**——所以每个临床 workflow 的「入门 SOP」第一约束永远是**安全与合规**（解剖避险 / 知情同意 / 资质械字号），不是美学最大化或转化率；每个经营 workflow 的近期变化几乎都被**监管 / 平台政策**改写。
>
> 工作流是 master skill 衰减最快的部分。本文件对每个 workflow 强制标 `last_updated` + 近期变化（变化 or 明确「稳态」）+ decay risk。**核心临床避险/并发症急救层衰减慢（low）；监管合规 + 获客平台 + 新品选型层衰减快（high）。**
>
> 安全/并发症流程一律基于专家共识 / 指南 / gov 监管原文 / 期刊，**诚实写出风险，绝不软化成「零风险」**。数字一律挂 source_id 或 caveat（专家共识 / 文献 / 业内估计 / 上市公司 / 机构数据）。
>
> 一手占比估计 ~55%（gov 法规 + FDA + PMC/PRS/ASJ 期刊全文 + NMPA/卫健委查询 + 学会共识为一手主体；临床注射手感属高 tacit，共识只给原则；机构经营数据、平台口径、营销趋势诚实留 secondary）。Wave 1+2 贡献 seed 充足（figures 叙事 + 工具典型场景 + canon 标准流程 + glossary 流程暗示），未触发冷僻协议。

## 详细条目

### 1. 求美者面诊与需求挖掘（含 BDD / 容貌焦虑筛查）

- **One-liner**: 从「一个带着模糊变美诉求来的人」到「一份主诉清晰、禁忌排除、预期对齐、并明确『该做 / 该劝退』的医疗诊断」。
- **Trigger**: 求美者到诊（线上引流来或转介绍），提出「我想变好看 / 想做某项目」。 evidence: [T03-S001]
- **Output**: 主诉与动机记录、解剖与皮肤评估（Fitzpatrick 分型）、既往史与禁忌排查、BDD/容貌焦虑初筛、预期对齐 + 「做 or 劝退」决策、初步方案方向。
- **入门 SOP（minimum viable steps）**:
  1. **接诊与建档**：核对身份、建电子病历、采集主诉（「你最想改善哪里」），区分「求美者」诉求 vs 销售报价（这是医疗诊断不是聊价）。 evidence: [T03-S001]
     - 跳过后果：无病历=违反《医疗美容服务管理办法》且日后纠纷无举证依据。
  2. **既往史与禁忌排查**：妊娠哺乳、过敏史、凝血/免疫疾病、感染、瘢痕体质、近期填充/线材史、用药史；排禁忌证。
     - 跳过后果：禁忌人群（如孕妇、活动性感染、自身免疫病）做项目=并发症/医疗事故高危。
  3. **解剖与皮肤评估**：面部分区视诊触诊 + Fitzpatrick 皮肤分型（决定光电参数与色沉风险）；可用 VISIA 客观量化。 evidence: [T03-S014]
     - 跳过后果：层次/参数错配（深肤色激进剥脱致色沉、容积/松弛误判）。
  4. **动机与预期对话**：问「为什么现在想做 / 想达到什么效果 / 看过谁的样子」；把不切实际预期（「变成某明星」「一次解决全部」）拉回现实。
     - 跳过后果：预期落差=术后投诉与维权的头号根源。
  5. **BDD / 容貌焦虑初筛**：诉求异常强烈、反复修整、对微小瑕疵病理性关注、多次他院不满意 → 警惕体象障碍，触发**劝退**而非升单。 evidence: [T03-S002]
     - 跳过后果：给 BDD 患者做项目几乎注定不满意 + 反复修整 + 成瘾，违劝退伦理。
  6. **给出方向或劝退**：适应证清晰→进入方案设计；无适应证/禁忌/BDD→诚实劝退（「这个不必做 / 你现在不适合做」）。
     - 跳过后果：无适应证强做=过度医疗，踩医疗伦理与监管红线。
- **资深路径（差异点）**：
  - **skip**：资深医生**跳过**冗长标准化问卷的逐题朗读——3-5 句开放式提问 + 一次触诊即抓住真实主诉与禁忌信号，但禁忌与 BDD 这两关绝不跳（经验=「问得少但问得准」）。
  - **optimize**：把「动机挖掘」做深——分辨「为自己 vs 为取悦他人/婚恋/职场焦虑」，前者可做，后者高度警惕；用 VISIA/3D 把抽象诉求转成可指认的客观问题，降低预期落差。 evidence: [T03-S014]
  - **add**：资深医生**额外做** BDD/容貌焦虑的主动筛查与「劝退话术」储备（郭树忠「信任而非收入是激励」、何黎「改善而非过度改变」是这一行劝退伦理的人格化）；并把「这个项目你其实不需要」说出口——初学者怕丢单不敢劝退。 evidence: [T03-S002]
- **近期变化（近 12 个月）**：
  - 2024–2025：**容貌焦虑筛查从「软技能」升级为「合规硬约束」**——《医疗美容广告执法指南》明令禁「制造容貌焦虑」，机构面诊话术须合规改造，不得以制造焦虑驱动升单。触发事件类型：法规/监管。 evidence: [T03-S002, T03-S013]
  - 2025：**AI 测脸 / 大模型问诊进入面诊前置环节**——更美「更美AI」、AI 面部分析按黄金比例生成多套微调方案、按肤龄给差异化建议（如方下颌+年轻肤质荐瘦脸针、老化肤质荐填充）。机构报告面诊效率提升、方案返修率下降（机构口径，须 caveat）。但 AI「审美平均化」会忽略个体解剖差异（网红鼻致呼吸问题），资深医生把 AI 当沟通/初筛工具而非诊断决策。触发事件类型：新工具/新模型。 evidence: [T03-S016]
- **典型耗时**：入门 SOP 30–60 分钟/人；资深路径 15–30 分钟（抓得更准）。
- **关键工具（与 Track 02 关联）**：电子病历/HIS（必备）、VISIA 皮肤检测（必备）、AI 测脸/3D 模拟（新兴，沟通工具）。
- **关键人物（与 Track 01 关联）**：郭树忠（劝退伦理「信任而非收入」）、何黎（「改善而非过度改变」「理性消费」）在专访中详谈面诊与劝退。
- **常见失败模式**：
  - **把面诊当销售而非诊断**：触发原因—咨询师主导、以转化率为 KPI。避免：医生主导、医疗诊断框架先行，把面诊当成做不做的医学判断。
  - **跳过 BDD 筛查给焦虑型客户做项目**：会导致反复修整 + 投诉 + 成瘾。actionable：诉求异常强烈/反复修整/多次他院不满意=触发劝退而非排期。
  - **预期不对齐就开做**：actionable—不要在求美者说「变成某明星/一次全解决」时直接排期，先用 VISIA/3D 把可达成范围讲清并书面记录，否则术后落差几乎必投诉。

### 2. 审美设计与方案规划（全脸结构化设计）

- **One-liner**: 从「求美者指着某一处说想改」到「一份基于面部分区/比例/动态/老化机制的全脸结构化方案——注射 vs 手术 vs 联合、单项 vs 联合、分次 vs 一次」。
- **Trigger**: 面诊确认有适应证，需要把诉求落成可执行方案。 evidence: [T03-S005]
- **Output**: 全脸/分区设计（容积+轮廓+动态）、术式分流（注射/光电/线雕/手术/联合）、分次规划与预算、可视化预期。
- **入门 SOP（minimum viable steps）**:
  1. **面部三维评估**：按三庭五眼/黄金比例 + 容积缺失（中面部老化核心）+ 动态（表情肌）+ 支撑（骨吸收/SMAS 松弛）分层评估，而非「头痛医头」。 evidence: [T03-S005]
     - 跳过后果：只填求美者指的那一处，整体不协调（如只填法令纹不评估中面部容积，越填越显老）。
  2. **机制→术式分流**：容积缺失→填充；动态皱纹/肌肥大→肉毒；肤质问题→光电；松弛按层次（浅射频/深超声/线雕/手术）。 evidence: [T03-S005]
     - 跳过后果：术式错配（肉毒填容积、填充治动态纹）=无效甚至更糟。
  3. **注射 vs 手术 vs 联合判断**：轻中度→微创注射/光电；重度松弛/结构问题→手术；多数复杂面容→联合方案。
     - 跳过后果：该手术的硬用注射堆（如重度松弛只打玻尿酸=「填充脸/馒化」）。
  4. **分次与预算规划**：把全脸方案拆成分次（先打地基支撑后做精细，留量观察），匹配求美者预算与恢复期。
     - 跳过后果：一次过量注射=不可控、过度医美、修复难。
  5. **可视化与方案确认**：用 VISIA/3D 模拟把方案讲给求美者，对齐预期，记录方案。 evidence: [T03-S014]
     - 跳过后果：方案停留在医生脑内，求美者预期与结果脱节。
- **资深路径（差异点）**：
  - **skip**：资深医生**跳过**对编码/模板的机械套用——MD Codes 等分区框架在手，但不逐码套；按个体解剖差异调整（学院派批评低年资「机械套 MD Codes 忽视个体」）。 evidence: [T03-S005]
  - **optimize**：把「容积 + 动态 + 老化预测」联合评估——不只看静态当下，预判 5–10 年老化轨迹做支撑性设计（先骨膜上支撑后浅层精细）；全脸设计而非单点。 evidence: [T03-S005]
  - **add**：资深医生**额外做**分次规划与「留白」——先做关键支撑点、留量观察 2–4 周再补，避免一次到位的不可逆风险；并把「自然 vs 改造」审美取向与求美者明确对齐（反「科技脸/欧式大平行」过度改造）。
- **近期变化（近 12 个月）**：近 12 个月**无重大方法学变化（OS 核心层稳态）**——全脸结构化/容积思维/分区设计自 MD Codes(2021) + 中国面部年轻化共识本土化以来稳定。唯一边际变化是 AI/3D 把「可视化预期」环节工具化（见 workflow 1 近期变化）。触发事件类型：无（稳态）。 evidence: [T03-S005]
- **典型耗时**：入门 SOP 20–40 分钟（含可视化）；资深路径 15–30 分钟（设计更立体但更快）。
- **关键工具**：MD Codes 方法学（必备认知）、VISIA/面部 3D 模拟（评估+沟通）、AI 测脸（新兴）。
- **关键人物**：de Maio（MD Codes 全脸分区设计）、吴溯帆（解剖功底优先于框架）、中国面部年轻化共识执笔体（容积思维本土化）。
- **常见失败模式**：
  - **单点思维/头痛医头**：只处理求美者指的部位，整体失衡。避免：永远做全脸结构评估再下针。
  - **机械套编码忽视个体**：actionable—不要把 MD Codes 当填空模板逐码套，先核对该求美者的真实解剖（血管走行/骨量/皮肤厚度）。
  - **一次过量/不可逆设计**：actionable—不要一次打满全脸，分次留量观察，尤其再生类（不可溶解，过量难修）。

### 3. 产品/术式选型与知情同意

- **One-liner**: 从「方案方向已定」到「选定具体材料/剂量/层次 + 逐项风险书面告知并签署知情同意 + 术前留照」——把医疗+法律双重责任落地。
- **Trigger**: 方案确定，需落到具体产品/术式并取得合法授权再执行。 evidence: [T03-S001, T03-S003]
- **Output**: 选定的正品械字号产品 + 剂量/层次方案、逐项知情同意书签署、术前标准照留存。
- **入门 SOP（minimum viable steps）**:
  1. **产品合规验证**：所选填充/肉毒/再生/光电查 NMPA「国械注准」三类证 + 现场拆封正品（热玛吉等手具现场拆封验真）；非「认贵」而是「认证」。 evidence: [T03-S003]
     - 跳过后果：用水货/假货/无证机=三非/非法行医，正品率业内估计市面曾仅约 33%（艾瑞口径，caveat）。
  2. **材料/剂量/层次确定**：按机制选材（玻尿酸可逆 vs 再生类不可逆）、定剂量（肉毒按 U、玻尿酸按支 ml）、定层次（皮内/真皮/皮下/骨膜上/SMAS）。 evidence: [T03-S007]
     - 跳过后果：剂量/层次错=丁达尔（过浅）、栓塞（误入血管）、表情僵硬（肉毒过量）。
  3. **逐项风险告知**：把血管栓塞/栓塞致盲/坏死/感染/结节肉芽肿/移位/丁达尔、效果不达预期、维持时长、是否可逆、超说明书用药（如肉毒瘦肩/颈部属 off-label）逐项讲清。 evidence: [T03-S007, T03-S009]
     - 跳过后果：未充分告知=知情同意无效，出事举证责任倒向机构。
  4. **签署知情同意书**：求美者签字留档（医疗+法律双重留痕），off-label 须特别声明并签字。 evidence: [T03-S001, T03-S007]
     - 跳过后果：违《医疗美容服务管理办法》，纠纷无法举证。
  5. **术前标准照留存**：多角度标准位留照入病历，作为效果对比与举证依据（注意：对比图不得用于广告宣传）。 evidence: [T03-S001, T03-S002]
     - 跳过后果：术后争议无基线，无法客观评估效果。
- **资深路径（差异点）**：
  - **skip**：资深医生**跳过**「进口 vs 国产/哪个大牌」的反复纠结——只要械字号正品 + 适配该解剖即可（反「产品论」：产品是地板、技术是天花板）；不被求美者「认大牌」执念带偏。
  - **optimize**：把「可逆性」作为选材安全权重——高危区/不确定时优先玻尿酸（出事可用透明质酸酶溶解），再生类/脂肪不可逆故更谨慎；优化剂量为「宁少勿多、留量补打」。 evidence: [T03-S004]
  - **add**：资深医生**额外做**真实而非走过场的知情同意——把最坏情况（栓塞致盲/坏死）明确说出口，并评估求美者是否真正理解；off-label 用药特别告知。初学者/营销驱动机构把知情同意当「签字走流程」。 evidence: [T03-S007]
- **近期变化（近 12 个月）**：
  - 2024–2025：**再生类新品密集获 NMPA 三类证（截至 2025 约 7 款童颜针 PLLA + 3 款少女针 PCL，2024–2025.7 共 43 款三类注射械获批）**——选型清单与知情同意话术须随之更新（再生类不可逆性、起效慢 1.5–2 年须特别告知，求美者常误以为即刻）。触发事件类型：新品/产品获批。 evidence: [T03-S015]
  - 2025：**「线上备案 + 线下核查」+ 药品溯源公示趋严**——机构选型环节须可出示械字号溯源，正品验真平台/扫码溯源成合规标配。触发事件类型：监管/平台政策。 evidence: [T03-S012, T03-S013]
- **典型耗时**：入门 SOP 15–30 分钟；资深路径 10–20 分钟（验证+告知更熟练）。
- **关键工具**：NMPA 器械查询/UDI（必备）、正品验真/原厂扫码（必备）、电子知情同意+病历系统（必备）。
- **关键人物**：吴溯帆（可逆性=玻尿酸安全优势）、注射学组共识（off-label 须签知情同意）。
- **常见失败模式**：
  - **认贵不认证**：贵≠正品，水货/假货致命。avoid：所有产品先跑 NMPA 国械注准查询 + 现场拆封。
  - **知情同意走过场**：actionable—不要让求美者在没听懂最坏后果（栓塞致盲）的情况下签字；off-label 不单独告知=高危。
  - **再生类按玻尿酸预期卖**：actionable—不要对再生针说「即刻见效/能溶就不怕」，PLLA/PCL 不可被透明质酸酶逆转，须书面告知不可逆+缓释。

### 4. 注射操作执行（安全核心）

- **One-liner**: 从「知情同意完成、准备进针」到「在正确层次、用安全剂量、避开危险区、把材料精确放到目标位置」的注射结果——这一行最高 tacit、安全红线最硬的环节。
- **Trigger**: 方案与同意书就绪，进入操作台。 evidence: [T03-S006]
- **Output**: 安全到位的注射结果（无血管内注射、层次正确、剂量可控）。
- **入门 SOP（minimum viable steps）**:
  1. **无菌准备**：消毒、铺巾、器械无菌；急救设备就位（**机构须现场常备透明质酸酶 + 肾上腺素 + 氧气**）。 evidence: [T03-S007]
     - 跳过后果：感染/结节肉芽肿；无溶解酶=栓塞无法急救=致盲风险陡增。
  2. **危险区识别 + 层次确认**：按面部危险区解剖（眉间/前额、鼻、鼻唇、颞、眶下、唇、颏与眼动脉系统吻合）标记避险，确认目标注射层次（皮内/真皮/皮下/骨膜上）。 evidence: [T03-S006]
     - 跳过后果：危险区误入血管=栓塞致盲/坏死/脑栓塞；层次错=丁达尔/团块。
  3. **避险手法三件套**：危险区优选钝针、推药前回抽确认未入血管、缓慢推注 + 单点限量（宁少勿多、宁慢勿快）。 evidence: [T03-S004, T03-S006, T03-S007]
     - 跳过后果：「不回抽」是黑医美/外行铁证；快推/大团注=栓塞与团块高危。
  4. **剂量分配 + 边注边评**：按方案分点给量，边注射边观察皮肤颜色/求美者疼痛（剧痛/苍白=栓塞预警）。 evidence: [T03-S008]
     - 跳过后果：一次推完不观察=错过栓塞早期窗口。
  5. **肉毒点位与剂量**（如涉及）：按肌肉解剖定点位、按 U 控量，避免扩散到非目标肌（如上睑下垂、面具脸）。 evidence: [T03-S009]
     - 跳过后果：剂量/点位错=表情僵硬、上睑下垂、扩散并发症。
- **资深路径（差异点）**：
  - **skip**：资深医生在**确知安全的浅层、远离血管区**会**简化部分常规回抽动作**，靠层次手感与解剖确定性判断——但**危险三角/鼻/颞/泪沟等高危区绝不跳回抽与缓推**（经验=「知道哪里能省哪里一步都不能省」）。 evidence: [T03-S006]
  - **optimize**：钝针的层次空间手感与「触到骨膜/穿过隔膜」的阻力反馈——把注射变成「手感导航」而非盲推；剂量上更敢留量分次。
  - **add**：资深医生在**高危区/修复/鼻部**会**额外上高频超声+多普勒做术前血管 mapping + 实时引导**（一项 503 例研究用血管 mapping 实现零血管内注射/压迫）；初学者不具备此能力是高危区出事主因。 evidence: [T03-S010]
- **近期变化（近 12 个月）**：近 12 个月**无重大避险原则变化（OS 核心层稳态）**——无菌/危险区/回抽/钝针/缓推/急救设备常备的安全核心 5-10 年稳定，最近一次方法学进展是超声引导注射的逐步推广（2021 起至今，仍属高阶非普及）。触发事件类型：无（稳态）；边际新工具=超声引导。 evidence: [T03-S006, T03-S010]
- **典型耗时**：入门 SOP 单部位 10–30 分钟；资深路径相近但更安全（高危区上超声反而更慢但更稳）。
- **关键工具**：钝针/锐针 + 回抽（必备）、透明质酸酶/肾上腺素/氧气急救包（必备）、高频超声+多普勒（高阶避险）。
- **关键人物**：吴溯帆（精细解剖/层次决定安全）、注射学组（宁浅勿深+急救设备配备）、de Maio（每个 code 标注须避开的危险结构）。
- **常见失败模式**：
  - **危险区不回抽/快推大团注**：栓塞头号成因。actionable—危险三角/鼻/颞区一律钝针+回抽+缓慢推注+单点限量，绝不图快。
  - **机构不备溶解酶/不会急救**：actionable—进操作台前确认透明质酸酶/肾上腺素/氧气在手边；不备=不应开展高危注射。
  - **高危区不上超声硬打**：actionable—鼻/修复/颞等高危区，没有超声 mapping 能力就转诊有能力的医生，不要赌。

  - [verified_primary] FDA 填充栓塞致盲警示: T03-S008

### 5. 并发症识别与急救（不可软化）

- **One-liner**: 从「注射中/后出现栓塞、坏死、中毒、感染、结节征象」到「按共识/指南即刻处理、溶解、转诊，最大限度保住组织或视力」——医美最凶险、绝不能软化为「零风险」的 workflow。
- **Trigger**: 注射中剧痛/皮肤苍白发白发灰、注射后视力骤降/视物模糊/眼痛、皮肤花斑/发紫坏死、肉毒后吞咽困难/呼吸异常、迟发结节肉芽肿、丁达尔青蓝色调。 evidence: [T03-S004, T03-S008, T03-S009, T03-S017]
- **Output**: 抢救/溶解/转诊后保住组织或视力，或在不可逆时正确转诊修复。
- **入门 SOP（minimum viable steps）**:
  1. **血管栓塞即刻处理**：怀疑栓塞立即**停针**；**24h 内（越早越好、把握 ~72h 组织耐缺血窗）大剂量透明质酸酶溶解**（栓塞处及波及远端，超声引导下交联 HA 栓子平均 35–60U、少数达 150U；可重复多次注射）；辅以热敷/硝酸甘油软膏改善循环。 evidence: [T03-S004, T03-S007, T03-S017]
     - 跳过后果：超 24h 溶解几近无效；不处理=组织坏死/永久瘢痕。
  2. **栓塞致盲急救**：出现视力障碍立即按致盲急救流程——眼科急会诊/转诊、球后或栓塞血管直接注射透明质酸酶、糖皮质激素、降眼压、高压氧抢救存活视网膜组织（**致盲多不可逆，强调争分夺秒**）。 evidence: [T03-S017, T03-S008]
     - 跳过后果：失明窗口极短，延误=永久失明。
  3. **再生类/脂肪栓塞警示**：透明质酸酶**只对玻尿酸有效**，对 PLLA/PCL/自体脂肪栓塞**无法溶解**——这是再生类/脂肪安全性逊于玻尿酸的关键，处理以血管再通/支持治疗为主，预后更差。 evidence: [T03-S004]
     - 跳过后果（认知）：误以为「再生针也能溶」=延误且无对应解药。
  4. **肉毒扩散/中毒识别**：上睑下垂、面具脸、吞咽/呼吸困难（严重需急诊），多为剂量/点位致扩散，部分可待代谢恢复，重者支持治疗+急诊。 evidence: [T03-S009]
     - 跳过后果：呼吸/吞咽受累延误=危及生命。
  5. **感染/结节/肉芽肿/丁达尔处理**：感染按抗感染；结节区分可揉散/溶解 vs 肉芽肿（免疫性，激素/手术，难处理）；丁达尔（过浅）用透明质酸酶溶解纠正。 evidence: [T03-S009]
     - 跳过后果：迟发肉芽肿误当普通结节硬挤=加重。
  6. **转诊与修复**：超出本机构/本人能力（致盲、大面积坏死、复杂修复）立即转诊上级/有能力机构，不硬扛。
     - 跳过后果：黑医美「不会急救+不转诊」是致残致死主因。
- **资深路径（差异点）**：
  - **skip**：急救流程**无可省步骤**——这是本文件唯一明确「不存在 skip」的 workflow（任何省略都可能致盲/致死）。资深差异在「更快识别 + 更敢果断动手」而非省步骤。
  - **optimize**：资深医生**识别更快**——注射当下凭皮肤颜色/疼痛性质/灌注变化秒级判断栓塞，争取黄金时间；溶解酶用量与路径判断更精准（必要时大剂量、沿血管走行布点）。 evidence: [T03-S004]
  - **add**：资深医生**额外做**球后注射等高阶致盲急救 + 与眼科/急诊的转诊绿色通道预案 + 高压氧资源对接；并在术前就备好栓塞急救预案（不是出事才找溶解酶）。 evidence: [T03-S017]
- **近期变化（近 12 个月）**：近 12 个月**无重大急救方案变化（OS 核心层稳态）**——透明质酸酶时间窗/剂量、致盲急救流程基于持续累积的专家共识与文献，稳定演进无颠覆。强调：随再生类用量上升，「再生类栓塞无解药」的风险敞口在扩大（这是产品结构变化带来的安全隐患，非急救方案变化）。触发事件类型：无（稳态）；风险敞口随再生类放量上升。 evidence: [T03-S004, T03-S009]
- **典型耗时**：栓塞急救分秒必争（黄金窗口 24h 内/致盲数小时内）；结节/肉芽肿为周–月级随访处理。
- **关键工具**：透明质酸酶（栓塞急救命门，仅对 HA 有效）、肾上腺素/氧气/糖皮质激素、高压氧、眼科会诊通道、高频超声（定位栓子）。
- **关键人物**：注射学组共识（急救设备配备）、吴溯帆（可逆性/急救）、FDA（栓塞致盲警示）。
- **常见失败模式**：
  - **错过 24h 溶解窗口**：actionable—注射后任何剧痛/苍白/视力异常立即按栓塞处理，不要「先观察看看」。
  - **把再生类/脂肪栓塞当玻尿酸溶解**：actionable—对非 HA 栓塞别等溶解酶，直接走血管再通/支持/转诊。
  - **不转诊硬扛致盲/大面积坏死**：actionable—超能力范围立即转上级 + 眼科/急诊，黑医美最致命的是不会救也不转。
  - **迟发肉芽肿当普通结节挤压**：加重免疫反应，须区分处理。

### 6. 术后医嘱与随访

- **One-liner**: 从「操作结束、求美者离院」到「护理依从 + 按期复查 + 早期并发症监测 + 效果维持/补打 + 留客复购」的闭环。
- **Trigger**: 注射/术式操作完成。 evidence: [T03-S001]
- **Output**: 术后护理依从、复查记录、并发症早期拦截、效果维持/补打安排。
- **入门 SOP（minimum viable steps）**:
  1. **即刻术后处置**：冷敷/压迫止血、观察 15–30 分钟（即刻过敏/血管事件窗）再离院。 evidence: [T03-S009]
     - 跳过后果：错过即刻并发症（栓塞早期/过敏）观察窗。
  2. **术后医嘱**：护理要点（清洁/防晒/忌烟酒辛辣/特定项目忌按压/防感染）、肿胀淤青预期、何时复查、出现哪些征象立即返院（剧痛/苍白/视力异常/发热）。 evidence: [T03-S009]
     - 跳过后果：求美者不知危险信号=迟报栓塞/感染，错过黄金处理窗。
  3. **复查与补打**：按项目周期复查（玻尿酸 2–4 周看吸收补量、肉毒 2 周看效果调整、再生类按月看胶原起效），评估效果与并发症。
     - 跳过后果：再生类未到起效期被求美者误判「无效」要求加打=过量风险。
  4. **病历与随访记录**：复查/补打/不良反应入病历，维护求美者档案。 evidence: [T03-S001]
     - 跳过后果：随访断档，效果争议无记录。
- **资深路径（差异点）**：
  - **skip**：资深机构**跳过**对低风险项目（如基础水光/IPL）的过度密集随访，靠标准化医嘱 + 求美者自助即可；但**高危注射后的主动随访不跳**。
  - **optimize**：把随访从「群发模板提醒」**优化为风险分层主动随访**——高危区注射/再生类/首次客户在 24–72h 主动电话/复诊监测早期并发症与预期管理。
  - **add**：资深医生**额外做**早期并发症窗口的主动监测（迟发结节/感染/血管事件的随访问诊）+ 把随访做成医疗关怀而非纯复购话术，反向沉淀口碑。
- **近期变化（近 12 个月）**：
  - 2024–2025：**随访/复购话术须合规化**——广告执法指南禁制造容貌焦虑，私域/SCRM 随访不得以焦虑驱动补单升单；病历与随访记录在「线下核查」中被重点抽查（病历规范、消毒、医疗废物）。触发事件类型：监管/平台政策。 evidence: [T03-S002, T03-S012, T03-S013]
- **典型耗时**：术后即刻观察 15–30 分钟；随访周期按项目跨周–月。
- **关键工具**：HIS/电子病历（随访记录，必备）、私域 SCRM/CRM（复购运营，须合规）、VISIA（疗效前后对比，不得用于广告）。
- **关键人物**：注射学组（术后观察/急救衔接）、何黎（理性消费、改善而非过度）。
- **常见失败模式**：
  - **离院前不留观/不交代危险信号**：求美者迟报栓塞。actionable—术后留观 15–30 分钟 + 明确告知「剧痛/苍白/视力异常立即返院」。
  - **再生类未到起效期被催加打**：actionable—术前+术后双重告知缓释起效曲线，避免「无效」误判致过量。
  - **随访沦为焦虑式升单**：踩广告合规红线。avoid：随访以医疗关怀+并发症监测为主，复购自然发生。
- **来源**：
  - [verified_primary] 管理办法（病历/随访留痕）: T03-S001
  - [verified_primary] 广告执法指南（随访话术合规）: T03-S002
  - [verified_primary] 不良反应综述（术后并发症窗口）: T03-S009
  - [verified_primary] 2025 线下核查/病历规范: T03-S012
- **Last_updated**: 2026-06-21
- **Decay risk**: high（医疗医嘱稳定，但随访/复购的合规约束与核查要求 12 月内活跃变化）

### 7. 机构合规获客 → 面诊转化 → 复购（机构经营链）

- **One-liner**: 从「机构需要持续获客并合规活下去」到「合规获客 → 咨询面诊转化 → 履约复购」的经营闭环——本行「医疗本质 ⇄ 消费营销」张力最集中的 workflow。
- **Trigger**: 机构日常经营，需稳定客流与营收。 evidence: [T03-S002, T03-S015]
- **Output**: 合规获客 + 面诊转化 + 复购/转介绍，且零监管处罚。
- **入门 SOP（minimum viable steps）**:
  1. **获客渠道布局**：线上垂直平台（新氧/美团医美/更美）+ 内容种草（抖音/小红书）+ 私域/转介绍，多渠道组合。 evidence: [T03-S015]
     - 跳过后果：单一渠道依赖（如纯平台低价引流）=获客成本高、利润被平台/返佣吃掉。
  2. **内容合规生产**：科普/案例内容须合规——不得制造容貌焦虑、不得用诊疗前后对比图、不得疗效保证、不得以科普/新闻形式变相发布广告、不得把无资质者包装成「医生/专家」。 evidence: [T03-S002]
     - 跳过后果：违广告执法指南，平台清理/封号 + 市监处罚（已有机构被罚百万级，caveat）。
  3. **咨询面诊转化**：线上到店后由医生主导面诊（见 workflow 1），把诉求转成有适应证的方案——医疗诊断框架而非纯销售话术。 evidence: [T03-S001]
     - 跳过后果：纯销售导向转化=过度治疗/投诉/口碑反噬。
  4. **履约与复购**：规范履约（合规操作+术后随访）沉淀信任，自然带动复购/转介绍/客单提升。 evidence: [T03-S001]
     - 跳过后果：靠焦虑升单的短期复购=口碑崩塌+投诉。
- **资深路径（差异点）**：
  - **skip**：成熟机构**跳过/降低**对纯低价体验价引流的依赖——低价钓鱼客忠诚度低、被平台/返佣吃利润；转向口碑+私域复购。
  - **optimize**：把经营重心**优化为私域复购与转介绍**——老客复购成本远低于新客获取，靠医疗信任而非营销话术留客（金星「回归医疗本质仍有高概率出线」）。 evidence: [T03-S015]
  - **add**：资深机构**额外做**合规风控前置（见 workflow 8）——把广告审查、资质核验、病历留痕做在获客之前，避免一边获客一边踩雷；并把「劝退」作为长期口碑投资（敢劝退的机构反而赢信任）。
- **近期变化（近 12 个月）**：
  - 2024–2025：**平台医美内容监管空前收紧**——小红书取消私立医美机构专业认证、仅认公立三甲/主治及以上医生认证、清理违规笔记 134 万+；抖音封禁违规直播间 10.6 万+、打击无资质商家医疗信息；监管常态化下 2025 已关停违规机构超 1.8 万家（监管/平台口径，须 caveat）。获客工作流被迫从「软文种草+对比图」改为「合规科普+真实资质」。触发事件类型：法规/平台政策。 evidence: [T03-S002, T03-S015]
  - 2025：**AI 赋能获客与转化**——AI 测脸/3D 模拟提面诊转化（机构称效率+60%、纠纷-28%，机构口径 caveat）；但 AI 用于「效果保证」违广告法，仅作沟通工具。触发事件类型：新工具/新模型。 evidence: [T03-S016]
- **典型耗时**：经营链是持续运营（非一次性）；单客线上到店转化周期数天–数周。
- **关键工具**：新氧/美团医美/更美（必备获客）、抖音/小红书种草（须合规）、私域 SCRM/CRM（复购）、AI 测脸/3D（新兴转化工具）。
- **关键人物**：金星（回归医疗本质 vs 流量生意）、申东日（连锁整合）、伪科普网红现象（反面：批量制造网红医生/暗广导私域）。
- **常见失败模式**：
  - **对比图/焦虑式种草**：actionable—不要发诊疗前后对比图或「五分钟变美/一针逆龄」文案，直接违广告执法指南，平台封号+罚款。
  - **包装无资质者为「专家」**：actionable—只让有主诊备案的医生出镜科普，MCN 批量孵化「网红医生」是监管重点打击对象。
  - **纯低价引流到店强升单**：体验价钓鱼+渠道返佣推高价=口碑反噬+投诉激增。
  - **AI 模拟图当「效果保证」**：违广告法，仅作预期沟通不可承诺。
- **来源**：
  - [verified_primary] 广告执法指南（内容合规红线）: T03-S002
  - [verified_primary] 管理办法（面诊/履约/资质）: T03-S001
  - [secondary] 2025 平台监管/关停数据(caveat): T03-S015
  - [secondary] 央广网卫健委严查乱象: T03-S013
  - [secondary] AI 获客转化(机构口径 caveat): T03-S016
- **Last_updated**: 2026-06-21
- **Decay risk**: high（平台政策+监管+AI 工具是本行衰减最快的工作流，强烈建议每 3 月跑 update）

### 8. 机构合规风控与投诉处理（资质 / 广告 / 价格 / 票据 / 病历）

- **One-liner**: 从「机构上新项目/发广告/处理纠纷」到「资质-广告-票据-病历四道合规闭环 + 投诉化解」——把监管红线变成可执行的日常风控 SOP。
- **Trigger**: 上新项目/招人上岗、发布广告、定价开票、发生投诉或纠纷。 evidence: [T03-S001, T03-S002, T03-S011]
- **Output**: 资质/广告/票据/病历四项合规留痕 + 投诉妥善化解，零监管处罚。
- **入门 SOP（minimum viable steps）**:
  1. **资质合规**：机构持《医疗机构执业许可证》且诊疗科目含「医疗美容科」；医生持《医师执业证书》+ 医美主诊医师备案（美容外科 6 年/牙 5 年/皮肤·中医 3 年临床门槛，每人限一专业）；项目按一至四级分级准入。 evidence: [T03-S001, T03-S011, T03-S012]
     - 跳过后果：无证机构/无备案医生操作/超分级=非法行医，致残致死的三非根源。
  2. **广告合规**：发布前取得《医疗广告审查证明》；内容遵「八不准」+ 不制造容貌焦虑/不疗效保证/不对比图/不软文伪科普。 evidence: [T03-S002]
     - 跳过后果：罚款（已有机构被罚百万，caveat）、平台下架封号。
  3. **价格与票据合规**：明码标价、规范开票，不做虚假折扣/钓鱼低价/医美贷套路诱导。 evidence: [T03-S002, T03-S015]
     - 跳过后果：价格欺诈/不正当竞争处罚、医美贷金融纠纷。
  4. **病历与术前照合规**：完整电子病历 + 知情同意 + 术前标准照 + 随访记录，加密合规存储。 evidence: [T03-S001]
     - 跳过后果：「线下核查」重点抽查项；病历造假/缺失=维权举证全输+违规。
  5. **投诉处理**：建立投诉响应流程，区分「效果不达预期」（沟通/修复/退费协商）vs「医疗并发症/事故」（先医疗处置+按医疗纠纷流程，必要时转诊/上报）。
     - 跳过后果：投诉激化为医闹/媒体事件/监管立案（黑医美投诉 5 年增 14 倍，caveat）。
- **资深路径（差异点）**：
  - **skip**：成熟机构**跳过**「出事后补材料」的被动模式——把合规做成前置常态，不靠事后救火。
  - **optimize**：把病历+术前照+知情同意**优化为操作前置强制项**（不签不照不开做），既是合规也是投诉举证护城河。
  - **add**：资深机构**额外做**广告前置审查 + 定期资质/械字号自查 + 投诉分级预案（并发症走医疗、效果争议走协商），把风控嵌进经营全流程。
- **近期变化（近 12 个月）**：
  - 2024–2025：**「线上备案 + 线下核查」模式落地 + 专项整治常态化**——2025 卫健委持续严查医美乱象，机构须公示药品溯源、接受病历/消毒/医疗废物细化抽查；主诊医师备案滚动收紧。触发事件类型：法规/监管。 evidence: [T03-S012, T03-S013]
  - 2024–2025：**互联网健康科普负面清单 + 平台资质核验**——多地出台负面清单，平台强化医生资质核验，机构「医生 IP 获客」须真资质。触发事件类型：法规/平台政策。 evidence: [T03-S015]
- **典型耗时**：合规是持续性日常；单次广告审查/资质备案为周–月级行政周期；投诉处理按性质数日–数月。
- **关键工具**：卫健委机构/医师执业查询（必备）、NMPA 械字号查询（必备）、医疗广告审查（必备）、HIS/电子病历（必备）、CAPA 三正规平台。
- **关键人物**：管理办法/广告执法指南（gov 原文）、伪科普网红现象（反面：无资质包装/暗广导私域）。
- **常见失败模式**：
  - **无备案医生/超分级操作**：actionable—上岗前核验主诊备案与项目分级，缺一即非法行医。
  - **广告未前置审查/用对比图**：actionable—任何对外内容先过广告合规，对比图/疗效保证/焦虑文案一律不发。
  - **病历术前照缺失**：actionable—操作前强制完成知情同意+术前照，否则纠纷举证必输。
  - **把医疗并发症当效果争议拖延**：actionable—并发症先医疗处置+按纠纷流程，不可当退费协商拖延致恶化。
- **来源**：
  - [verified_primary] 管理办法（资质/病历/分级）: T03-S001
  - [verified_primary] 广告执法指南（广告/价格红线）: T03-S002
  - [verified_primary] 项目分级管理目录: T03-S011
  - [verified_primary] 2025 规范执业/线下核查通知: T03-S012
  - [secondary] 2025 严查乱象/年度事件(caveat): T03-S013, T03-S015

- **Decay risk**: high（监管/专项整治/备案规则滚动更新，强建议每 3 月跑 update）

### 反复出现 ≥3 个 workflow 都包含的步骤（候选 playbook 通则）

- **「合规/资质/械字号验证」**出现于 workflows: 3(选型) / 7(获客转化) / 8(风控) → 候选「如果要落地任何项目/产品/获客，则先验机构资质+主诊备案+NMPA 械字号」playbook（合规是入场券，前置于一切）。 evidence: [T03-S001, T03-S003, T03-S011]
- **「知情同意 + 病历 + 术前照留痕」**出现于 workflows: 3(选型同意) / 6(随访记录) / 8(投诉举证) → 候选「如果做医疗操作，则全程留痕（医疗+法律双重）」playbook。 evidence: [T03-S001]
- **「BDD/容貌焦虑筛查 + 劝退」**出现于 workflows: 1(面诊) / 7(转化合规) / 8(广告合规) → 候选「如果诉求异常强烈/被焦虑驱动，则劝退而非升单」playbook（劝退伦理跨临床与经营两线）。 evidence: [T03-S002]
- **「危险区→回抽/钝针/缓推/宁少勿多 + 备溶解酶」**出现于 workflows: 4(执行) / 5(急救) / 3(选型可逆性) → 已是行业最硬 playbook「先安全后审美」。 evidence: [T03-S004, T03-S006, T03-S007]

### 入门 SOP 和资深路径之间最大的差距（候选心智模型）

- 入门 SOP 平均 5–6 步，资深路径核心差异集中在「**知道哪一步能省、哪一步一步都不能省**」——尤其 workflow 4：资深医生在安全浅层敢简化回抽，但危险三角绝不跳；workflow 5（急救）则明确「无可省步骤」。→ 推断这一行资深人特别擅长**风险分层判断**（在安全处提效、在危险处零妥协）。
- 资深路径反复出现「**劝退 + 留量分次 + 真实知情同意**」三件初学者因「怕丢单/赶产能/走流程」而省略的事。→ 候选心智模型「**不要在面诊处过度迎合、在注射处过度一次到位、在同意处过度走形式**」——医疗克制优先于消费冲动。
- 经营链资深差异普遍是「**从低价引流/焦虑升单 → 医疗信任/私域复购/合规前置**」。→ 候选心智模型「回归医疗本质本身就是最强经营护城河」（与 figures 金星/郭树忠/何黎一致）。

### 近期工作流变化的根本原因（触发事件类型分布）

- **法规/监管驱动**（最强）：4 个 workflow 受改写——面诊(禁容貌焦虑)、选型(溯源公示)、随访(话术合规)、获客+风控(广告执法指南/专项整治/线上备案+线下核查/平台资质核验)。 evidence: [T03-S002, T03-S012, T03-S013, T03-S015]
- **新工具/新模型驱动**：2 个 workflow——面诊与获客转化的 AI 测脸/3D 模拟/大模型问诊（提效但有「审美平均化/效果保证」合规风险）。 evidence: [T03-S016]
- **新品/产品获批驱动**：1 个 workflow——选型与知情同意随再生类密集获批更新（不可逆性须特别告知）。 evidence: [T03-S015]
- **稳态（无重大变化）**：3 个 OS 核心 workflow——审美设计原则、注射避险、并发症急救（low decay，最近进展为超声引导推广）。
- → 主要外部驱动力 = **强监管常态化**（Phase 2 识别为本行第一外部驱动力，影响反模式与经营心智）。

- **诚实边界（喂 Phase 2.8）**：
  1. **临床注射手感（层次空间感/危险区避险手感/急救果断决断）是高 tacit**——共识/期刊只能给原则与剂量窗，真功夫靠规范化培训 + 大量临床实操 + 并发症处理经验，SKILL.md 工作流节给的是「框架与红线」不是「手把手手感」。
  2. **安全/并发症流程已诚实写出最坏后果（栓塞致盲/坏死/肉毒中毒/再生类无解药），未软化为「零风险」**（intake warnings 硬要求）；急救剂量/时间窗均挂专家共识/指南/文献 source。
  3. **经营/平台/营销数字（机构关停 1.8 万+、笔记清理 134 万+、效率+60%、纠纷-28%、正品率~33%、投诉 5 年增 14 倍）全部 secondary + caveat**（监管通报/平台口径/业内估计/机构数据），不可当 universal 事实，须随年份更新。
