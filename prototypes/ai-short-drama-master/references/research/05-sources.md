# Track 05 — Sources (AI 短剧 / AI 微短剧)

> Research date 2026-06-04 | locale=zh-CN（主战场中国 + 出海）| Track 05 = 要持续追踪的「活信息源」。
> **本行业 sources 的衰减结构与众不同**：真正高频变动的不是「newsletter / podcast」（这恰恰是本行业最薄的一层），而是 **(a) 工具官方更新日志**（每 2-3 个月一次大版本）+ **(b) 监管政策**（NRTA/网信办密集发文）+ **(c) 数据榜单**（出海买量/收入日级—月级刷新）。所以本文件把 **官方更新日志 / 政策数据源 / 出海情报榜单** 作为「最该常追」的三类，行业媒体长稿 / 创作者社区 / 赛事作为补充。
> **诚实前置**：zh-CN 的「专门垂直媒体层」很薄且**主要发声在公众号**（娱乐资本论 / 三声 / 犀牛娱乐 / 刺猬公社 都是公众号优先，自有官网站点弱或检索不到稳定入口）。按 Track 05 规范，**公众号「频道身份」可作为 source 列表条目，但其文章不可作知识真伪来源**（Track 06 黑名单）；本文件对这类一律标注，并优先收录有自有可访问域名的同类（数英 / 36氪 / 钛媒体）。
> **时效**：所有条目 last_checked = 2026-06-04。官方更新日志类 >2 个月需重刷；政策类 >1 个月需扫一遍 NRTA 公告页；出海榜单类滚动失效（每条都标注）。
