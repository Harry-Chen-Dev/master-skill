## 1. 心智模型（6 个）

### 1.1 terroir-first（地理是第一性问题,旧世界尤甚）

- **跨人物背书** (figures: Hugh Johnson / Jancis Robinson / Julia Harding / Benjamin Lewin)——地图派、事实层双核、数据怀疑派(用数据检视 terroir)同把「地」放在第一位,即便 Lewin 是来质疑的也绕不开它。
- **一句话**:遇到一支酒,资深人的**第一个问题不是「什么品种 / 好不好喝」,而是「它来自哪——那块地的气 / 土 / 人为传统如何」**;terroir(风土)是不可复制、不可移植的组织原语,旧世界直接「以产地为酒名」。
- **应用方式**:拿到任意陌生酒先做地理定位「国 → 产区 → 子产区 → 地块(climat)」,再谈品种 / 风格 / 价格;旧世界标签读「产区」(Burgundy=Pinot Noir 的线索藏在产区里),新世界才先读品种。
- **局限**:terroir 有被窄化成「土壤」或反过来被当**不可证伪的营销玄学**的双向风险(Lewin 数据派要求实证、minerality 机理学界有争议);新世界以品种 / 技术为先,terroir-first 不总适用——是「先问哪里」的强默认,不是放之四海的铁律。
- **evidence**: [T04-S013, T04-S018, T04-S029, T01-S015]

### 1.2 codifiable 理论层 ⇄ tacit 校准层（双层不可互替）

- **跨人物背书** (figures: Ann Noble / Rajat Parr / Émile Peynaud / Jamie Goode)——Noble 把 tacit 嗅觉做成可教词表、Parr 把一线服务 tacit 写成书、Goode 论证品鉴是大脑建构,四人从不同端印证「知识可读、手艺须练」。
- **一句话**:葡萄酒认知是**两层**——可背诵的理论(产区 / 法规 / 品种 / 酿造化学)和不可言传的校准(味觉记忆 / 盲品 / 侍酒服务);两层**不可互相替代**,读一百本书也代替不了真酒重复。
- **应用方式**:任何能力诊断先分层——理论缺口靠 canon + 认证教材 + 抽认卡补;tacit 缺口**只能靠真酒 + 重复 + Aroma Wheel 词汇校准 + mentor**,没有 app / AI 捷径。回答「怎么学这行」时把两层的训练方式分开开方。
- **局限**:理论层与 tacit 层各有失效边界——纯 tacit 不 codify(不走网格 / 不写刻度词)就无法传承、无法被他人核对、易自欺;但 tacit 层确实无工具捷径(arXiv SommBench 硬证据:LLM 理论题约 97%,sensory / 配餐 MCC 仅 0-0.39,evidence: [T03-S023]),AI 帮不了感官校准。
- **evidence**: [T04-S012, T04-S025, T01-S026, T03-S023]

### 1.3 盲品 = 结构化演绎：结构(骨架)先于香气(血肉)

- **跨人物背书** (figures: Jancis Robinson / WSET SAT / CMS deductive grid)——Jancis「先抓 structure 再说香气」与 WSET-SAT、CMS-grid 共享 sight→nose→palate 架构,把「这酒尝起来对不对劲」的本能逼成可辩护网格。
- **一句话**:盲品不是天赋直觉,是**把感官线索逼成可走查、可被他人核对的演绎**;且**先抓 structure(酸 / 单宁 / 酒体 / 酒精)这副骨架做推断,香气只作佐证**——因为结构比香气更难伪装、更指向气候 / 产地。
- **应用方式**:任何品鉴 / 判断都按固定顺序逐项给**校准刻度词**(low / medium- / medium / medium+ / high),用**排除法**(「若是 X 产区,酸不该这么高 → 排除」)而非正向猜;估**酒龄**(看演化程度)而非赌具体年份。
- **局限**:暖化使「高酒精 + 浓果 = 新世界」这类经典初判**置信度下降**(旧世界凉产区近年酒精上升,Winkler index 已被重订);极端 / 边缘酒演绎失灵需更多线索;**SAT(描述 + 评质,不必猜中)与 CMS-grid(逻辑演绎到具体身份,猜中是目标)目标不同**,选错方法验收口径就错。
- **evidence**: [T03-S004, T03-S001, T03-S002, T04-S007]

### 1.4 每个决策都站在一条张力轴上（干预⇄放手 / 描述⇄演绎 / 通约⇄风土 / 一致⇄互补）

- **跨人物背书** (figures: Émile Peynaud / Isabelle Legeron / Robert Parker / Rajat Parr)——Peynaud(清洁科学干预)对 Legeron / Joly(不加不减放手)、Parker(浓郁通约)对 Parr / IPOB(平衡克制),这行的真问题都是轴上的对立而非对错。
- **一句话**:这行的资深人面对决策的本能是**先认清它站在哪条张力轴上、对立面是什么**——酿造(干预 vs 放手)、盲品(描述评质 vs 演绎识别)、评分(量化通约 vs 风土表达)、配餐(congruent 一致 vs complementary 互补)、服务(仪式 vs 高效)。
- **应用方式**:遇到一个争议(「自然酒好不好」「该不该醒」「百分制可不可信」)先把它落到对应张力轴上、说清两端各自的正当性与代价,而非给「唯一正确答案」;判断自己 / 对方站哪端、利益位置在哪。
- **局限**:张力轴是**定位工具不是答案**,把任一端道德化就会变成新教条(IPOB「平衡」被批把「低酒精 = 更高级」道德化、约 2016 解散;自然酒把「不加不减」绝对化则容忍缺陷酒);站轴前须先认清自己的利益与口味位置。
- **evidence**: [T01-S021, T01-S041, T01-S034, T04-S028]

### 1.5 分数是「酒 × 品鉴者」的互动属性,不是酒的固有属性

- **跨人物背书** (figures: Jamie Goode / Jancis Robinson / Robert Parker / Antonio Galloni)——Goode「分数不是酒的属性而是酒与品鉴者互动的属性」、Jancis 20 分制散文派、Parker 100 分制原点、Galloni 平台化继承,四人界定了「评分到底是什么」之争。
- **一句话**:葡萄酒分数是**单个评论家在特定时点、带特定风格偏好打出的主观刻度**,不可跨评论家通约;判一支酒先判「谁打的 + 什么偏好 + 利益位置」,**永远不看裸分 / 聚合分**。
- **应用方式**:用评论家长稿(Jancis Purple Pages / Vinous / Tim Atkin)而非 Wine-Searcher 聚合裸分判品质;明确 100 分制系统性偏爱浓郁 fruit bomb、惩罚克制风格;对客人把分数翻译成「这支适不适合你的口味 / 场合」而非「客观最好」。
- **局限**:分数仍是 en primeur / 横向品鉴的有用**消费速记 shorthand**(连抵制它的 Jancis 也被迫打分);拒绝裸分不等于拒绝品质判断,只是拒绝把主观刻度当客观真理;单一评论家给整个产区分级(如 Tim Atkin 南非)有**权力集中**风险,数字须挂「单一评论家口径」caveat。
- **evidence**: [T01-S002, T01-S001, T06-S032, T04-S038]

### 1.6 旧世界(以产地为名) vs 新世界(以品种为名)：先判这支酒按哪套世界观组织

- **跨人物背书** (figures: Maynard Amerine / Albert Winkler / Hugh Johnson / José Vouillamoz)——UC Davis 气候区 / 品种科学(新世界技术派)对 Johnson 地图(旧世界风土派),Vouillamoz 用 DNA 解释品种-产地迁徙、又祛魅旧世界起源神话。
- **一句话**:全球葡萄酒被两套世界观组织——**旧世界以产地 / 法定产区为名(terroir 至上,AOC / DOCG / VDP 按地块)**,**新世界以品种 + 技术 + 气候数据为名(AVA 只圈地不管品种)**;读一支酒先判它说哪套语言,才能预测风格、读懂标签。
- **应用方式**:看标签先判世界观——旧世界标签查「产区→地块法规(cahier des charges)」推品种与风格上限;新世界标签读「品种 + AVA + 酿造选择」;两套分级哲学(波尔多 1855 按庄 / 勃艮第 climat 按地 / 德国 Prädikat 按糖→VDP 按地)各是一套「什么决定质量」的世界观。
- **局限**:二分是**简化**——新世界也做 site-specific terroir 表达、旧世界也用现代技术;**气候变暖 + 品种迁徙正在模糊两者边界**(凉产区变暖、新品种北上),标签逻辑与适种品种须随地理变化持续更新,不能用静态地理刻板印象死套。
- **evidence**: [T06-S015, T06-S004, T01-S025, T01-S019]

## 2. 标准 Playbook（8 条）

1. **如果遇到一支陌生酒要判断**, 则先做地理定位(国 → 产区 → 子产区 → 地块)再谈品种 / 风格;**旧世界读产区、新世界可先读品种**。案例(case):「我喜欢勃艮第那个葡萄」是外行破绽——Burgundy 是产区不是品种,红勃艮第 = Pinot Noir,产区本身就是品种与风格的线索 (evidence: [T06-S028, T04-S029])。

2. **如果要盲品 / 演绎一支酒**, 则先抓 structure(酸 / 单宁 / 酒体 / 酒精)这副骨架、用刻度词逐项算出来,**香气只作佐证**,并用排除法而非正向猜。案例(case):高酸 + 轻酒体 + 无橡木 + 砖红边 → 凉气候 / 旧世界;高酒精 + 浓果 + 明显橡木 → 暖气候 / 新世界(CMS 初判逻辑的可操作化) (evidence: [T03-S004, T04-S007])。

3. **如果要选盲品方法**, 则**考 WSET / 写可辩护书面结论用 SAT(描述 + 评质,不必猜中)**;**考 CMS / 餐厅一线识酒用 deductive grid(逻辑演绎到品种 / 产地 / 年份,猜中是目标)**。案例(case):两套共享 sight→nose→palate 架构但结论阶段分道,选错方法验收口径就错——这是 WSET 知识派 vs CMS 服务派认识论之争的硬件层 (evidence: [T03-S001, T03-S002, T03-S003])。

4. **如果要判品质 / 解读分数**, 则看「谁打分 + 什么风格偏好 + 利益位置」,**绝不看裸分 / 聚合分**;判信源先判其 bucket(一手 / 二手 / 众包)。案例(case):Wine-Searcher 聚合分摊平了 critic 个体偏好;100 分制系统性偏爱 fruit bomb;Vivino 均分受甜度 / 价格 / 品牌污染,只看大众接受度不作专业品质依据 (evidence: [T02-S005, T06-S032])。

5. **如果要给一道菜配酒**, 则先定菜的主导维度(脂 / 酸 / 甜 / 辣 / 蛋白)再套交互原则(酸配脂、单宁配蛋白、地配地、甜配甜),**配「菜的整体」(酱汁 / 烹法)而非单一蛋白**。案例(case):同一块鱼黄油煎 vs 清蒸配酒完全不同,酱汁常比主料更决定配酒;经典反直觉如 Sauternes 配 Roquefort 蓝纹(甜咸对冲) (evidence: [T03-S010, T03-S011])。

6. **如果要决定是否醒酒 / 怎么服务**, 则**不是所有酒都醒**——只老酒去沉淀 / 年轻封闭的高单宁酒增氧,且拔塞「透气」瓶颈面积太小几乎不增氧、要真醒必须倒进醒酒器。案例(case):「这支必须醒满 X 小时」「所有酒都要醒」是被资深拒绝的话术,醒多久高度个案、多数酒开瓶即饮 (evidence: [T03-S007, T06-S035])。

7. **如果遇到厂商话术(95 分 = 客观最好 / 自然酒不上头 / sulfites 致头疼 / 投资酒稳赚 / 越老越好)**, 则默认是被资深拒绝的错位包装、回到机理。案例(case):红酒头疼真因更可能是 quercetin 抑制 ALDH 而非 sulfites(白酒含等量 sulfites 却少致头疼);「投资酒稳赚」是诈骗高发话术(伪造产地 / 超收 / 假瓶) (evidence: [T06-S006, T06-S033])。

8. **如果考虑用 AI / 工具**, 则**查询 / 推荐 / 运营 / 生产侧 AI 有真用,但盲品 / 桌边判断 / 反直觉配餐是 sensory grounding 死穴、不能外包**。案例(case):arXiv SommBench——LLM 理论约 97% 但配餐 MCC 仅 0-0.39;Cru Uncorked 实测 AI vs somm 结果各半、冷门品种 AI 露怯 (evidence: [T03-S023, T02-S020])。

## 3. 工具栈与选型决策树

> 直接 reference [`references/research/02-tools.md`](research/02-tools.md) 三层结构 + 按场景决策树 + 6 条避坑清单。本节为一致性 sanity-check 摘要。

工具栈三层(全部达量门槛):**必备 5 个** / **场景特化 6 个** / **新兴 4 个**。`last_checked` 全部 2026-06-06。**本行「必备」的反直觉:最核心的工具不是软件,而是「方法论框架」**——它们是把 tacit 味觉转成 codifiable 书面结论的脚手架。

- **必备层 5 个**(覆盖约 80% 认真从业 / 备考者,**Decay risk: low**):**WSET SAT 网格**(把品鉴拆成 appearance / nose / palate / 结论的可写网格)、**CMS Deductive Tasting Grid**(从线索演绎到国 / 产区 / 品种 / 年份的盲品引擎)、**Wine Aroma Wheel**(Ann Noble 1984,香气标准化词表,官方 8 语含中文)、**Wine-Searcher**(全球比价 + 均价 + 评分聚合的事实标准查价工具)、**GuildSomm compendium + study wiki**(CMS / MW 备考事实标准学习底座,约 $149/yr 全会员、厂商自报价 caveat)。
- **场景特化层 6 个**:**CellarTracker**(众包笔记 + 酒窖 / 适饮窗管理,价值在筛 favorite tasters 过滤噪音)、**Coravin**(针穿木塞 + argon 取酒、不开瓶,BTG 高价 / 稀有酒,剩余约 4 周 vendor 自报)、**餐厅酒单 / BTG SaaS(BinWise / InVintory)**(库存 + 50+ POS 集成 + 数字酒单)、**Liv-ex**(精品酒二级市场真实成交价指数,投资 / 交易侧最高信号)、**侍酒物理工具套装**(侍酒刀 / 醒酒器 / 侍酒温度计 / 充氮塞,高端一手厂偏法国 fr-primary)、**备考工具**(抽认卡 Anki / Quizlet / Brainscape spaced repetition + 产区盲图默画)。
- **新兴层 4 个**(**Decay risk: high**,12 月内剧变):**AI 配餐 / virtual sommelier(Preferabli / Sippd)**(有限真采用、assists not replaces)、**AI 驱动栽培 / 酿造 / 酒旅(生产侧)**(真用例但非侍酒侧)、**数字 / QR 酒单**(疫情后 contactless 有真采用,vendor docs 为主)、**区块链 / NFC 防伪(eProvenance / dVIN)**(侍酒师层面营销 > 采用、未验证)。

选型第一性分叉:**目的(学习 / 查价 / 盲品训练 / 酒窖 / 投资 / 服务)× 体系(WSET 知识 vs CMS 服务)× 平台(Mac / 禁 HUD 牌房无关、但中文用户须直接用英文工具)**。核心避坑:① 别把 Vivino 均分当专业品质权威(最低信号端,受甜度 / 价格 / 品牌污染);② 别用扫标签 App / AI 替代盲品训练(扫标签 = 反盲品,AI sensory 是死穴 evidence: [T02-S016]);③ 别把 critic 聚合裸分当客观真理(不可跨评论家通约);④ 别把 SaaS / AI 厂商「采用数」当事实(混了已部署 + 仍在导入);⑤ 别被区块链「假酒占 20% / 损失 $70B」吓单买入(数字多来自厂商 / crypto 媒体未独立核实,业内估);⑥ 别迷信醒酒器 / aerator「神效」(多数酒开瓶即饮、拔塞几乎不增氧)。**结构性事实**:工具权威几乎不在 .edu/.gov(在认证机构 cert body + 厂商 vendor docs),verified_primary 占比低是行业事实非偷工 (evidence: [T02-S008, T02-S005])。

## 4. 工作流 / Pipeline

> 直接 reference [`references/research/03-workflows.md`](research/03-workflows.md) 8 个完整 workflow(全 last_updated 2026-06-06)。每个 workflow 入门 SOP 与资深路径分列,资深差异覆盖 跳过 / 优化 / 额外 三类。

### 4.1 盲品演绎流程（deductive tasting）— tacit→codified 的核心化身

入门 SOP(以 CMS grid 走一支红酒):Sight 看(clarity / color / 砖红边提示酒龄)→ Nose 闻(先判 clean / faulty TCA·brett·VA·还原硫,再用 Aroma Wheel 中心→外缘逼近)→ Palate 品(逐项给刻度:残糖 / 酸 / 单宁 / 酒精 / 酒体 / 风味强度 / 余味,**结构必须算出来**)→ Initial(旧 / 新世界 + 气候 + 品种)→ Final(国 / 产区 / 品种 / 酒龄 / 等级)。**资深差异**:**【优化】**网格内化、闻香 + 品味并行形成假设,先抓 structure 骨架做演绎;**【跳过】**跳过 legs / 挂杯、过度纠结具体果香名等低信息步骤;**【额外】**用排除法 + 味觉记忆库标志物锚定 + 估酒龄而非赌年份 + 现场追踪杯中演化。**Decay risk: 网格 low / 演绎基线 medium**(气候侵蚀经典规则) (evidence: [T03-S002, T03-S004])。

### 4.2 侍酒服务流程（floor service）— 一线 tacit 桌边手艺

入门 SOP:点酒确认(复述酒庄 / 年份 / 产区防开错)→ 开瓶(割箔 / 双节杠杆 / 检塞判 TCA;起泡转瓶不转塞;老酒提前直立沉淀)→ 判断是否醒酒(**不是所有酒都醒**:老酒去沉淀 / 年轻封闭高单宁增氧)→ 醒酒器 + 侍酒温度(起泡约 6-9°C → 饱满红约 16-18°C,约定俗成 / 各家不同 caveat)→ 试酒(查缺陷不是「喜不喜欢」)→ 斟酒 → BTG 保存(Coravin / 充氮)。**资深差异**:**【优化】**一眼读桌、把展示-开瓶-试酒-斟酒做成无停顿连贯动作,极老酒用滗酒篮原瓶上桌;**【跳过】**休闲场所 / 熟客省去全套展示话术与性别斟酒顺序;**【额外】**点单时就预判醒酒时机、管理多桌节奏 + BTG 损耗经济、缺陷边界临场拿捏(低量 brett 是否换瓶)。**Decay risk: 礼仪 low / BTG 工具 medium** (evidence: [T03-S007, T03-S009])。

### 4.3 配餐流程（food pairing）— 从经典原则到反直觉

入门 SOP:定菜主导维度(脂 / 酸 / 甜 / 辣 / 蛋白)→ 套交互原则(酸配脂、酸配酸、单宁配蛋白 / 脂、地配地、甜配甜、congruent vs complementary)→ 避雷危险食材(芦笋 / 朝鲜蓟 / 极酸醋 / 大量辣)→ 给 2-3 安全候选 + 一句体验语言。**资深差异**:**【跳过】**不查配餐表、原则已内化成直觉直接从酒单调取;**【优化】**配「菜的整体」(酱汁 / 烹法 / 配菜)而非单一蛋白;**【额外】**反直觉配对(Sauternes 配蓝纹、Pinot 配三文鱼、起泡配炸鸡)、找风味桥、故意制造张力创造记忆点、顾及整桌酒序(轻→重、干→甜)。**反例不软化**:「必须红配红 / 白配鱼」是被资深拒绝的僵化规则。**Decay risk: 经典原则 low / AI 配餐 high** (evidence: [T03-S010, T03-S023])。

### 4.4 认证备考工作流（WSET Diploma / CMS / MW）— 入门与资深分野最大

入门 SOP:选体系对齐目标(零售 / 教育 → WSET;餐厅服务 → CMS;研究 / 权威 → MW)→ 走阶梯起步级 → 建三件套学习底座(教材 + 抽认卡 spaced repetition + 盲品网格练到本能 + 产区盲图默画)→ 盲品肌肉训练(固定酒单按 grid 走、对答案、复盘)→ 应试。**资深差异**:**【跳过】**跳级 + 免前置(WSET Diploma 可作 MW 前置)、用既有产区知识跳过基础课;**【优化】**靶向补弱项产区(用 GuildSomm / Matthew's World)、组盲品小组 + 找 MS / MW mentor 校准;**【额外】**顶级路径独有动作(WSET D6 研究论文 / CMS 服务 + 盲品 + 理论三关 / MW 6000-10000 词 research paper + 3×12 盲品,全球约 300+ MW,evidence: [T03-S013])。**反例不软化**:CMS-Americas 2018 曝考试舞弊(盲品试题泄露、撤销当年头衔),暴露服务盲品派认证可信度脆弱。**Decay risk: 路径结构 low / 考纲事实 medium** (evidence: [T03-S012, T03-S013])。

### 4.5 酒窖 / 酒单管理工作流（cellar + wine list + BTG 经济学）

入门 SOP:CellarTracker 入库录入(给适饮窗估计)→ 存储校准(约 12-14°C / 避光 / 横放 / 低震)→ 盘点 + 适饮跟踪 →(餐厅)构建分层酒单 → BTG 定价(批发价 × markup 约 4-5x ÷ 每瓶杯数,目标 pour cost 约 20-25%,trade 基准 / 各店不同,evidence: [T03-S017])。**资深差异**:**【优化】**graduated pricing 阶梯加价(低价酒高倍率、高价酒低倍率)+ 15-25% waste buffer;**【跳过】**跳过手工 Excel、用 SaaS + POS 集成自动下架(降 86 list 错误);**【额外】**周转 / 库存经济、BTG vs 整瓶 + Coravin 决策、配额 allocation 管理(维护 distributor 关系争膜拜酒)、酒单叙事 + 季节性 BTG 轮换(BTG 毛利约 75-80% 远高于整瓶约 35%、损耗也高,trade 基准 caveat)。**Decay risk: 存储 low / SaaS+定价 medium / 列级触发 high** (evidence: [T03-S015, T03-S017])。

### 4.6 采购 / 查价 / 选酒工作流（Wine-Searcher + en primeur + 三级分销）

入门 SOP:Wine-Searcher 查全球零售价 / 均价 / 资料 → 判品质(看「谁打分」而非裸分,查具体评论家长稿)→ 走合规采购渠道(美国餐厅 / 零售须经持牌 distributor、不能绕开直购,三级制)→ 记录入库。**资深差异**:**【优化】**多渠道比价 + Liv-ex 验资产价 + 用 distributor / 进口商关系拿配额;**【跳过】**对配额制膜拜酒 / 期酒跳过常规比价(靠 allocation,能不能拿到货才是关键);**【额外】**en primeur 期酒决策(年份桶样分 × 放价 vs 二级市场 × 配额,18-24 月后交割)+ 各州 DTC 法规。**反例不软化**:「这支投资酒稳赚 / 越陈越值钱」是诈骗高发话术,二级市场波动且充满 fraud;2025 例——Cheval Blanc 2025 期酒约 €336 ex-négoce(+20%、弱美元放大美国零售约 $473、34 年最小年份,厂商放价 / 评论口径 caveat,evidence: [T03-S020])。**Decay risk: 三级制法理 low / 期酒价格 high** (evidence: [T03-S018, T03-S019])。

**近期工作流变化的四大外部驱动**:(A) **气候变暖重写采收窗口与演绎基线**(糖先于酚类成熟的「采收悖论」、凉产区酒精上升、适饮窗前移,Winkler index 重订);(B) **法规 / 列级变动**(德国 2021 产地金字塔、Saint-Émilion 约 10 年重审 2022 多家退出、EU 2021/2117 地理标志修订 → 改写酒标合规 + 酒单估值);(C) **市场触底回升**(Liv-ex 100 在 2025 末-2026 初连续 6 个月上涨、勃艮第占交易额约 31.8%,平台自报指数 caveat,evidence: [T05-S012]);(D) **AI / 数字工具进侍酒侧但有硬边界**(AI 推荐 / 数字酒单有真采用,盲品 / 桌边判断 sensory 死穴) (evidence: [T03-S023, T06-S017])。

## 5. 表达 DNA

**高频黑话 / register 标志**:说话默认**先问「哪里」(terroir-first)、先抓 structure 骨架、用校准刻度词而非情绪词**。核心词:terroir / tannin / acidity / minerality / structure / balance / complexity / finish(length) / appellation(AOC·AVA·PDO) / vintage / cru(grand·premier·climat) / blend(assemblage) / MLF(苹乳发酵) / brett / TCA(corked) / decant / retronasal / residual sugar / botrytis(贵腐) / SAT / deductive grid。读法:dry / sweet 只指残糖(与单宁 / 酸无关)、tannin 是触觉不是味觉、aroma(一二级品种 / 发酵香)≠ bouquet(三级陈年香)。

**严肃 vs 轻松 register**:严肃讨论(品鉴 / 备考)走 SAT / deductive grid 固定顺序 + 刻度词,目标可复现、可被他人核对;轻松吐槽用 slang(flabby 缺酸软塌 / hot 酒精灼 / closed·dumb 哑期 / bretty / VA 醋味 / petrol 陈年 Riesling 汽油味——但 petrol 在 Riesling 是褒义、在别处是缺陷,**褒贬取决于语境**)。

**内 vs 外沟通差异**:对客 / 新手把技术词翻译成体验语言(「这支酸度活泼、配生蚝很搭」);对同业默认共享术语栈直接上判断。**流派在表达层也分裂**:风土派说 terroir / climat / appellation,科学派说 quercetin / rotundone / MLF / 数据检视,自然酒派说不加不减 / 原生酵母 / 低干预,评分派说 95 分 / fruit bomb——四套话语暴露四种世界观。

### 5.1 对话样本库（industry voice 实战语料）

#### 5.1.1 客户版（面客解释 / 去精英化教育）

- 「这支酸度活泼、尾巴干净,配生蚝或柠檬黄油白鱼很搭——它的酸像挤一把柠檬,帮你切开油脂、清口。」(source: T06-S036, 转述, 把技术判断翻译成体验语言)
- 「葡萄酒不该是势利俱乐部的入场券;用普通人能懂的话讲清它来自哪、什么品种、怎么酿就够了,不必背术语。」(source: T01-S037, 转述, Schneider 去 snobbery 立场)

#### 5.1.2 同业版（私下 / 内训 / 盲品组）

- 「先把 structure 拎出来再说香气——酸、单宁、酒体、酒精这副骨架比香气难伪装、更指向气候和产地。」(source: T03-S004, 转述, Jancis 盲品演绎口径)
- 「这支有点 flabby、缺酸,尾巴还 hot;低量 brett 我无所谓,但这量已经是 fault 了,该跟厨房说换瓶。」(source: T06-S025, 转述, slang register + 缺陷边界)
- 「别背 solver 式标准答案——SAT 是描述评质、不必猜中,deductive grid 才是演绎到产地年份,你考哪个体系就练哪套。」(source: T03-S003, 转述, 两套盲品方法之别)

#### 5.1.3 专业 / 理论版（公开谈标准 / 学术 / 评论独立性）

- 「分数不是酒的属性,而是酒与品鉴者互动的属性。」(source: T01-S002, 原话, Jamie Goode,被 Jancis 引用)
- 「把复杂的感官压成单一数字会损害葡萄酒文化;我宁可用刻度粗的 20 分制,让分数不假装精确。」(source: T01-S001, 转述, Jancis 反百分制论证)
- 「曾被当作『病』的苹乳发酵,其实是应被主动鼓励和控制的过程——这是把酿酒从经验 / 迷信变成可控科学的一步。」(source: T04-S028, 转述, Peynaud 现代酿酒学范式)
- 「真酒不一定好喝或好看,但有打动你的内在;评分与技术化是『冷聪明』,生物动力才捕捉得到一地的生命力。」(source: T01-S044, 转述, Nicolas Joly 生物动力宣言,争议立场如实记录)

#### 5.1.4 反例版（这一行资深人**绝不会**这样说 / 被错位包装的销售话术）

- 「95 分 = 客观上最好的酒。」(source: T06-S032, why 反例:分数是单一评论家特定时点的主观印象、系统性偏爱 fruit bomb、不可跨评论家通约)
- 「自然酒更健康、不上头、不宿醉。」(source: T06-S030, why 反例:宿醉主因是酒精本身,有机 / 低干预不改变这点,是 wellness 营销错位)
- 「sulfites 让我头疼,所以买无硫酒。」(source: T06-S006, why 反例:白葡萄酒含等量 sulfites 却少致头疼,红酒头疼真因更可能是 quercetin)
- 「这支投资酒稳赚、保证升值;反正越老越好。」(source: T06-S033, why 反例:「稳赚」是投资酒诈骗典型话术,二级市场波动且充满 fraud,且 95-98% 的酒不为陈年而造、放老只会衰败)

**voice_confidence: medium**——tagged 样本 9 段(客户 2 / 同业 3 / 专业 4,各带 原话 / 转述 标)+ 反例 4 段(带「why 反例」标),其中仅 1 段为 figure 原话(Goode「分数不是酒的属性」),其余为忠实转述。**根因是结构性的**:本行核心 figure 原话几乎全为英文,中文 voice 只能转译,故标 medium 而非 high(Phase 2.8 诚实边界明示)。来源可回溯 Track 01 figures + Track 06 厂商话术拒绝 / register 节。

## 6. 质量基准 + 反模式

**什么算「好」(质量基准,4 条具体可验证)**:
1. **terroir-first 定位**:讨论一支酒先做地理定位(国 → 产区 → 地块)再谈品种 / 风格,而非一上来「好不好喝 / 多少分」。
2. **structure 先于香气、用刻度词**:盲品 / 品鉴按 sight→nose→palate 逐项给校准刻度(low / medium+ / high),从结构骨架演绎、用排除法,而非堆情绪形容词。
3. **判分数看「谁打 + 偏好」不看裸分**:能说清一个分数是哪位评论家、什么风格偏好、什么利益位置下打的,不把聚合裸分 / Vivino 均分当客观品质。
4. **数字挂来源 / caveat + 识破厂商话术**:醒酒时间 / 侍酒温度 / markup / 配额 / 期酒涨幅都标「约定俗成 / 各家不同 / 厂商自报 / 评论口径」,遇「必须醒 X 小时 / 自然酒不上头 / 投资酒稳赚」能回到机理拒绝。

**反模式(外行 / 入门常犯,9 条)**:
1. 把品种当产区(「勃艮第那个葡萄」)——Burgundy 是产区,红勃艮第 = Pinot Noir (evidence: [T06-S028])。
2. 「越老越好」——95-98% 的酒应趁年轻喝,只有少数高酸 / 高单宁酒受益于陈年 (evidence: [T06-S008])。
3. 「挂杯多 = 好酒」——legs 只反映酒精度 + 表面张力,与品质无关 (evidence: [T06-S025])。
4. 「所有酒都要醒」+ 拔塞「呼吸」当醒酒——多数酒开瓶即饮,瓶颈几乎不增氧、要真醒得倒醒酒器 (evidence: [T06-S035])。
5. 「sulfites 让我头疼」——白酒含等量 sulfites 却少致头疼,真因更可能是 quercetin / 酒精代谢 (evidence: [T06-S006])。
6. 「这起泡酒是香槟」——只有 Champagne 法定产区的才叫香槟,其余叫 sparkling / Cava / Prosecco (evidence: [T06-S034])。
7. 把「smooth / 顺滑」当唯一褒义——内行用 balance / structure / length / complexity,smooth 多指低单宁低酸 (evidence: [T06-S036])。
8. 把高单宁红酒说「太甜 / 太干」——dry / sweet 只指残糖,单宁是收敛感、是另一维度 (evidence: [T06-S010])。
9. 把 Vivino 均分 / critic 裸聚合分当客观品质权威——均分受甜度 / 价格 / 品牌污染,分数不可跨评论家通约 (evidence: [T02-S008, T06-S032])。

## 7. 智识谱系

**流派 ① WSET 知识体系派 vs CMS 服务盲品派** (figures: Levi Dalton / Rajat Parr / Madeline Puckette / Elizabeth Schneider)——WSET(知识广度 + SAT 可写网格、Diploma 笔试导向,知识普及侧有 Puckette / Schneider / MacNeil 图解·播客·叙事三入口)vs CMS(餐厅一线服务 + 演绎盲品 + 理论三合一,MS = 侍酒师金标,当前代表 Dalton / Parr 侍酒师出身)。核心分歧:「专业 = 知识广度」vs「专业 = 一线服务手艺 + 现场演绎」;盲品认识论 SAT(描述评质) ⇄ deductive grid(演绎识别)。**反例不软化**:CMS-Americas 2018 考试舞弊丑闻暴露服务盲品派认证可信度脆弱 (evidence: [T01-S036, T01-S034])。

**流派 ② 旧世界风土派 vs 新世界品种 / 技术派** (figures: Hugh Johnson / Jancis Robinson / Maynard Amerine / Albert Winkler)——旧世界(Johnson《World Atlas》地理优先 + Jancis / Harding Oxford 事实层 + AOC / DOCG / VDP 按地块)vs 新世界(UC Davis Amerine / Winkler 气候区指导「哪里种什么」+ AVA 只圈地 + varietal 标签)。核心分歧:「地理 / 产区先于品种」⇄「品种 / 技术先于地理」;José Vouillamoz 用 DNA 在此交叉(支持品种科学、又祛魅旧世界起源神话) (evidence: [T01-S015, T01-S025])。

**流派 ③ 自然酒 / 低干预派 vs 传统技术 / 科学审视派** (figures: Nicolas Joly / Isabelle Legeron / Émile Peynaud / Jamie Goode)——自然酒(生物动力源头 Joly / Coulée de Serrant + 当代旗手 Legeron《Natural Wine》+ RAW WINE,有机 / 生物动力 + 酒窖「不加不减」)vs 传统技术(Peynaud 清洁科学控制 + UC Davis)vs 科学审视中间派(Goode / Lewin 要求实证、不照单全收护教叙事)。核心分歧:「最低干预 = 捕捉地方生命力」⇄「干预 = 卫生科学保障」⇄「两边都要实证」。**反例不软化**:自然酒无官方定义 / 缺陷酒被美化是科学派实打实质疑;生物动力「宇宙能量」解释未被科学接受(有机农艺效果与玄学解释须分开看) (evidence: [T01-S041, T01-S021])。

**流派 ④ Parker 百分制点评派 vs Jancis 20 分制散文派** (figures: Robert Parker / Antonio Galloni / Jancis Robinson / Tim Atkin)——百分制(Parker 50-100 分消费者友好 / 偏浓郁果味 + Galloni Vinous 平台化继承 + Tim Atkin 产区年报)vs 20 分制(UC Davis Amerine 1959 学术校准 → Jancis 散文式独立评论)。核心分歧:「分数可跨评论家通约、是消费速记」⇄「分数是酒与品鉴者的互动、不可通约、宁可粗」。**反例不软化**:Parker「Parkerization 同质化」批评有大量从业者体感、其自辩亦是一手立场;Galloni Vinous Preview 提前放分被批评论独立性向商业倾斜;百分制数字(含 Tim Atkin 单一评论家产区分级)是主观刻度须挂 caveat (evidence: [T01-S007, T01-S001])。

**流派 ⑤ 经典平衡派 vs 浓郁现代派酿造** (figures: Émile Peynaud / Rajat Parr / Robert Parker / Nicolas Joly)——经典平衡(Parr + IPOB 加州反 fruit bomb,求优雅 / 低酒精 / 鲜酸 / 可陈年)vs 浓郁现代(Parker 偏好经市场放大成「国际风格」+ 飞行酿酒师 micro-oxygenation 软单宁)vs 第三条路(Joly 生物动力反技术主义)。**反例不软化**:Peynaud 是这条轴双重原点(现代酿酒学之父 + 「Peynaudization 同质化」反思对象);IPOB「平衡」也被批为另一种风格教条(把低酒精道德化)、约 2016 解散 (evidence: [T01-S021, T01-S034])。

**核心分歧(保留不抹平)**:① 知识广度 vs 服务手艺(WSET / CMS);② 地理先于品种 vs 品种先于地理(旧 / 新世界);③ 干预 vs 放手(现代主义 / 自然酒,科学审视为第三方);④ 量化通约 vs 风土表达(百分制 / 20 分制);⑤ 平衡克制 vs 浓郁高分(经典 / 现代)。五轴交叉,同一人常跨多轴(如 Parr 兼 CMS 服务派 + 经典平衡派 + 低干预审美)。

**不可蒸馏的边界信号**:法 / 意 / 西本土当代名酿与波尔多 oenologue 当代代表多以本土语种发声、未进入英文 canon 生态,本谱系对旧世界本土一手覆盖薄(诚实边界)。

## 9. Agentic Protocol — 研究维度

> 这一行人拿到一个新问题(「这支酒怎么判 / 这道菜怎么配 / 这个采购该不该做 / 这个说法该不该信」)时,默认按以下维度做功课再答。7 个维度互斥,各产出不同结论。

### 9.1 地理与世界观定位（terroir-first）

- **看什么**:这支酒来自哪(国 → 产区 → 子产区 → 地块)?属旧世界(以产地为名、terroir 至上)还是新世界(以品种 / 技术为名)?那块地的气候 / 土壤 / 法定分级如何?
- **在哪看**:Oxford Companion / World Atlas of Wine(产区空间锚)、GuildSomm 产区地图、AOC cahier des charges / DOCG disciplinare、新世界看 AVA + varietal 标签。
- **输出格式**:1 句「本酒属 {旧 / 新世界},地理锚 = {产区 → 地块},风格 / 品种线索 = {…},是否受气候变暖改写地理基线」。

### 9.2 结构化品鉴（SAT / deductive grid）

- **看什么**:sight→nose→palate 各项刻度(残糖 / 酸 / 单宁 / 酒精 / 酒体 / 风味强度 / 余味);structure 骨架指向什么气候 / 产地?clean 还是 faulty(TCA / brett / VA / 还原)?
- **在哪看**:WSET SAT 网格(描述评质)或 CMS deductive grid(演绎识别)按目的选;Aroma Wheel 从中心向外缘逼近落香气词;标准物盲闻校准味觉记忆。
- **输出格式**:1-2 句「结构 = {酸 / 单宁 / 酒体 / 酒精刻度},初判 = {旧 / 新世界 + 气候 + 品种},终判 = {国 / 产区 / 品种 / 酒龄 / 等级 + 可辩护理由链}」。

### 9.3 信源与分数解码

- **看什么**:这个说法 / 分数是谁说的、什么风格偏好、什么利益位置?是评论家一手长稿、trade 二手、还是众包均分?数字是单一评论家主观刻度 / 厂商自报 / 平台口径吗?
- **在哪看**:评论家长稿(Jancis Purple Pages / Vinous / Tim Atkin 年报)而非裸聚合分;Wine-Searcher 看价不看裸分;Liv-ex 验「作为资产」的真实成交价;CellarTracker 筛 favorite tasters 后的笔记。
- **输出格式**:1 句「信源 bucket = {一手 / 二手 / 众包},分数口径 = {单一评论家主观 / 平台自报 / 大众污染},可信度 = {高 / 中 / 低 + 须挂什么 caveat}」。

### 9.4 配餐推理

- **看什么**:菜的主导维度(脂 / 酸 / 甜 / 辣 / 蛋白)是什么?酱汁 / 烹法 / 配菜如何改变配酒?有无危险食材(芦笋 / 朝鲜蓟 / 极酸醋 / 大量辣)?是求安全和谐还是反直觉记忆点?
- **在哪看**:配餐交互原则(congruent vs complementary、酸配脂 / 单宁配蛋白 / 地配地,Wine Folly / WineWiki)、危险食材清单、整桌酒序(轻→重、干→甜)。
- **输出格式**:1 句「主导维度 = {…},2-3 个安全候选 + 1 个反直觉选项,对客翻译成一句体验语言」。

### 9.5 法规 / 分级 / 合规定位

- **看什么**:本酒受哪套法定分级约束(AOC / AVA / PDO / Prädikat / VDP / 1855 / Saint-Émilion / Burgundy climat)?酒标合规阈值(产地 / 品种 / 年份 %)?采购受什么约束(三级分销 / DTC / 配额)?
- **在哪看**:INAO(AOC / AOP)、VDP(德)、Bordeaux.com(1855)、TTB 27 CFR Part 4 / eCFR(美国酒标)、EUR-Lex(EU PDO / PGI);近期变动看德国 2021 法改 / Saint-Émilion 重审。
- **输出格式**:1 句「本酒受 {哪套分级} 约束,标签合规点 = {产地 / 品种 / 年份 % 阈值},采购渠道约束 = {三级制 / DTC / 配额}」。

### 9.6 张力轴定位

- **看什么**:本决策 / 争议站在哪条张力轴上(干预⇄放手 / 描述⇄演绎 / 量化通约⇄风土表达 / congruent⇄complementary / 仪式⇄高效)?两端各自的正当性与代价?我方 / 对方利益与口味位置在哪?
- **在哪看**:智识谱系 5 派立场对照(Track 01 figures),把争议方落到对应奠基人 → 当前代表 → 核心分歧上。
- **输出格式**:1 句「本问题属 {哪条轴},我方立场 = {…},对立面 = {…},别把任一端道德化成教条」。

### 9.7 时效与近期变化

- **看什么**:本结论会被什么侵蚀——气候(采收窗 / 适饮窗 / 演绎基线)、法规(列级重审 / 产地法改)、市场(Liv-ex 指数 / 期酒)、工具(Coravin / 数字酒单 / AI)?哪些是 high decay 须重校准?
- **在哪看**:GuildSomm / Jancis / VinePair 最近期内容、Liv-ex News & Insights、Saint-Émilion / 德国法改 / EU 2021/2117、ProWein 趋势(no-low / 自然酒)。
- **输出格式**:1 句「本结论衰减点 = {气候 / 法规 / 市场 / 工具中哪几条},decay = {high / medium / low},是否需重校准 + 重校准看哪个源」。
