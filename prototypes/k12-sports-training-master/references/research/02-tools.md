## 这一行「工具」的特殊定义（先校准，再看清单）

**这一行不是软件行业**。「工具」≠ 垂直 SaaS，而是「让一家体培机构跑起来的软硬件 + 教研体系」拼装。三个结构性事实，决定了本文件和 LLM-agent / 跨境电商那种「GitHub stars 排座次」完全不同：

1. **没有强主导垂直 SaaS**。体培机构用的运营系统，绝大多数是**通用 K12 教培 SaaS**（校宝 / 校管家 / 爱耕云）或**泛健身 SaaS**（三体云动）「借用」过来的——没有一个「体育培训专用、占有率 > 50%」的系统。必备层因此**只能凑到 3 个真·普及工具**（排课收银 SaaS 类 + 企业微信 + 美团/抖音到店），且第 3 个（本地生活）的「必备」属性偏获客侧而非教务侧，已诚实标注 partial。(evidence: [T02-S001, T02-S002, T02-S003])
2. **真正的「护城河工具」是自研课程体系**，不是软件。头部机构（动因 / 小象皮尼 / 各加盟品牌）的核心资产是「分龄分级课程体系 + 教案库 + 动作示范视频库」，这部分**几乎全部闭源、无公开 URL**，只能从行业报告 / 协会标准侧面推断。(evidence: [T02-S015, T02-S017, T02-S031])
3. **唯一真·爆发的新工具品类是 AI 体测 / 智慧操场**（科大讯飞 / 有道 / 格灵深瞳 / 恒鸿达），但它**主战场在 toB 进校（教育局采购）**，体培机构是「中考体育考前训练」场景的二级用户——它在重塑「中考备考」业态，是本 track 唯一有清晰「新兴 → 主流」曲线的品类。(evidence: [T02-S006, T02-S007, T02-S008, T02-S009])

因此：**必备层证据不走「stars 中位数 ×2」，走「≥3 独立 source 点名 + 行业普遍当 default」的采用率替代信号**（与 private-domain-ops / insurance-broker 同行的做法一致，已在 Phase 2 接口「冷僻」节标注）。

## 必备层

### 1. 教务/收银 SaaS（校宝在线 / 校管家 / 爱耕云 三选一）

- **One-liner**: 让机构「排课-签到-课消-收费-续费提醒」跑成闭环的运营底座。**本行最关键的诚实点：没有体育专用款**，机构一律借用通用 K12 教培 SaaS，或泛健身 SaaS（三体云动）。(evidence: [T02-S001, T02-S023])
- **Tier**: 必备（采用率替代信号达标：≥3 独立 source 点名 + 行业 default）
- **Maintainer / Owner**: 校宝在线（https://www.xiaobaoonline.com/）/ 校管家（https://pc.schoolpal.cn/）/ 爱耕云（http://aigengyun.cn/）
- **License**: proprietary（校宝/校管家按年付费，基础版免费；爱耕云基础版年费千元级）
- **Maturity signals**:
  - GitHub stars: N/A（闭源 SaaS）
  - 采用信号：校宝「深耕 16 年」服务教培+民办校；三大平台被 ≥3 第三方横评反复对比（2026-06-04）(evidence: [T02-S023, T02-S029])
  - Activity: healthy（持续迭代，AI 排课为 2025-2026 新增点）
- **典型使用场景**:
  - 排课-查课（校宝排课查课口碑最好）、课时统计、移动考勤、补课请假
  - 前台收费+课消（签到即课消、付费即开课），剩余课时不足自动提醒家长续费
  - 招生 CRM + 多校区报表（校管家报表维度最全，60+ 报表 / 七大维度）(evidence: [T02-S001, T02-S002, T02-S023])
- **不适合 / 替代方案**:
  - **单店重门禁/IoT 的大场馆**（大体适能中心/泳馆）→ 用三体云动（健身 SaaS，IoT 生态全）替代 (evidence: [T02-S004])
  - **纯进校 toB**（课后服务）→ 教务 SaaS 用不上，走课后三点半 / 国家平台 (evidence: [T02-S012, T02-S013])
  - ❌ **别用纯学科类教培 SaaS**：学科款的排课模型是「固定班级+周期」，体培是「储值课包+灵活约课+按次课消」，套上去课消和约课全拧巴（详见避坑清单）
- **生产案例**: 三大平台均有万级教培机构客户；本行体培机构普遍从「学科教培邻居」处迁移过来同款（行业横评佐证，非单一 case）(evidence: [T02-S023, T02-S029])
- **近期变化**（近 12 个月）: AI 智能排课普及（输入师资/教室/课程一键排课、自动规避冲突）成为各家 2025-2026 主打 (evidence: [T02-S003])
- **可信度**: high
- **Decay risk**: **low**（教务收银是稳态基础设施，24+ 月被取代概率 < 20%；变的是 AI 功能皮肤，不是底座）

### 2. 企业微信（+ 轻量 SCRM 模块）

- **One-liner**: 双减后家校沟通 + 家长私域的合规底座；续费/转介绍的触达全靠它，多数机构用「企微 + 通用 SCRM 插件」拼，**没有体育专用 SCRM**。(evidence: [T02-S005, T02-S016])
- **Tier**: 必备
- **Maintainer / Owner**: 腾讯（https://work.weixin.qq.com/）
- **License**: proprietary（企微免费；SCRM 插件按年付费）
- **Maturity signals**:
  - GitHub stars: N/A
  - 采用信号：2024 起企微已是 zh-CN 服务业私域 default 主战场（跨 Track 复用 private-domain-ops 结论）；体培家校沟通普遍用企微而非个人号 (evidence: [T02-S005])
  - Activity: healthy
- **典型使用场景**:
  - 家长群 SOP（课前提醒/考勤通知/作业打卡）、1v1 续费跟进、朋友圈做体测成果展示
  - 试听到店后承接到企微，做转化与转介绍裂变
  - 教务 SaaS 数据 → 企微推送（课时不足/请假提醒）(evidence: [T02-S016])
- **不适合 / 替代方案**:
  - 规模化前（单店 < 200 学员）不必上重 SCRM，企微原生「客户联系」够用；超 100 群再上 SOP 工具
  - ❌ 别再用个人号多开/外挂自动加好友——封号红线（行业通识，private-domain-ops 已立标本）
- **生产案例**: 行业普遍实践（教培 SCRM 解决方案明确以「家校服务全场景 + 续费」为卖点）(evidence: [T02-S016])
- **可信度**: high

  - 注意中考考场规格管控（统一采购 PVC、直径 6-7mm、60-80g），门店训练款 ≠ 考场款
- **不适合**: 把消费级智能跳绳（Keep/李宁）当考场款——考场认标准款 (evidence: [T02-S018])
- **可信度**: medium-high（gov 报道 + 行业事实）
- **Decay risk**: **high**（硬件品类迭代快 + 政策规格可能变）

### 16. 上门体育 / 轻资产撮合工具（experimental）

- **One-liner**: 「教练送课到家」线上撮合平台——2024 占青少年体培增量 41%，是门店动荡期的「救命稻草」，但工具 SaaS 化仍早期、靠谱性受质疑。(evidence: [T02-S021])
- **Tier**: 新兴
- **stability**: experimental
- **Maturity signals**:
  - 早期采用者：2024 上门体育占青少年体培增量 41%（36氪 + 央广/体坛报均有报道与质疑）(evidence: [T02-S021])
  - 出现 → 主流：早期，撮合平台标准化/资质/安全是瓶颈
- **不适合**: 重教研/重续费的体系化机构（上门难标准化、难沉淀续费）
- **可信度**: medium（媒体报道，无成熟头部平台公开数据）
- **Decay risk**: **high**（模式未跑通，6 月后可能洗牌/收缩）

## 选型决策树

> 节点 8 个。每个分支可追溯到 manifest 证据。核心分歧轴：**业态（社区店 / Mall 店 / 进校 / 中考备考）× 直营自研 vs 加盟通用**。

### 决策树 A：你的机构是什么业态？

#### Branch 1: 社区店 / 轻资产小店（少儿体适能 / 单项启蒙，< 200 学员）
→ **教务收银**: 爱耕云（性价比 + 智能排课，年费千元级，中小机构核心需求最优解；vs 教务宝差异见横评）(evidence: [T02-S003, T02-S029, T02-S024])
→ **家长私域**: 企业微信原生「客户联系」即可，先别上重 SCRM (evidence: [T02-S005])
→ **获客**: 美团团购（体验课钩子）+ 点评口碑；有内容能力再上抖音 (evidence: [T02-S022])
→ **续费弹药**: 睿莱体测体系，把体测报告做成成长曲线给家长 (evidence: [T02-S010])
→ **不推荐**: 三体云动（IoT 太重，社区店用不上门禁/储物柜）(evidence: [T02-S004])
→ 真实参照: 儿童体适能行业 300-400 家机构多为此形态，FMS 课程体系 + 轻 SaaS；青少儿体育 2350 亿潜在市场、2023 全国体培 1324 亿/18 万校区，素质教育赛道体育占比领先 (evidence: [T02-S020, T02-S030, T02-S019, T02-S025])

#### Branch 2: Mall 店 / 大场馆（大型体适能中心 / 泳馆，有实体场馆动线）
→ **教务收银 + IoT**: 三体云动（门禁/人脸闸机/储物柜生态最全，310+ 模块）(evidence: [T02-S004])
→ **替代**: 校宝旗舰版（多校区 + 强报表，若不需要 IoT）(evidence: [T02-S001])
→ **获客**: 美团 + 抖音双投（Mall 店客流依赖线上种草）(evidence: [T02-S022])
→ **不推荐**: 纯学科教培 SaaS（课消/约课模型不匹配，见避坑）

#### Branch 3: 进校 / 课后服务（toB，课后三点半 / 课后体育）
→ **进校接口**: 课后三点半平台（报名/排课/资金监管）+ 国家中小学智慧教育平台（合规身份）(evidence: [T02-S012, T02-S013])
→ **教研对齐**: 《义务教育体育与健康课程标准 2022》为纲 (evidence: [T02-S014])
→ **若做进校中考训练**: 恒鸿达 AI 中考体育（进校 3180+，移动/固定式）(evidence: [T02-S008])
→ **不推荐**: 教务收银 SaaS（toB 进校用不上 C 端课消/收银）+ 本地生活（进校不靠到店获客）

#### Branch 4: 中考体育备考（应试提分，初中冲刺）
##### Branch 4a: 要「对齐真实考场评分」
→ **科大讯飞智慧体育**（其引擎就是多省考场同款，违规识别+精准赋分）(evidence: [T02-S006, T02-S027])
##### Branch 4b: 门店预算有限、要单机轻量
→ **有道趣动屏**（无穿戴、单机多项目切换，当考前打卡机）(evidence: [T02-S007])
##### Branch 4c: 单项刚需（跳绳）
→ **智能跳绳**（注意考场认标准款 PVC 6-7mm/60-80g，训练款 ≠ 考场款）(evidence: [T02-S018])
→ **共同基建**: 体测数据追踪 + 考纲拆解（政策要求过程性考核，数据要留痕）(evidence: [T02-S016])

### 决策树 B：直营自研 vs 加盟扩张？（横切轴）

#### Branch 5: 直营 / 重模型连锁（如动因式）
→ 倾向**自研课程体系 + 自研/半自研系统**（护城河在课程，不在买 SaaS）；运营系统选可定制的校宝旗舰/校管家 (evidence: [T02-S028, T02-S001])
→ 真实参照: 动因体育 5 大专项、18 城直营，靠品牌+地推+转介绍，本地生活弱依赖 (evidence: [T02-S028])

#### Branch 6: 加盟 / 轻扩张
→ 倾向**通用 SaaS（爱耕云/校宝）+ 加盟总部输出的课程体系**（加盟商要的是开箱即用 + 可复制）(evidence: [T02-S003, T02-S030])
→ 课程标准化（分龄分级）是加盟可复制性的命门 (evidence: [T02-S030])

## 避坑清单（≥ 5 条）

- ❌ **别用纯学科类教培 SaaS 套体育排课**：学科款排课模型是「固定班级 + 周期续报」，体培是「储值课包 + 灵活约课 + 按次/按时课消」，套上去课消统计和约课全拧巴。选明确支持「课时包 + 灵活约课 + 签到即课消」的（校宝/校管家/爱耕云均支持）。(evidence: [T02-S001, T02-S023])
- ❌ **别给社区店/小课包机构上三体云动这类重 IoT 系统**：门禁/人脸闸机/储物柜是大场馆需求，轻资产社区店付了重订阅费却用不上，性价比差；用爱耕云。(evidence: [T02-S004, T02-S003])
- ❌ **别把消费级智能跳绳/体测仪当中考考场款**：多地考场认统一采购的标准款（PVC 6-7mm、60-80g、特定品牌识别率 99%），训练用消费款和考场款规格/识别不一致，考前用错款误导成绩。(evidence: [T02-S018])
- ❌ **别把抖音本地生活当「躺赚获客」**：体培是低频高决策成本品类，无短视频/直播内容能力的小店在服务商抽佣模式下成本反高于美团；先用美团承接主动搜索，有内容能力再上抖音。(evidence: [T02-S022])
- ❌ **别用个人微信号多开/外挂做家长私域自动化**：封号红线（zh-CN 私域通识，WeTool 式外挂已成封禁标本）；家校沟通走企业微信合规底座。(evidence: [T02-S005])
- ❌ **别把「教练能打」当「能教 + 能用工具」**：行业「能打 ≠ 能教」断层普遍，教练认证（CNPF 少儿体适能 / 各单项协会教练员）+ 标准化教案库是补课程交付质量的工具，别只招退役运动员不给体系。(evidence: [T02-S011, T02-S030])
- ❌ **别迷信「买了 AI 体测一体机就能提分」**：AI 体测是「数据采集 + 评分对齐」工具，不替代训练计划；且这是 Decay high 的新兴品类，6-12 月产品形态可能大变，重资产采购前评估折旧。(evidence: [T02-S006, T02-S017])
- ❌ **别在「应试提分」和「长期运动素养」工具上混用同一套评估**：中考备考要对齐考纲评分引擎（讯飞/有道），体适能成长要用发展性测评（睿莱/FMS），两者评估逻辑不同，混用会让家长沟通错位。(evidence: [T02-S006, T02-S010])

## Phase 2 提炼提示

**反复出现 ≥ 3 source 都强调的「工具选型原则」**（候选 playbook 规则）:
- **「没有体育专用垂直 SaaS，靠通用工具拼装 + 自研课程体系」**——必备层是借用品（通用教培/健身 SaaS + 企微 + 本地生活），真护城河是闭源课程体系（出现于 T02-S001 / S004 / S030 / S015）
- **「业态决定工具，不是工具决定业态」**——社区店 / Mall 店 / 进校 / 中考备考 四类业态用完全不同的工具栈，按业态分流是第一决策（出现于 T02-S003 / S004 / S012 / S006）
- **「测评 = 续费弹药」**——把体测/考级数据做成给家长的成长报告是续费核心动作，工具（睿莱/CNPF/AI 体测）的真实价值是「家长可感知的结果证明」（出现于 T02-S010 / S011 / S016）
- **「政策是工具的需求引擎」**——双减 / 中考分值改革 / 课程标准 2022 / 过程性考核，直接催生了 AI 体测、进校平台、体测追踪三类工具的刚需（出现于 T02-S014 / S016 / S013）

**显著的工具流派分裂**（候选 智识谱系条目，对应 Track 01 figures / Phase 2.1 心智模型）:
- **引进体系派（美国 SPARK / 国外认证 / 体适能引进）** vs **本土自研派（按中国中考考纲 + 课标 2022 自研课程体系）**——课程工具的根本分歧 (evidence: [T02-S030, T02-S014])
- **体适能通识派（FMS 基本运动技能、3-12 岁、发展性测评工具如睿莱）** vs **专项技能派（篮球/游泳/击剑，单项协会认证 + 专项教案）** (evidence: [T02-S030, T02-S011])
- **中考应试提分派（AI 体测对齐考场评分：讯飞/有道）** vs **长期运动素养派（发展性测评 + 兴趣养成）**——评估工具逻辑对立 (evidence: [T02-S006, T02-S010])
- **直营重模型派（自研课程 + 可定制系统，护城河在体系：动因式）** vs **加盟轻扩张派（通用 SaaS + 总部输出课程，要开箱即用：爱耕云式）** (evidence: [T02-S028, T02-S003])
- **校外门店派（toC：教务 SaaS + 美团/抖音）** vs **进校派（toB：课后三点半 + 国家平台 + AI 体测进校）** (evidence: [T02-S012, T02-S022])

**新兴工具信号**:
- 当前活跃 / 上升的新工具数: **3**（AI 视觉体测一体机·训练版 / 智能跳绳进校 / 上门体育撮合）

- **结构性盲区**: ① zh-CN 服务/运营行业一手内容大量沉淀在被排除信源（公众号/知乎/抖音），一手率结构性偏低，已用 gov/协会/vendor docs 补厚度（一手率 61%）。② **真护城河「自研课程体系」几乎全闭源、无公开 URL**，只能侧面推断——SKILL.md 工具栈节的「课程体系」claim 需向用户标注「基于行业报告 + 协会标准推断，非直接观测机构内部教研」。
