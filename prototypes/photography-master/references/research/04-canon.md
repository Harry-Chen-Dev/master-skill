## 重点候选详述

### 📖 1. The Camera / The Negative / The Print（三部曲）
- **Author**: Ansel Adams（与 Track 01 figures 强关联）
- **Year**: 1980-83 (Bulfinch revised ed., 仍在印)
- **Type**: technical textbook trilogy
- **One-liner**: 黑白技术摄影的圣经，把摄影从「按快门」提升为「可控制的影调工程」。
- **核心论点**: ① previsualization——成像前先在脑中预想最终照片；② Zone System 把影调切成 0-X 11 区做精确曝光/显影控制；③ The Camera 讲控制成像、The Negative 讲曝光与显影、The Print 讲暗房放大。(evidence: [T04-S001, T04-S024])
- **读完得到什么**: 理解「曝光不是测光读数，是对最终影调的决策」这一 craft 思维内核。
- **难度**: 进阶-高阶。**Decay risk**: low（数字时代仍是影调思维底座）。**可信度**: high。
- **如果只读 1 本**: The Negative（Zone System 在此）(evidence: [T04-S001])
- **Endorsement**: course_syllabus（无数院校胶片课列入）+ 数字时代 Zone System 应用长文 [T04-S002] + 三部曲综述 [T04-S024]。

### 📖 2. Light Science & Magic
- **Author**: Fil Hunter / Steven Biver / Paul Fuqua
- **Type**: lighting textbook（多版，最新 6th ed.）
- **One-liner**: 唯一从光的物理第一原理（family of angles、反射定律）教布光的教材，而非「照抄这个布光图」。
- **核心论点**: ① 三种表面（漫反/直接反/抛光）决定打光策略；② 金属/玻璃这类难物体有可推导的布光逻辑；③ 布光是科学不是玄学。(evidence: [T04-S007])
- **读完得到什么**: 面对任何材质能自己推导布光，商业/产品/美食摄影的硬功底。
- **难度**: 进阶。**Decay risk**: low。**Endorsement**: 商业摄影课 syllabus 常列 + 多处 blog/媒体 [T04-S007]。

### 📄 3. The Decisive Moment（Images à la Sauvette）
- **Author**: Henri Cartier-Bresson；序由作者自撰，封面 Matisse 拼贴
- **Venue + Year**: Simon & Schuster / Verve, 1952（2014 重印）
- **One-liner**: 定义了「决定性瞬间」，街头/纪实摄影与现代摄影新闻的奠基文本。
- **核心 idea**: 一个瞬间里画面所有元素汇成揭示更大真相的完美几何构图。(evidence: [T04-S004, T04-S005])
- **为什么经典**: Capa 称其为「摄影师的圣经」；公认 20 世纪最重要摄影书之一；塑造了几代摄影记者。(evidence: [T04-S005])
- **争议**: 「决定性瞬间」被后世（如 the (in)decisive moment 论述）批评为对偶发/未完成性的排斥——构建派与抓拍派的张力源头。(evidence: [T04-S004])
- **Endorsement**: 基金会原文 + ICP 展览 + 媒体 ×多。

### 📄 4. Strobist «Lighting 101» 博客系列
- **Author**: David Hobby
- **Year**: 2006 起（rolling）
- **One-liner**: 把 off-camera flash 从专业影棚秘技变成全民可学，掀起「strobist」运动。
- **核心 idea**: 一两支廉价小型闪光灯 + 离机引闪 = 可控影棚级布光。(evidence: [T04-S011])
- **为什么经典**: 与 Joe McNally 同台 Flash Bus Tour；公认现代离机闪光教学的起点之一。(evidence: [T04-S011, T04-S012])
- **如何读**: 按 Lighting 101 → 102 顺序系统读，配合实拍。
- **Endorsement**: figure_long（McNally 公开同台/互推）+ 媒体。

### 📖 5. On Photography / Camera Lucida（理论双柱）
- **Authors**: Susan Sontag (1977) / Roland Barthes (1980, 英译 1982)
- **One-liner**: 摄影批评的两座理论灯塔——前者谈摄影的伦理/凝视/真实，后者谈照片为何「刺痛」你。
- **核心 idea**: Sontag——拍照是一种占有与凝视的权力行为，影像改变我们与现实的关系；Barthes——studium（可被文化解读的兴趣）vs punctum（无法言说、刺痛个体的细节）。(evidence: [T04-S008, T04-S009, T04-S010])
- **读完得到什么**: 从「服务交付的拍照人」升维到「对影像之真实/作者性有批判自觉的作者」。
- **难度**: 进阶（理论）。**Endorsement**: 几乎所有摄影理论课 syllabus 列为必读三件套（+Szarkowski）。(evidence: [T04-S008])

### 📖 6. The Photographer's Eye（Szarkowski, MoMA, 1966）
- **Author**: John Szarkowski（MoMA 摄影部主任，本人先是摄影师）
- **One-liner**: 提出摄影作为独立媒介自有的「看」的语法（the thing itself / detail / frame / time / vantage point）。
- **核心 idea**: 不用绘画的标准评判照片，照片有自己的五要素语言。(evidence: [T04-S008])
- **注意**: 与 Michael Freeman 的同名 The Photographer's Eye (2007, 构图教材) 是**两本不同书**——前者是策展理论，后者是实操构图。

### 🎓 7. Fundamentals of Photography（John Greengo, CreativeLive）
- **Lecturer**: John Greengo　**Institution**: CreativeLive
- **Format**: 讲座视频　**Duration**: 多 lessons（曝光/构图/器材）
- **Last_updated**: ~2020（自述为十年迭代后的最终版）(evidence: [T04-S016])
- **One-liner**: 把曝光三角、构图、相机控制讲得最系统的付费视频入门，CreativeLive 最热门课之一。
- **完整 vs 关键**: 想速成只看 Exposure + Composition 章；想打底建议完整。
- **难度**: 入门。**Decay risk**: medium（器材部分会衰减，原理不会）。

## Phase 2 接口

### 反复出现 ≥ 3 个 canon 都讨论的核心 idea（候选行业心智模型）
- **「光是主语，器材是宾语」**：出现于 Light Science & Magic / Strobist / McNally / Greengo → 候选心智模型「先问光，再问相机」。(evidence: [T04-S007, T04-S011, T04-S012, T04-S016])
- **「成像前就决定成像」（previsualization / 决定性瞬间）**：Adams 的 previsualization 与 HCB 的决定性瞬间是同一直觉的两极——前者是构建式预想、后者是抓取式预判 → 候选心智模型「拍摄是决策前置，不是事后补救」。(evidence: [T04-S001, T04-S004])
- **「照片有自己的语言，不借绘画的标准」**：Szarkowski / Sontag / Barthes → 候选心智模型「作者性 = 对媒介本身的自觉」。(evidence: [T04-S008])
- **「曝光是对最终影调的决策，不是测光读数」**：Adams Zone System + ETTR + 曝光三角共享 → 候选 playbook。(evidence: [T04-S001, T04-S018, T04-S023])

### 智识谱系种子（流派奠基 → 当前代表）
- **技术/影调流派**：奠基 Ansel Adams（Zone System, f/64 Group）→ 当前代表 风光/打印派、ETTR 数字延续。
- **纪实/街头流派**：奠基 Henri Cartier-Bresson（决定性瞬间）→ Magnum 谱系 → 当代街拍。
- **离机布光/商业流派**：奠基 Strobist (David Hobby) + Joe McNally（小灯派）+ Light Science & Magic（物理派）→ 当代商业/人像布光。
- **批评/理论流派**：奠基 Sontag + Barthes + Szarkowski（1970s 摄影理论转向）→ 当代影像伦理/作者性讨论。
- **主要分歧（still-debated）**：① 决定性瞬间(抓取) vs 构建布光(摆拍)；② 前期(in-camera)正确 vs 后期(frequency separation/dodge&burn)塑造；③ 真实纪录 vs 修饰美化（纪实伦理 vs 商业/人像可接受的 retouch 程度）。(evidence: [T04-S004, T04-S022])

### 核心概念 → 候选 playbook
- **Zone System / previsualization**：遇到高反差场景 → 先想最终想保留哪些影调区域，再据此定曝光/显影（数字时代映射为 ETTR + 后期还原）。
- **family of angles（L S&M）**：遇到反光/金属/玻璃材质 → 不是加光，是先算反射角再定光位。
- **决定性瞬间**：街拍/纪实 → 预判动作峰值与几何对齐，提前构图等待，而非连拍碰运气。
- **布光模式库（蝴蝶/环形/伦勃朗/分割）**：人像 → 按脸型与想要的情绪从模式库选 key light 位，再用 fill 定光比。
